　　第659章
<br/>　　柳芷双手搅在一起，眼眶泛红，双目垂泪，像是有一腔心思委屈，但却一个字都没说，只是眼巴巴看着程大雷。
<br/>　　程大雷心里琢磨着，如果真有了意中人总是好事，她本不该这个样子。现在，程大雷隐隐觉得事情有些不对，至于究竟何处不对，他现在还没琢磨明白。
<br/>　　柳芷的确有满腹心事要聊，只不过有些话，她一个女儿家委实有些不太好说出口。她很小就跟了程大雷，说是为奴为仆，照顾程大雷衣食起居，但程大雷对身边人还都蛮算可以的。
<br/>　　在许多人看来，柳芷早已经算是程大雷的屋里人，日后迟早都要被程大雷收了房做小。可事实上，柳芷知道自己并不是程大雷的屋里人。
<br/>　　日子一天天过去，随着柳芷一日一日长大，她的状况也变得越来越微妙。目前，柳芷面前只有两条路可走，要么是被程大雷收了房，要么就是程大雷那天一开心，随便赏给身边的某个下人。
<br/>　　这种事在程大雷看来颇为不可理喻，但却又是帝国无时无刻不在发生的事情。
<br/>　　程大雷自己是意识不到，他有了樊梨花与苏樱，早已一个头两个大。于是，也就没有扩充后院的心思，对身边的很多事情也没看见。
<br/>　　委实说，程大雷在生活作风上还是很检点的，但有时候，太过检点也未必是什么好事。
<br/>　　这件事程大雷不说，如徐神机刘悲这些人，也不会去关心一个小丫鬟的想法。而柳芷虽然想提，也在苏樱面前透过心思，可关键是苏樱也有苏樱的烦恼。更何况，她还没有大度到要为程大雷扩充后院的程度。
<br/>　　这件事程大雷假若提了，于她而言不算什么，可如果程大雷不提，苏樱自然也乐得不提。
<br/>　　平日里，柳芷想见程大雷的机会也不多，更何况是单独说话。也就是今天，她逢到这个机会，看程大雷已经很疲惫了，但自己的终身，还是要自己争取的。
<br/>　　于是，便拐弯抹角点了程大雷一句。
<br/>　　程大雷仍旧心里糊涂，只是心里有些不舒服，见柳芷已经落下泪来，他忙道：“好了，好了，你不要哭么，是不是那个混蛋欺负你了，你不要着急，本当家于你做主，说说，你有什么委屈？”
<br/>　　柳芷忽然屈膝跪倒，以手撑地，程大雷委实吓了一跳，扶也不是，不扶也不是你。
<br/>　　“柳芷本就是无根之草，又哪里敢说什么委屈不委屈，大当家身边端茶递水的小丫头，大当家既然看不上我，就该早早将我舍了与别人，也不要让外面风言风语。”
<br/>　　“什么风言风语？”
<br/>　　“有些话太脏，柳芷不好意思说出口，他们只是说，当初我是大当家从青楼抢出来的，一双朱唇任人尝……”
<br/>　　“何其大胆，是谁如此胡说八道。”程大雷顿时怒不可遏，狠狠的一拍桌子。
<br/>　　发作过后，程大雷忽然冷静下来，隐隐觉得这里面的事情不对。柳芷当初的确是程大雷从青楼抢出来的，可她跟了程大雷，若说她清白，怕程大雷都不好意思说这两个字。
<br/>　　你若委屈可以，受了欺负也可以理解，但这话里话外，怎么仿佛都是自己的不是。
<br/>　　“起来说话，起来说话，不管怎样的事，都有本当家于你做主。”
<br/>　　柳芷颤巍巍起身，忽然一头跌在程大雷怀里，头靠着程大雷肩膀，轻轻啜泣。
<br/>　　“大当家，柳芷心里委屈。”
<br/>　　程大雷身子一下僵住，温玉满怀，他顿时像被施了定身法时的。柳芷一声声哭着，委实令程大雷觉得有些不是滋味。
<br/>　　今夜柳芷明显已经有些动情，想半推半就将自己舍了程大雷，也算落个踏实。程大雷也并非是不吃肉的主儿，但今天他先后在樊梨花与苏樱处完成任务，确实是有心无力。
<br/>　　柳芷没有达成自己的小心思，颇有些无可奈何，心里也揣测不明白程大雷是如何想的。
<br/>　　关键是程大雷自己心里也糊涂着呢，到了第二天，程大雷养足精神后，仍旧挂念昨夜的事，心里想着是找个明白人问问。
<br/>　　不过，面前还有一个问题：蛤蟆城没有明白人。
<br/>　　蛤蟆城不缺能打的武将，也有出谋划策的谋士，但这种儿女情长，家长里短的事……徐神机自然不用提，刘悲怎么看都不想个明白人，至于其他人，程大雷不觉得他们比自己更明白。
<br/>　　和珅倒勉强能算一个明白人，但他如今在朔方城，程大雷琢磨着找机会问问他。
<br/>　　程大雷又在蛤蟆城住了几日，委实说，日子过得不是很愉快。应付苏樱已经是一件令人头疼的事情，关键是现在每次看到柳芷，她都用极其幽怨的眼神看着自己，看得程大雷毛骨悚然。
<br/>　　直到这一日，程大雷收到情报，说有大批流民涌向蛤蟆城。
<br/>　　程大雷与徐神机几人着急来到城门上，远远观去，只将一伙人聚在城门口，口中大声嚷嚷着什么。
<br/>　　“让我们进去，我们要见程当家，不让我们见程当家，我们就砸破城门。”
<br/>　　程大雷立在城头，问身边的阿喜：“这个……什么情况？”
<br/>　　“早就得到情报，有大批江湖人士涌入凉州，刚刚查明，他们似乎是想要投靠大当家您。”
<br/>　　“投靠我？”
<br/>　　“这只是一小部分，目前还有大批人涌来，他们都是同一个目的，要在蛤蟆城落草。”
<br/>　　程大雷远远望去，只见辽阔的大地上，一个人一个人如小黑点一般移动，最后汇成河流，涌向蛤蟆城。
<br/>　　“大当家，要不要放他们进来？”刘悲问。
<br/>　　程大雷渐渐有些明白过来，广播技能虽然让自己恶名满天下，为仁人志士所不耻，可同样也吸引了自己的同类人。
<br/>　　或者，他们自以为是自己的同类人。
<br/>　　这些人包括江湖上的亡命徒，占山为王的山大王，无计可施的水贼海盗，当然，也有大批活不下去，怒而起义的反贼。
<br/>　　既然想要落草做山贼，就找个名声大的，背景厚的做靠山。理所应当，程大雷是第一选择。
<br/>(本章完)
