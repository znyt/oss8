　　天王宫，一间极为隐秘的密室之前，楚天墟静静的站立在密室的门前。
<br/>　　深深的吸了口气，他轻轻的敲了敲密室的石门。
<br/>　　轰隆隆！！
<br/>　　随后，在一阵轰隆声之中，密室的石门大开了，露出了其内盘膝而坐的紫霄。
<br/>　　“什么事？”紫霄双眸紧闭，沉声问道。显然，他并不知晓楚天墟被擒之事，或者说天王宫并没有通知。
<br/>　　“宫主，今日我被殿主擒了。”楚天墟走进密室之中，声音低沉的说道。
<br/>　　“什么？”紫霄闻言，顿时睁开了双眸，眸中闪过锋锐的光芒。
<br/>　　“他在我识海中下了死符，让我引你出天王宫！”楚天墟继续道，他并未隐瞒紫霄，随即将今日之事全部交代了出来，包括苏莫的参与。
<br/>　　他虽然会背叛终极神王殿，但是，却不会背叛紫霄，因为对方是他的师尊。
<br/>　　“死符？”紫霄低声自喃，脸色瞬间阴沉了下来，他自然知道终极神王的死符是什么。
<br/>　　“师尊，天墟的生死不重要，只要您安全就好！”楚天墟单膝跪地，向紫霄恭敬的说道。
<br/>　　楚天墟心中非常清楚，无论他引不引出紫霄，他都会被终极神王杀死。
<br/>　　所以，他想要活命，只有跟着紫霄，一路走到黑。
<br/>　　“你放心，我绝对不会让你死！”紫霄沉声道，随即，他立刻起身向外走去，道：“走，我们去找天君河！”
<br/>　　而后，紫霄带着楚天墟，前去寻找天王宫宫主天君河。
<br/>　　在下人的带领之下，紫霄和楚天墟来到天王宫的议事大殿。
<br/>　　大殿中，不仅有天君河，帝一魂、天尘宇、天罡雷、天黄神王等天王宫的高层均在。
<br/>　　“紫霄，见过天宫主，以及各位长老！”紫霄走进大殿，向天君河以及众位天王宫的长老行礼。
<br/>　　而一众天王宫之人，目光却是落在了楚天墟的身上，显然都知道了之前发生的事情，现在见到楚天墟回来，不由得都有些诧异。
<br/>　　“紫霄，你来的正好！”
<br/>　　天君河微微点头，道：“楚天被终极神王所擒，我们正在商议如何营救，没想他到已经回来了。
<br/>　　“多谢天宫主了！”紫霄再次抱拳，心中却是冷笑，营救吗？是在商议怎么对付苏莫吧？
<br/>　　略一停顿，紫霄继续道：“终极神王放了他，在他身上种下了死符，想要要挟他害我，不知天宫主可有破解死符之法？”
<br/>　　“死符？”众人闻言，顿时眼眸一凝，这可是一种极为狠毒的秘法，又是由终极神王施展，想要破解几乎不可能。
<br/>　　一时间，所有人都沉默了，他们根本无能为力。
<br/>　　“诸位，我们现在可是同仇敌忾，拥有苏莫这个共同的敌人，你们务必相帮！”紫霄郑重的说道，楚天墟是他一手培养起来的，忠心耿耿，天赋很强，几乎堪比万古天骄了，他会尽力相救。
<br/>　　“紫霄啊！不是我们不救，这死符你自己应该了解，想要破解机会渺茫！”天君河叹了口气道。
<br/>　　楚天墟闻言，眸中闪过一道灰暗的光芒，真的无法破解了吗？若是就这样死了，他真的很不甘心。
<br/>　　“我可以救他！”就在此刻，一道淡淡的声音，突然在寂静的大殿中响起，显得格外的突兀。
<br/>　　随即，众人纷纷转头，看向说话之人，正是坐在天君河坐下首的帝一魂。
<br/>　　楚天墟心中一震，目光露出希冀的光芒，第一时间看向了帝一魂。
<br/>　　“帝兄，你真的有办法？”紫霄惊讶的望着帝一魂，他只是想要天王宫尽力一试，没想到帝一魂居然有办法。
<br/>　　“有，轻易可破！”帝一魂面色平静，古井无波，仿佛死符在他眼中，就是普通的神通一般，根本不足为虑。
<br/>　　这不仅让紫霄惊讶，就连天王宫的一众强者，都是微微惊讶了起来。
<br/>　　“请前辈施以援手，晚辈感激不尽！”楚天墟立刻向帝一魂抱拳，面色恭敬至极，他毫不怀疑帝一魂的话，对方乃是轮回之身，手段不是一般神王可以比拟的。
<br/>　　“楚天，只要你答应我一件事，我便可以帮你解除死符。”帝一魂凌厉的目光，落在了楚天墟的身上。
<br/>　　“前辈请说。”楚天墟郑重的说道，只要能解除死符，别说一件事，就是十件事那也是义不容辞。
<br/>　　“紫霄如此保你，想必你的天赋和实力不会差，天王宫的千选战名额给你一个，你唯一的任务，就是擒拿苏莫，或者斩杀苏莫。”帝一魂冷漠的说道。
<br/>　　众人闻言，顿时眼眸一亮，他们找看出来了，这楚天不简单，若是能代天王宫出战，也算是一大助力。
<br/>　　“前辈放心，晚辈定能完成任务！”楚天墟立刻抱拳，郑重的保证道，不管能不能完成任务，先将死符解除才是当务之急。
<br/>　　至于苏莫，就算帝一魂没有吩咐，只要是他遇到苏莫，也绝不会留手。
<br/>　　“好，那我便救你！”帝一魂微微点了点头。
<br/>　　“呵呵，帝兄果然非凡，我紫霄欠你一个人情！”紫霄的脸上露出了微笑。
<br/>　　“紫霄兄客气了，确如你所说，我们现在是同仇敌忾！”帝一魂微微一笑，随即缓缓起身，道：“楚天，跟我来！”
<br/>　　言罢，帝一魂转身大步离去，虽然他说的轻松，但是，想要破解死符，还需要废一番功夫。
<br/>　　很快，楚天墟便跟着帝一魂离开了，大殿中的众人并未散去。
<br/>　　“紫霄，坐吧！”天君河挥了挥手，示意紫霄就坐，继续道：“今日终极神王和苏莫一起出现，当真是奇怪，你有什么看法？”
<br/>　　“他们两人会和平的相处在一起，想必是其中一人妥协了！”紫霄的脸色郑重无比，他之前也在思索，苏莫怎么会和终极神王在一起，这完全不符合常理。
<br/>　　难道说，苏莫已经将终极神剑和终极剑道的秘密交于了终极神王？但是，苏莫有龙族这个虎皮，怎么会妥协的？
<br/>　　而且，就算苏莫妥协了，终极神王为何还和苏莫走的这么近？两人达成了什么条件？
<br/>　　“可能是这样！”
<br/>　　天君河点了点头，继续道：“所以，苏莫的势力，现在是更加的强大了，不仅有龙族，还有终极神王殿，想要对付他更加的困难了？”
<br/>　　“天宫主，苏莫目前不在龙都城，就在轮回神朝，难道你们不行动吗？”紫霄皱眉问道，这可是一个千载难逢的机会。
<br/>　　“之前我们也在商谈，最终还是决定暂不出手了！”天君河沉声道。
<br/>　　“为何？”紫霄不解，这么一个机会，居然就这样白白错过了。
<br/>　　“苏莫心思极为缜密，他既然敢来轮回神朝，又敢现身，想必是底气十足，而且，现在估计早已经走了，还是待千选战之时一劳永逸吧！”天君河道，他算是很了解苏莫的性格，现在想要再找到苏莫，怕是不可能了。
<br/>　　“也是，那就在千选战之时，在彻底解决吧！”紫霄点了点头，他早已经知道，帝一魂已经拉拢了不少大势力，千选战，就是苏莫的坟墓。
<br/>　　虽然终极神剑可能已经不在苏莫身上了，但是，终极剑道的秘密，苏莫肯定还是知道的。
<br/>　　所以，在千选战之时擒拿苏莫，他就还有机会得到终极剑道的秘密。
<br/>　　随后，众人又商谈了一番千选战的诸多事宜，便各自而散，纷纷离去。
