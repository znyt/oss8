　　灵韵一边说着，一边亲自斟了一杯酒，奉到大师兄跟前。
<br/>　　大师兄顿时大喜过望，连忙双手接过，笑道：
<br/>　　“灵韵师妹客气了！你我既是同门，你家中有难，我作为大师兄，自然不能不管不顾！”
<br/>　　不料，
<br/>　　大师兄这话才刚刚说完，
<br/>　　咚咚咚！
<br/>　　一名城主府的下属，急匆匆从外面冲了进来，
<br/>　　“城主大人，不好了！邪光宗的人又来了！”
<br/>　　“什么！”
<br/>　　所有人除了叶尘之后，全都脸色大变，纷纷从座位上站了起来。
<br/>　　片刻之后，
<br/>　　一道阴测测的声音，陡然从外面传来，
<br/>　　“灵山！老朋友来了，你还不快点出门迎接？”
<br/>　　砰！
<br/>　　黑岩城城主手掌在桌子上一拍，脸色十分凝重，道：
<br/>　　“这是邪光宗宗主墨化风的声音，没想到连他都来了，看来这次，来着不善啊！”
<br/>　　这时，
<br/>　　大师兄站了出来，淡然一笑道：
<br/>　　“城主莫忧！他们来的正好，我正要去会一会这邪光宗，倒是省的去找他们了！”
<br/>　　说完这话，大师兄一马当先，缓步朝外面走去。
<br/>　　黑岩城的众人见状，顿时纷纷心头大定，
<br/>　　“这位柳公子，乃是飞雪宗宗主的首席大弟子，将来飞雪宗宗主的继承人，有他出面，想那邪光宗，也不敢放肆！”
<br/>　　“飞雪宗可是咱们巨门星域的一流势力，小小的一个邪光宗，又能翻出什么浪了？”
<br/>　　“还是多亏了大小姐，及时请来了柳公子，这下看那邪光宗还如何嚣张！”
<br/>　　……
<br/>　　众人一边随着大师兄，朝门外走去，一边开始兴奋的议论起来，个个趾高气扬，仿佛已经是胜券在握。
<br/>　　“叶尘哥哥，要不我们也出去看看吧？”
<br/>　　等到众人全都起身之后，灵溪也不由得眨了眨眼睛，冲着一旁的叶尘开口道。
<br/>　　“我就不……”
<br/>　　叶尘觉得索然无趣，原本想拒绝，却陡然感应城主府外，有这一股颇为不弱的气息，顿时眉头微微一皱。
<br/>　　‘看来那个邪光宗，应该也是有备而来，他们双方谁能笑到最后，还真未可知！旁人的生死，我自然懒得过问，可是这小丫头，是真心待我，我定要护她周全才是！’
<br/>　　一念及此，叶尘已经到嘴边的话，硬生生收了回去，冲着灵溪点了点头，
<br/>　　“好吧，我也去看看！”
<br/>　　……
<br/>　　很快，
<br/>　　众人走出城主府，
<br/>　　此时城主府外，聚集了一群人，大多数金丹境的气息，还有几位气息浑厚，应该是元婴境的高手。
<br/>　　而且这些人身上，都透着一股邪气，一看就知道不是什么善良之辈。
<br/>　　“桀桀！灵山，你旁边的这位大美人，想必应该就是你的那位宝贝千金，号称是黑岩城第一美女的灵大小姐吧？今日一见，果然是名不虚传啊！”
<br/>　　说话的，是一名身穿黑色长袍的中年男子，说话的同时，一双贼眼肆无忌惮的在灵韵那妙曼的身姿上打量着，满脸的淫秽之色。
<br/>　　黑岩城城主灵山，听到这话，顿时勃然大怒，
<br/>　　“墨化风！你好大的胆子，竟然敢明目张胆的来我黑岩城，你是想找死吗？”
<br/>　　原来这中年男子，便是邪光宗的宗主墨化风！
<br/>　　墨化风桀桀一笑，脸上露出一抹嘲弄之色，道：
<br/>　　“得了吧！灵山，就凭你们黑岩城这点人马，也想和我们邪光宗作对，你也不看看自己有几斤几两！”
<br/>　　“本宗主今天还就把话撂在这儿，要么交出十万块上品灵石，否则本宗主今天，就灭了你们黑岩城！”
<br/>　　轰！
<br/>　　墨化风这话一出口，整个黑岩城的众高手，一个个顿时全都炸毛了。
<br/>　　十万块上品灵石！
<br/>　　黑岩城不过就是一个二流势力，而地处偏僻，就算把全城的灵石都搜集起来，恐怕也没有十万块上品灵石啊！
<br/>　　这个邪光宗，可真是越来越肆无忌惮了！
<br/>　　一时间，所有人纷纷望向大师兄。
<br/>　　“呵呵！”
<br/>　　大师兄感受到黑岩城众人期盼的目光，顿时微微一笑，缓缓走上前来，
<br/>　　“墨宗主是吧？你可真是好大的口气啊！”
<br/>　　墨化风看到眼前这个，气质不凡的英俊青年，不由得眉头微微一皱，
<br/>　　“你是什么人？莫非想要插手我们邪光宗和黑岩城的私事吗？”
<br/>　　大师兄淡淡一笑，
<br/>　　“你们两派的私事，本公子自然不会插手，可是你们邪光宗，如此仗势欺人，我飞雪宗可不能不管！”
<br/>　　“什么？飞雪宗！”
<br/>　　邪光宗众人，听到这话，顿时全都为之一惊。
<br/>　　要知道，
<br/>　　飞雪宗是巨门星域的一流势力，门中更有化神天君坐镇，可不是他们区区一个邪光宗可以招惹的。
<br/>　　“哼！”
<br/>　　不过那墨化风微微一惊之后，当即冷哼一声道：
<br/>　　“小子，我看你年纪不大，最多也就是飞雪宗的三代弟子吧？难道你还能代表整个飞雪宗不成？”
<br/>　　“哈哈哈！”
<br/>　　墨化风声音落下，灵山等人顿时纷纷大笑，
<br/>　　“墨化风，竖起你的耳朵，给我听好了！这位乃是飞雪宗首席大弟子柳云飞！你说他能不呢代表飞雪宗呢？呵呵！”
<br/>　　“什么！”
<br/>　　“他就是飞雪宗的那位绝顶天才柳云飞？据说他二十五岁便突破元婴境，实力深不可测！”
<br/>　　“而且还是飞雪宗宗主最得意的大弟子，下任飞雪宗的不二人选！”
<br/>　　邪风宗众高手，听到柳云飞的名头，顿时纷纷脸色大变，开始低声议论起来。
<br/>　　听到众人的议论声，柳云飞顿时更加得意，冷冷瞥了一眼邪风宗的众人，淡淡道：
<br/>　　“给你们十息时间，速速离开黑岩城！否则休怪本公子不客气！”
<br/>　　柳云飞俯视着邪风宗的众人，犹如俯视蝼蚁，平淡的语气中，却透着一股不容置疑的威严。
<br/>　　黑岩城众人见柳云飞开口，顿时全都精神大震，就连灵韵的美眸之中，也露出一抹崇拜之色。
<br/>　　不愧是飞雪宗首席大弟子，说话的口气就是不一样！
<br/>　　邪光宗众人，则个个脸色难看，纷纷望向宗主墨化风。
<br/>　　不料，
<br/>　　那墨化风似乎并没有太把柳云飞放在心上，两眼微微一眯，道：
<br/>　　“柳公子这么做，未免有些太霸道了吧？”
<br/>　　“嗯？”
<br/>　　柳云飞顿时眉头一皱，显然没有想到，这个墨化风竟然敢违逆自己！
<br/>　　“找死！”
<br/>　　柳云飞显然动了怒气，直接双指作剑，手臂一起一落，
<br/>　　轰！
<br/>　　一道长达数百米的青色剑芒，在空中划过一道优美的弧线，继而朝墨化风狠狠劈落下去！
