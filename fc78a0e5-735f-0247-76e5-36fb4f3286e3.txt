        &lt;h3&gt;第五十四章  弥清云偶遇宝宝&lt;/h3&gt;
<br/>
<br/>        弥青云也一直想知道陈子昂的情况，有了冯睿这个传话筒，她们省事了很多。
<br/>
<br/>        前几年一直都不知道陈子昂的下落，却没有想到陈子昂一直都在他们的眼皮子底下生活，看来年轻人还是健忘，俩个人这么些年来也没有什么往来，似乎都是忘记了彼此。
<br/>
<br/>        这几年张函和陆玉的感情不错，最起码她们是这样认为的，只是俩个人在一起生活了这么多年连个孩子都没有，这让弥青云心里多少还是有点犯堵。
<br/>
<br/>        她还是很希望抱个孙子的。
<br/>
<br/>        她倒是没有想到陈子昂这么多年以来一直都没有嫁人，当初那个孩子将钱扔下楼的时候，她就知道她不是一般的人，果不其然。
<br/>
<br/>        自从知道陈子昂在北京之后，她也有关注着她，看见她发展越来越好，心里也算是莫名轻松。前几年没有陈子昂的消息，她心里也一直是揪着的，毕竟和她儿子在一起呆了那么久，一个女孩子最美好的青春年华啊。
<br/>
<br/>        她和张沛尧的感情很好，老张家的人都是痴情种，爱上一个就抓住不放，她担心张函会一直和陆玉闹腾下去，没想到过了一段时间俩个人恩恩爱爱的，张函竟然经常开车接送陆玉，而且还浪漫的给陆玉送花。
<br/>
<br/>        到底是年轻，也还是爱得不够深，时间长了就会忘记之前的事情。
<br/>
<br/>        俩个人拿着冯睿送的东西喜笑颜开的离开了。
<br/>
<br/>        “少芬啊，我还想抱孙子呢，你们倒好，将玉儿送走了。”
<br/>
<br/>        “着什么急？他们还年轻，不让他们干一番事业怎么成？”蔡少芬嘴里这样说着，心里却说“你这辈子怕是没有孙子了，你儿子不行。”
<br/>
<br/>        “唉，你们啊，就光想着事业，也不想着俩个孩子多大年龄了。”弥青云要了摇头，年龄大了要孩子会有危险的。
<br/>
<br/>        “年轻人嘛，你不让他们闹够怎么办？”
<br/>
<br/>        “也是。”
<br/>
<br/>        “冯睿这次来是想让我们帮忙呢。”
<br/>
<br/>        “你想帮我不拦着你，我呢......和她没有什么关系。”
<br/>
<br/>        “少芬，你可是拿了人家不少好处的哦。”
<br/>
<br/>        “拿了什么？”
<br/>
<br/>        “你手中的东西怕也值个几十万吧？冯睿不懂，你可是很懂的哦。”
<br/>
<br/>        “好了，好了，我给二弟说一声，你让冯睿自己去找二弟就是。再说，冯睿本身就在内部工作，偶然也是能见到二弟的吧？”
<br/>
<br/>        “哎，我说青云啊，你这对陈家人够好的了啊。不沾亲带故的。”蔡少芬有些不满。
<br/>
<br/>        “我这不是要掌握陈家那小蹄子的信息嘛。再怎么说，也和小函有那么一段经历。”
<br/>
<br/>        “怕什么，她要是再敢和张函来往就不要在北京混下去了。这都多少年了？难不成还能旧情复燃不成？”蔡少芬站在来接她的车门口看着弥青云。
<br/>
<br/>        “不可能了，他们之间已经没有任何瓜葛了，小函都不知道她在北京呢。”
<br/>
<br/>        “那就行了，别想着那些有的无的，一惊一乍的，看在手中东西的份上，就帮一下冯睿吧。我走了啊，改天联系。”蔡少芬一屁股坐进车里。
<br/>
<br/>        弥青云看着汽车绝尘而去，深深的叹了口气。
<br/>
<br/>        蔡少芬这几年是越发的托大了。
<br/>
<br/>        对她说话都是颐指气使的，而，陆玉在家里就是一个“爷”，家里虽然是雇有保姆，但是保姆是不可能给她洗内衣内裤的，她弥青云沦落到给儿媳妇洗内衣内裤的地步了。
<br/>
<br/>        陆玉在家里的这几年，他们都不敢多说话，怕一个不小心就惹得陆玉不开心，整个家里冷冰冰的。
<br/>
<br/>        且，陆玉见不得她和张沛尧俩个人亲亲蜜蜜的说话。
<br/>
<br/>        毕竟北京这地方，寸土寸金，他们还是要住在一个地方的，按照他们这几年的努力怎么能像陆家那样能到处买房产？所以一家人还是住在一个四合院里的，抬头不见低头见的。
<br/>
<br/>        张家二爷是主事之人，张家这几年一直低调行事，二爷对世事不闻不问，所以张家只能靠着陆家这棵大树。
<br/>
<br/>        弥青云叹了口气，今天周末，不上班，就当是散散心，排遣一下郁闷的心情。
<br/>
<br/>        这几年过得小心翼翼的，她倒是很想念在西城的日子，和张沛尧俩个人下班了有说有笑，日子也算是过得幸福，虽然儿子和他父亲关系不怎么好，但是正常的情况下，一家人还是和和美美的，哪像现在的这种日子啊，张沛尧从回到北京之后，一直忙着工作，而且回家都很晚，儿子也一样，家里就冷冷清清的。
<br/>
<br/>        有时候她都不想回家。
<br/>
<br/>        得知陆玉要去甸城工作，而且工作几年，她竟然心里莫名的轻松了很多。
<br/>
<br/>        陆玉一走，家里是不是会好很多？
<br/>
<br/>        她一个人漫无目的的走着，也不知道走了多久，突然被一个童声童气的声音打断了思绪。
<br/>
<br/>        “奶奶，你的东西掉了。”
<br/>
<br/>        一个小学生模样的孩子将弥青云掉在地上的字画捡起来交给弥青云。
<br/>
<br/>        “谢谢你，小朋友。”弥青云都不知道自己掉了东西。
<br/>
<br/>        “不客气，奶奶，你要将东西拿好，别再丢了哦。”小男孩眉眼俊秀，可爱至极。
<br/>
<br/>        “哥哥，人家明明是阿姨嘛，你非要叫人家奶奶。”从旁边又跑过来一个小女孩，小女孩扎着的马尾欢快的摇摆着。
<br/>
<br/>        “阿姨好！”小女孩调皮的一笑，漏出一排整齐洁白的牙齿，小小年纪笑容灿若星辰。
<br/>
<br/>        弥青云看着俩个孩子心情莫名其妙的的好了很多。
<br/>
<br/>        “谢谢小朋友哦。”弥青云看着俩个孩子似曾相识，好似再哪里见过。
<br/>
<br/>        “团团，嘟嘟，你们在干嘛呢？快过来，一会要迟到哦。”一个老人家在那边喊着俩个孩子。
<br/>
<br/>        “奶奶再见，阿姨再见！”小男孩一本正经，小女孩淘气可爱。
<br/>
<br/>        “团团，嘟嘟。”弥青云嘴里念叨着俩个孩子的名字，真是可爱呢，连名字都起得这么可爱，也不知道是谁家的孩子。
<br/>
<br/>        “唉，小函要是有这么可爱的俩个孩子就好了。”弥青云又想到了她的儿子张函。
<br/>
<br/>        张函和陆玉结婚这么久，连个影子都没有，也不知道俩个人是谁的原因，这陆玉一走，这几年怕是没有希望了。
<br/>
<br/>        陆玉走就走吧，走了他们家也会轻松很多，至于和张函有没有孩子，也不是她能着急就着急得来的。
<br/>
<br/>        最起码，她弥青云不用给陆玉洗内衣内裤了！
<br/>
<br/>        弥青云看着俩个孩子远去的影子，怔怔的看了一会，眼里是隐藏不住的羡慕。
<br/>
<br/>        这两个孩子怎么看着这么熟悉呢？是谁家的孩子？好像身边也没有人有这么大的孩子啊，而且这俩个孩子好似双胞胎。
<br/>
<br/>        也不知道谁家这么有福气，一胎生俩，还是一儿一女，太幸福了。
<br/>
<br/>        弥青云心里的郁闷一扫而空，取而代之的是满心的羡慕，他们家什么要是有这样两个孩子就好了，到时候家里不知道会有多欢乐。
