        &lt;h3&gt;第三卷第三百九十五章  欢送会&lt;/h3&gt;
<br/>
<br/>        宁雅在办理离职手术，她将手中的工作一一交接，原本是要给田星交接的，谁知道要交接给钱喵喵。
<br/>
<br/>        钱喵喵从运营部突然调到他们这个部门，她害怕有些东西钱喵喵能不能干好。
<br/>
<br/>        她准备去找陈子昂的时候，发现陈子昂的办公室没有人。
<br/>
<br/>        “思宇，你见陈总了吗？”宁雅看见路过的陈思宇，拦着陈思宇问。
<br/>
<br/>        宁雅是办公室主任，按说陈子昂的行程她都知道，早上到现在就没有看见陈子昂。
<br/>
<br/>        “陈总.......”陈思宇本来说陈总去欧洲了，但是想到宁雅这个办公室主任都不知道陈子昂的行程，她就住口了。“我昨天见的陈总，今天没有看见，怎么陈总没有来吗？”
<br/>
<br/>        “没有呢，到现在都没有看见她，也不知道去哪里了，昨天也没有应酬啊。”宁雅叹了口气回到办公室，打开电脑先看看有没有要处理的邮件，等一会陈子昂来了再说。
<br/>
<br/>        宁雅打开邮箱发现有陈子昂发给她的邮件：“小雅，当你打开这封邮件的时候，我已经到了欧洲。因为有事情要紧急处理，我来不及通知你，也来不及让你给我订机票。工作上的事情你和钱喵喵进行交接，让田星继续主持，办公室的事情钱喵喵为正，田星为副，钱喵喵已经在我们公司工作了很久，是时候让他有个成长的机会，他的父母亲都是人民教师，以后钱喵喵有可能会继承他父母亲的工作，认识一些人对他有好处，如果时间来得及的话，你公司开业我一定会回来的。祝安好！”
<br/>
<br/>        宁雅不知道陈子昂为什么急急匆匆去了欧洲，难道说有什么新业务？
<br/>
<br/>        宁雅也不去想了，既然陈子昂已经做了决定，她也不好再说什么，毕竟陈子昂有自己的安排。
<br/>
<br/>        宁雅将工作进行了交接，准备离开，心里又多了很多的不舍，在这个公司她学到了很多东西，也感受到了这个公司不一样的文化氛围，同事之间的相亲相爱，相互理解和鼓励，大家拧着一股绳朝前努力着，每一个人都在尽心尽力，尤其是陈子昂给了她信心。
<br/>
<br/>        “小雅。”李凌菲走过来叫宁雅。
<br/>
<br/>        “凌菲，我要离开这里了，你们保重！”万分不舍啊。
<br/>
<br/>        “小雅，首先祝贺你开启自己的人生之旅，祝你好运！今天你要离开公司，我们为你开了一个欢送会，参加完欢送会，我们一起狂欢，然后送你回家！”李凌菲微笑着看着宁雅吃惊的样子。
<br/>
<br/>        “还开欢送会，我怎么不知道？”
<br/>
<br/>        “请吧，工会请。”李凌菲亲昵的挽着宁雅的胳膊朝工会走去。
<br/>
<br/>        当宁雅和李凌菲走进工会会议室的时候，工会里响起了掌上：“欢迎我们的老同事新朋友宁总光临！”皇甫双双大声说。
<br/>
<br/>        “大家应该知道，宁总是我们的前办公室主任，为了我们公司鞠躬尽瘁付出了很多的心血，现在宁总有了自己的事业，我们恭送宁总恭喜宁总！”
<br/>
<br/>        “恭送宁总，恭喜宁总！”大家齐声呐喊。
<br/>
<br/>        “我还没有走呢，你们就迫不及待的让我走，我不想走了。”宁雅没有想到，同事们都齐聚会一堂为她举行欢送仪式。
<br/>
<br/>        “那就不走了，留下来我们一起奋斗不香吗？”皇甫双双大声说。
<br/>
<br/>        “香，很香！”宁雅走到主持台前，深深的给大家鞠个一躬。
<br/>
<br/>        “感谢大家这几年的支持和帮助，宁雅在这里感受到了家的温暖，亲人之间的关怀，谢谢大家这几年对我的包容和鼓励，能和大家一起共事是宁雅一生的幸运。”宁雅声音哽咽：
<br/>
<br/>        “虽然我走了，但是我的心永远和大家在一起，只要公司有召唤宁雅一定会回来的。”
<br/>
<br/>        “每一个从这里进来的人都是我们的亲人，每一个从这里走出去的人也都是我们的亲人，无论你们去向哪里，请记住，这里是你们的避风港，我们会和你们永远在一起！”皇甫双双拿起话筒对大家说。
<br/>
<br/>        “今天陈总和江总有事情不能参加宁总的欢送会，他们分别录了视频给宁总，我们现在有请播放陈总的视频。”
<br/>
<br/>        “小雅，因为有事情，没有能参加你的欢送会，很抱歉！希望你事业有成，红红火火，得到你想得到了，爱上你爱上的，加油！”
<br/>
<br/>        “现在播放江总的视频。”
<br/>
<br/>        “宁雅，祝你的事业蒸蒸日上，改天我带着凌辰去你的单位，看看我们两个人哪里不得劲，给我们做个美容，让我们更帅一些，早日抱得美人归！加油！”
<br/>
<br/>        “赵绾绾和呼延寰宇也发了视频。”
<br/>
<br/>        “小雅，事业美美的，人也要美美的，等我回来去看美美的你的美美的事业，加油！”
<br/>
<br/>        “宁总，祝事业一帆风顺，蒸蒸日上，回来一起喝酒！加油！”
<br/>
<br/>        “宁总，各个分部的人也都发来了祝福词，我都放给你看。”
<br/>
<br/>        每个人一句祝福词，看得宁雅唏嘘不已，她都不知道公司还有欢送会，也不知道是对她一个人了，还是大家都一样。
<br/>
<br/>        “焕焕，以前公司也开欢送会吗？”宁雅低声问身旁的谷焕焕。
<br/>
<br/>        “开呀！我们公司只要是正常离职的，或者是去外地的同事，都会开欢送会的。”谷焕焕看得也是心潮澎湃。
<br/>
<br/>        “可是这两年我都没有发现有开欢送会的。”
<br/>
<br/>        “因为这两年离职的人很少。我们公司一直发展的很好，基本上没有人离职。而且现在分部的人员分布也均匀，也不用大批量调动，所以就很少开欢送会的。”
<br/>
<br/>        “我们公司的氛围真的很好，我真的都舍不得离开了。”
<br/>
<br/>        “没事啊！你想回来还是可以回来的，这里永远都会欢迎你。”谷焕焕轻声细语的对宁雅说：
<br/>
<br/>        “我是不会离开公司的，公司给予我平台，给我机会，没有这个公司，我可能就是一个小职员，感受不到工作的幸福和快乐，所以，只要公司存在一天，我都会在这里。”
<br/>
<br/>        “焕焕你真棒！”宁雅笑着说。
<br/>
<br/>        “宁雅姐，我等一会送你一首歌：铿锵玫瑰！祝你事业有成！越来越美丽！”谷焕焕看着屏幕，他们的这个欢送会实际上就是一台晚会，大家齐聚一堂，各自展现着自己的才艺。
<br/>
<br/>        这会是倪小美和苟江凌的相声小品，下一个就是谷焕焕的“铿锵玫瑰！”
<br/>
<br/>        最后是由倪小美的“难忘今宵”作为结束。
<br/>
<br/>        “人生路长，我们一直都在，宁总，欢迎经常回家！”
<br/>
<br/>        宁雅以一种欢快而感动的方式离开了陈子昂的公司投入到自己的事业中，人生路上，我们一直都在！
