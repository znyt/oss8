　　神武学府门前，人声鼎沸，大量的神武学府弟子陆续到来。
<br/>　　待到朝阳升空之时，已经聚集了不下六千七人。
<br/>　　这些人之中，只有一部分是想要获得真圣之气，从而在真圣碑留名，另外一部分纯粹是凑热闹的人。
<br/>　　真圣界开启，这可是古灵星河年轻一代的盛会，一般无事的人，都会前往观看。
<br/>　　苏莫目光转动，打量着人群，发现神武学府的阵容的确不弱。
<br/>　　武帝境九重修为之人，足有两百多人，而其中有些人，观其气息甚至不比尤天韩弱多少。
<br/>　　众人一边闲聊一边等待，待到日上三竿之时，学府门前聚集的人已经接近了万人。
<br/>　　而这个时候，从学府内走出了一群导师人物，这群导师足有十五、六人，为首的是两名中年导师。
<br/>　　这两人是一男一女，男的高大魁梧，面容粗犷，女子则正好相反，身材娇小，纤瘦无比。
<br/>　　两人气势极为内敛，给人一种深不可测之感。
<br/>　　“是吴晋导师和唐婉导师！”
<br/>　　“没想到是这两位强大的导师带队！”
<br/>　　“随行的导师真不少啊！
<br/>　　众人见此顿时眼眸一亮，这两位导师可都是内府导师，修为全部是高阶武尊。
<br/>　　苏莫也是眼眸一亮，因为随行的导师之中，居然还有他认识的木璃导师。
<br/>　　他倒是没想到，木璃导师居然也一同前往，
<br/>　　一众导师来到学府门外，那位吴晋导师和唐婉导师目光扫视众人，随即两人齐齐一挥手。
<br/>　　霎时之间，天空昏暗了下来，半空之上突兀的出现了两艘庞大的战船。
<br/>　　这两艘战船均是长达数百丈，通体宝光流转，气势摄人，给人巨大的压迫之感。
<br/>　　苏莫瞪大了眼眸，目光直直的盯着这两艘战船，这是什么？难道是赶路用的飞行宝器？
<br/>　　果然不出他的所料，就在此时，那名高大粗狂的吴晋导师朗声高喝了起来。
<br/>　　“所有要去真圣界的弟子，全部上舰！”
<br/>　　吴晋导师浑厚的话音一落，近万弟子顿时行动了起来，一时间人影闪烁，破空之声不绝于耳。
<br/>　　一名名弟子飞身而起，飞到了两艘战舰之上。
<br/>　　“苏兄，我们也走吧！”谢彬招呼苏莫一声，便和同一派系的众弟子一起，飞向其中一艘战船。
<br/>　　苏莫没有犹豫，跟随着谢彬等人一起，飞到了其中一艘战船之上。
<br/>　　少倾之后，近万名弟子全部登上了战船，两艘战船上熙熙攘攘，挤满了人。
<br/>　　这时，吴晋导师和唐婉导师，各自带着数名导师，分开飞到了两艘战船上。
<br/>　　随后，两位导师分别催动了各自所在战船上的阵法，整个战船被包裹在一片光幕之中。
<br/>　　下一息，两艘战船嗖一声，直冲九天，瞬间便穿过了古周星的九天罡风层，进入了浩瀚无尽的星空之中消失不见。
<br/>　　苏莫所在的战船，是由吴晋导师操控和率领，速度简直快到极致。
<br/>　　在他的视线之中，战船之外是一片虚无，彻底进入了虚空深处，极致的速度与恐怖的空间气流冲击在一起，让得战船外的光幕爆射出刺目的火光。
<br/>　　这种速度，可谓是瞬息千万里，就算比不上大祭司的空间跳跃，也相差不远了。
<br/>　　转头看去，苏莫已经看不到另外一艘战船了，唐婉导师带领着木璃导师等人，乘坐另一艘战船。
<br/>　　两艘战船虽然是一同前往，但在此种速度之下，即便是一丝一毫之差，都相隔不知道多少里的距离。
<br/>　　“真是惊人！”苏莫目露惊叹之色，啧啧称奇。
<br/>　　“苏兄，你没有见过这种战船吧？”谢彬见苏莫满脸惊奇的模样，笑着问道。
<br/>　　“嗯，从未见过！”苏莫点了点头。
<br/>　　“这种战船名为虚行舰，整个神武学府之内，也就火元圣师前辈能炼制出来！”谢彬解释道。
<br/>　　“哦，原来是火元前辈炼制而出！”苏莫心中恍然，这种战船如此不凡，不是圣级炼器师的确难以炼制出来。
<br/>　　“是啊，这种虚行舰可是相当于圣器，能爆发出一般武圣境初期强者的速度！”谢彬沉声道。
<br/>　　“真是厉害！”苏莫感慨，他若是能弄一艘这种虚行舰，畅游星空，驰骋天荒星域都不成问题了。
<br/>　　当然，以他的实力很难保住这种战船，怕是很容易被别人抢走。
<br/>　　从古周星到真圣界所在之地非常的远，即便是有虚行舰赶路，也要三、四天的功夫。
<br/>　　苏莫和谢彬、任云都等人闲聊了良久之后，便进入了船舱中，盘膝而坐，打坐修炼。
<br/>　　自从突破了修为了之后，他一直在炼制本命灵剑，以及演戏新的本命灵剑阵，都没有好好的巩固修为。
<br/>　　现在，则正好利用赶路的时间，好好的巩固一番刚刚暴涨的修为。
<br/>　　时间在苏莫的静修之中飞快流逝，转眼间变过去了四天。
<br/>　　四天的时间，众人乘坐虚行舰，几乎跨越了接近小半个古灵星河。
<br/>　　某一刻，正在修炼中的速度，突然感觉到了强大的惯性，是虚行舰突然减速了。
<br/>　　他急忙稳住身形，立刻走出了船舱。
<br/>　　“快要到了！”
<br/>　　“终于要到真圣界了！”
<br/>　　“哈哈！这一次我定要扬名古灵星河！”
<br/>　　虚行舰的甲板上，聚集了大量的弟子，声音嘈杂一片，几乎所有的弟子都是满脸兴奋之色。
<br/>　　苏莫向外望去，发现他们已经来到了一颗星辰上空，这颗星辰非常的巨大，比之古周星都只大不小。
<br/>　　但是这却实一颗半废的星辰，灵气极为淡薄，其上生灵极少，到处都是荒芜的杂草。
<br/>　　此刻，虚行舰正滑翔在这颗半废星辰的上空，速度大降了下来。
<br/>　　“众位弟子，距离真圣界的入口，还有一个时辰的时间！”
<br/>　　吴晋导师高大的身躯立在最前方，朗声喝道。
<br/>　　众人闻言暗暗激动，眸中精光闪烁，战意澎湃。
<br/>　　苏莫亦是深深的吸了口气，他面临的挑战，也即将来临了。
<br/>　　少倾，他转头看向后方，发现唐婉导师操纵的另一艘虚行舰紧跟在后方，两艘战舰匀速前进。
<br/>　　“那是什么？”
<br/>　　就在此刻，一名弟子抬手指向侧方，满脸惊讶的大声问道。
