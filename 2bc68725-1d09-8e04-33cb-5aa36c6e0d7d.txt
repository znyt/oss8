　　“苏莫？”
<br/>　　“苏兄！”
<br/>　　“居然是苏莫！”
<br/>　　看清这道从巨坑中飞出的身影，一众神武学府的弟子，顿时纷纷惊呼出声。
<br/>　　因为这从巨坑中飞出的人影，不是别人，正是他们的同伴。
<br/>　　只见此刻的苏莫，非常的狼狈，披头散发，灰头土脸，全身衣衫褴褛。
<br/>　　不仅如此，苏莫的身上以及手臂上的皮肤和血肉，大面积的龟裂，有金色血液在不停的流淌而出，可以说是相当的凄惨。
<br/>　　所有人均是没有想到，这隐藏在暗中，几乎将血瞳族之人杀光的强者，居然是苏莫！
<br/>　　只有谢彬和苏莫较熟，之前就有所猜测，其他人则是没有注意苏莫。
<br/>　　众人心中震惊不已，苏莫这不过是突破了一层修为，达到了武帝境五重而已，居然能挡住武尊境的强者一击？
<br/>　　这怎么可能？
<br/>　　血瞳族老者虽然只是武尊境一重的修为，但再怎么说也是武尊，与武帝境武者不可同日而语。
<br/>　　刚才血瞳族老者那一击，足以轻易击杀普通的武帝境九重巅峰武者。
<br/>　　也就只有任云都这种战力超凡的武帝境九重武者，才能与其抗衡。
<br/>　　而苏莫，不过武帝境五重的修为，就算战力再逆天、再恐怖，也不可能挡住刚才那一击吧！
<br/>　　此人，众人已经不关心苏莫是如何隐藏身形的，均是被苏莫的实力震惊了。
<br/>　　那些神武学府的弟子，都知道苏莫是绝世天才，战力逆天无比，但是能挡住武尊境的尊者一击而不死，这也太夸张了吧。
<br/>　　此刻，不仅是神武学府的弟子，和黑甲士兵们震惊了，就连仅剩的两名血瞳族之人，和远处大战的血瞳族老者，以及任云都都震惊了。
<br/>　　任云都和血瞳族老者，不约而同的停止了战斗。
<br/>　　苏莫飞出巨坑之后，远远的望了血瞳族老者一眼，便向谢彬等神武学府的众人飞了过去。
<br/>　　只有和众人在一起，他才能保证安全，无惧血瞳族老者的攻击。
<br/>　　刚才一击，他虽然挡住了，但身受重创，连金刚之体和琉璃宝体这两种以防御著称的特殊体质，都抗不住巨大的力量，被震的险些彻底崩碎。
<br/>　　好在对方刚才那一击，距离很远，没有发挥出最强的攻击力，不然他怕是很难挡住。
<br/>　　他可谓是动用了三层防御，第一层就是自己的攻击，一剑将血色神剑斩击崩断。
<br/>　　然后又用盾牌防御住了绝大部分的攻击力，倾泄在他身上的力量，已经只有两成。
<br/>　　但即便只有两成攻击力，也将由琉璃宝体和金刚之体加固的超强肉身震裂了。
<br/>　　嗖！
<br/>　　瞬息之间，苏莫飞到了一众神武学府弟子的人群中。
<br/>　　“苏兄，你没事吧？”谢彬来到苏莫的身边，紧张的问道。
<br/>　　“没有大碍！”苏莫摇了摇头，随即吞服了几枚疗伤丹药。
<br/>　　“哈哈！苏兄，你太彪悍了！”
<br/>　　“是啊！轻易的就将百名血瞳族的人斩杀，真是大快人心！”
<br/>　　“该我们反击了！”
<br/>　　一众神武学府的弟子们哈哈大笑，之前他们一直被血瞳族的人压着打，现在血瞳族之人死的只剩下了三人，是该他们反击了。
<br/>　　那名血瞳族尊者，自有任云都牵制，他们只需对付剩下的两人便可。
<br/>　　“兄弟们，一起出手！”
<br/>　　其中一名弟子大喝一声，随即率先出手，一拳临空像那两名血瞳族之人轰去。
<br/>　　其他人也没有袖手旁观，纷纷出手了。
<br/>　　霎时之间，拳茫、剑气、刀气等等各式各样的攻击排山倒海，遮天蔽日，如同天地洪流一般，像那两名血瞳族之人杀去。
<br/>　　“这……！”
<br/>　　那两名血瞳族之人，虽然分别是武帝境八重和九重的修为，但看到如此多的攻击，亦是心中发寒，哪里还敢抵挡。
<br/>　　嗖嗖！！
<br/>　　两人毫不犹如，，身形急速暴退闪避，直接退到了极远之处，众人的攻击范围之外。
<br/>　　“可恶啊！”那名血瞳族的武尊境老者见此，怒不可竭。
<br/>　　一名小小的武帝境武者，不仅差点将他麾下的人杀光，居然硬挡了他一击还不死。
<br/>　　现在，他们则是彻底的败了！
<br/>　　这次来袭击环山矿脉，本以为环山矿脉的守将王副尉不在，他能轻易得手，但没想到会是这个结果。
<br/>　　“撤！”
<br/>　　一声大喝声响起，血瞳族老者急速向任云都猛攻两招，然后身形一闪，向那两名血瞳族的方向撤离而去。
<br/>　　他无法打败任云都，再继续战斗下去已经没有任何意义了，而现在浪费的时间已经太多了，若是黑山要塞的增援一到，他想走都难了。
<br/>　　“死！”
<br/>　　即便是要撤离了，血瞳族老者也还是想杀死苏莫，以报仇雪恨。
<br/>　　其飞行之中，眸光一转，眸中血光爆射，再次向苏莫击出了一击血脉神剑。
<br/>　　这一击血脉神剑，比刚才一击还要巨大，威势更甚两分。
<br/>　　血色神剑横贯天地之间，裹挟惊天威势，洞穿层层虚空，向苏莫所在的方向爆射而去。
<br/>　　“大家一起抵挡！”苏莫见此立刻大喝一声，随即一剑凌空斩向血色神剑。
<br/>　　“灭！”
<br/>　　“破！”
<br/>　　“杀！”
<br/>　　其他人纷纷再次出手，无数的攻击铺天盖地，纷纷击向血色神剑。
<br/>　　众人丝毫无惧，虽然他们的实力远不如武尊境的强者，但众人联手挡下一击不成问题。
<br/>　　轰隆隆！！
<br/>　　众多的攻击纷纷击中血色神剑，巨响震天，猛烈的爆炸震动八荒，逆卷九霄。
<br/>　　咻！
<br/>　　天穹之上一刀璀璨的金色刀气，长达十里，撕裂虚空，斜劈而下，同样狠狠的斩在了血色神剑之上。
<br/>　　是任云都出手了！
<br/>　　轰！
<br/>　　血色神剑彻底被击碎开来，化为漫天气劲，排山倒海的涌向八方。
<br/>　　轰隆隆！！
<br/>　　气浪席卷，一切为止破灭，空间崩塌，大地沉沦，烟消云散。
<br/>　　少倾之后，一切回归平静，方圆数万里之内全部被夷为了平地，还有一个恐怖的巨坑。
<br/>　　“大家都没事吧？”任云都回到了众人的前方，目光扫视众人，沉声问道。
<br/>　　“任大哥，我们没事！”
<br/>　　“受了点小伤，没有大碍！”
<br/>　　“不过，之前的战斗，我们陨落了十几人！”
<br/>　　众人纷纷开口说道，个个面色有些沉重。
<br/>　　任云都清点了一番人数，心中暗叹一声，他带出来一百零六人，现在只剩下了八十九人，足足陨落了十七人之多。
<br/>　　当然，那些镇守在矿脉的黑甲士兵，陨落的人更是多不胜数。
<br/>　　但黑甲士兵陨落多少，任云都不关心，他只在乎自己带出来的人手。
<br/>　　“苏莫，你还好吧？”任云都的目光落在了苏莫身上，眸光闪亮，这一次真的是多亏了苏莫。
<br/>　　不然的话，他们真的危险了，怕是会伤亡惨重。
<br/>　　“我没事！”苏莫摇了摇头，他虽然受了很重的伤，但以他的修为，并没有什么大碍！
<br/>　　“那就好！”
<br/>　　任云都点了点头，他正欲开口继续询问，就在此时，蓦地，一阵破空声从远处急速传来。
