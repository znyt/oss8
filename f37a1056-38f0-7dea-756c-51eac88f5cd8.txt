　　走进图书馆，映入眼中的就是一排排书架，不管是书架，或者是地面，都干净整洁，甚至可以说一尘不染。
<br/>　　“这里是学院重地，没有导师的允许，一般的学生都不能擅自进图书馆，书架上的书籍，记载着斗气大陆上的各种信息...”
<br/>　　若琳走在前面，边走边介绍，最后领着两人顺着楼梯来到二楼的一扇木门前，轻轻敲响木门，其中就传来一道沉稳的声音。
<br/>　　“进来。”
<br/>　　转过身来，若琳道：“你们进去吧，我有其他事情需要处理，待会我安排你们的住处，至于其他事情，就由霍德导师自己处理了。”
<br/>　　慕清霄微微点头，道：“多谢。”
<br/>　　若琳微微一笑，转身就向图书馆外走去。
<br/>　　待若琳导师离开，慕清霄推开木门，迈入宽敞明亮的房间中，其中书架交错纵横，书架上摆满了各种各样的古朴书籍，显得格外具有书香气息。
<br/>　　此时，在靠窗口边的书桌前，一名发须皆白的老者，正缓缓翻看着手中的资料，整个房间，都是处于一种寂静氛围。
<br/>　　老者看似普通，但体内却隐藏着一股极其恐怖的气息，慕清霄一眼就看出，后者的修为竟是斗皇巅峰。
<br/>　　整个外院，拥有斗皇巅峰实力的，恐怕也就只有外院副院长，琥乾了。
<br/>　　寂静的气氛，足足持续了十多分钟，只见琥乾将资料随手丢在桌上，锐利的目光扫了慕清霄与小医仙一眼。
<br/>　　“你们的信息，我看过了，天赋非常不错，嗯，与信息上一样，八星斗师，七星斗师，欢迎来到迦南学院。”
<br/>　　说话的同时，他不禁多看了一眼慕清霄，他总觉得，面前的青年有些古怪，但那里古怪，他又说不上来。
<br/>　　诧异过后，琥乾语气淡然，道：“学院招生要求虽简单，但入学后也需要通过选拔赛，如果连选拔赛都无法通过，那就注定与学院无缘，而你们的去与留，也需要通过选拔赛来决定。”
<br/>　　他的意思很明显在告诉两人，如果没有天赋，那么学院就不会挽留，有足够的天赋，那么学院就给予相应的舞台任由发挥。
<br/>　　“不过，选拔赛一年只有一次，今年的选拔赛已经在半个月前结束。”
<br/>　　话语刚落，隐晦的看了一眼两人的神色，见两人面色如常，琥乾眼中微微有些诧异，同时，微不可查的点了点头。
<br/>　　“但你们资质达到了内院的标准，所以学院决定破例让你们继续参加接下来的选拔赛。”
<br/>　　慕清霄微微点头，后者的一举一动如何瞒得过他，但他也清楚，这是在考验他们的心性，显然也合格了。
<br/>　　就在这时，房门再度敲响，平淡的语气自琥乾嘴中吐出：“进来。”
<br/>　　房门打开，只见六名穿着学院制服的学生相继走了进来，当他们看到已经有学生率先来到此后，目光都微微一愣。
<br/>　　慕清霄嘴角牵起一丝莫名的笑意，因为在这六名学生中，有一名可是老熟人。
<br/>　　走在最前面的女子，容貌倾城，优雅高贵，气质清冷淡然，犹如清莲初绽，身材窈窕动人，以至于走在她身后的男子，目光全都定格在她的身上。
<br/>　　女子不是别人，正是乌坦城萧家的古薰儿。
<br/>　　古薰儿看到慕清霄的瞬间，瞳孔剧烈收缩，脚步顿时停下，不再前进一步，使得身后的五位男学员差点撞在一起。
<br/>　　小医仙自然也看到了古薰儿，美眸中不禁闪过一丝惊艳，发现她的目光落在慕清霄身上，美眸闪烁了一下。
<br/>　　转过身来，慕清霄微微一笑，道：“熏儿姑娘，别来无恙啊。”
<br/>　　见他露出一丝笑容，古薰儿脸色不变，但心中却升起一丝警惕，当初凌老说过，此子极其危险，可能是八族中人。
<br/>　　不过，让她困惑的是，此刻站在慕清霄面前，并没有当初那种极致的压迫感，只有体内的金帝焚天炎，传来一丝余悸的感觉。
<br/>　　她当然不知道，慕清霄未免暴露自己的实力，可是将气息控制的极其完美，完全压制在了斗师层次。
<br/>　　至于金帝焚天炎，可能是同为火焰的缘故，它能察觉到慕清霄体内的九霄琉璃焱，本能的会产生一丝恐惧。
<br/>　　“你...怎么会在这里？”
<br/>　　“嗯？我为何就不能在这里？”
<br/>　　古薰儿为之气结，干脆转过头去，不再理他，更不想与他有任何接触。
<br/>　　琥乾也看出了两人似乎相视，也没有在意，随后目光落在其余五名男子身上。
<br/>　　“先恭喜你们通过内院的考核，先在这里提醒你们，内院可不比外院，其中，实力为尊，谁的拳头大，就能获得最好的修炼条件，在那里，你们任何的背景都没用。”
<br/>　　“既然已经来了，那我也就不再多啰嗦，你们也应该知道，叫你们过来的原因。”
<br/>　　说着，琥乾看向古薰儿，道：“熏儿，你当真不打算进入内院？”
<br/>　　闻言，古薰儿眼中满是坚定，道：“多谢副院长爷爷，熏儿确定。”
<br/>　　琥乾微微点头，眼中闪过一丝可惜，也不再啰嗦，来到身后的墙壁，手掌随意的拍了几下，一阵低沉的轰鸣声响起。
<br/>　　旋即，一条漆黑的通道就出现在了众人的面前。
<br/>　　“都跟我来吧。”
<br/>　　对着房间里的众人挥了挥手，琥乾就率先走进漆黑的通道中，慕清霄紧随其后，小医仙将目光从古薰儿身上收回，也快步跟了上去。
<br/>　　见两人都进去，古薰儿也迈入其中，身后的五名男子都面露好奇的跟了上去。
<br/>　　走进通道后，经过拐角，狭窄的通道就变得宽敞起来，两边墙壁上，镶嵌着一颗颗硕大的夜明珠，将通道中照耀的极为明亮。
<br/>　　通道内部，气氛安静，只有细微的脚步声沙沙响起，小医仙跟在慕清霄的身后，纤纤玉手拉着他的衣袖，寸步不离。
<br/>　　通道很长，仿佛没有尽头，慕清霄面色如常静静的跟在琥乾身后，小医仙亦是如此，古薰儿则不知在想什么。
<br/>　　至于走在最后面的五名男学员，都觉得百般无聊，也只有走在前面，身段婀娜的小医仙与古薰儿，能让他们心情舒畅一些。
