
<br/>　　<br/>　　    王惊蛰走了以后，李兰荣和茅清水都不觉得这一个小插曲有什么，在他们看来王惊蛰和陇西李氏之间隔着一条宽宽的渭河，奔腾的河水两岸，是完全没有任何可能合拢到一起的。<br/>　　<br/>　　    反倒是活了一大把年纪的李伯颇有些担忧的说道：“这青年说时隔一月再来，那时差不多到了清河崔氏过来提亲的时候，到时他要是在这个节骨眼上闹一闹的话，影响恐怕就不太好了这不是让人家看了笑话么？”<br/>　　<br/>　　    李兰荣笑了：“他要真的过来，到时扫地出门就可以了”<br/>　　<br/>　　    茅清水则直接说道：“那就让他连进门的机会都没有就行了”<br/>　　<br/>　　    关于李家老太爷当初为孙女定下的这门婚事，其实李兰荣和茅清水真的一直都是不太同意的，倒不是他们瞧不起王惊蛰，而是觉得李家长孙女的婚姻大事用指腹为婚来定的话，有点太草率了。<br/>　　<br/>　　    二十几年前，李兰荣和茅清水结婚后没多久就怀孕了，而那时也是李家过得稍微有些艰苦的时候，很多产业都青黄不接，很多难关都摆在了面前要解决。<br/>　　<br/>　　    那时正是国内经济即将要腾飞的时候，如果一个选择不好走错了路的话，李家偌大的家业恐怕就要走向衰败了。<br/>　　<br/>　　    当时身为家主的老太爷整日里都有些颇为愁眉苦脸的，族中一些有威望的人甚至都要开始逼宫让他退位了，就在这时老太爷跟李兰荣说自己要出陇西一趟。<br/>　　<br/>　　    “武则天当年登基为皇，你知道除了因为她手段强硬，狠辣以外，真正能让她登基的原因是什么嘛？还得除了被世家门阀支持这一点······”<br/>　　<br/>　　    李兰荣说道：“家谱中讲，是因为两位当朝大国师为武则天推断运势，辅助她上了皇位，一是袁天罡二是李淳风，没有他们两人就没有女皇武则天，也可能没有大唐的盛世”<br/>　　<br/>　　    “李世民曾经让李淳风和袁天罡为大唐推算国运，可惜只推到两百多年两人就不再推了，当时李世民就问你们不是可以推出后五百年的事么，怎么就不推了呢？两人当时就跪下了，说强行为一国推算国运有损人寿，真要是把大唐的国运全都推算一遍的话，恐怕他俩当场就得暴毙了，我们要是死了大唐没有国师的话恐非什么好事，所以就只推到两百多年为止吧，李世民当时也没强求，他对袁天罡和李淳风的话都是非常信服的，想了一下觉得大唐反正都有快到三百年的运势也算可以了，哪曾想到的是李唐家的天下也就到了二百八十九年就结束了呢，其实当年推算国运，是他们知道李唐的江山也就只有二百八十九年而已，但这话却万万不能和李世民讲的”<br/>　　<br/>　　    李兰荣说道：“爸，这李淳风和袁天罡的推演一术真的就那么邪乎么？”<br/>　　<br/>　　    “那是自然了······”老太爷当时也没深说什么就告诉李兰荣道：“我出去一趟找个人，此人也懂推算一道，并且似乎和袁天罡还有李淳风都出自同源，咱家的问题就问问他吧”<br/>　　<br/>　　    老太爷一走就两个多月，再回来的时候茅清水怀孕已经五个多月能看出是男还是女了，于是知道李家有了长孙女以后，老太爷就跟他们夫妻说。<br/>　　<br/>　　    “等这孩子长大以后二十五岁本历年之前，会有人来我们家提亲，到时就把这孩子许配给人家”<br/>　　<br/>　　    至于为何说要把李家长孙女指腹为婚的嫁出去老太爷却没有多说，只是叮嘱这婚事必须得成。<br/>　　<br/>　　    本来茅清水是想反对来的，但李兰荣却拉着没让她说，就跟她讲道：“我爸身体再好也肯定活不到二十五年后了，到时候姑娘嫁娶就是我们自己的事了，再一个嫁的那人家也未必会真会等到二十几年后，这婚事没准就是个玩笑，随便说说的呢？”<br/>　　<br/>　　    几个月后，茅清水产女，老爷子的身体却每况日下了，没过几年就躺在床上要撒手人寰了，他在临死之前还把茅清水和李兰荣叫到身前吩咐他们，一定要记得那个婚约不可反悔。<br/>　　<br/>　　    哪曾想到的是，李兰荣和茅清水都认为老爷子糊涂了，都病入膏肓了居然还记得二十多年后的婚约，不过两人当时也没反驳他，但却等老爷子去世之后，直接把女儿的姓给改了，不姓李。<br/>　　<br/>　　    这婚约说的不是李家的长孙女大小姐么，我都不姓李了，看你这婚约还如何作数！<br/>　　<br/>　　    两人想的是，自己的女儿暂且不说媒妁之言什么的了，就是要嫁也得嫁个登对的人家才行，世家门阀之间最重联姻，女儿要选夫的话，寻常人是肯定不行的。<br/>　　<br/>　　    所以，李兰荣和茅清水喜上眉梢，却是不为惊蛰。<br/>　　<br/>　　    再说王惊蛰，从李家出来后就也没觉得有什么所谓的，王爹只说让他来提亲，可没说一定要非娶不可啊，难不成人家不愿意我还得强买强卖不成？<br/>　　<br/>　　    退婚其实也没什么，但这口气必须得要！<br/>　　<br/>　　    出了李家没多久，王惊蛰打了辆车就来到了渭河边上，不到一月还得再上李家，他打算最近这段日子里就呆在附近等着了。<br/>　　<br/>　　    渭河岸边停着许多渔船和木舟，有的是为了载客过岸也有的是为做生意，渭河鲤鱼还有河鲜很有名，经常有人坐渔船到河上然后直接从河里打捞上鲤鱼河鲜就在船上直接做了然后开吃。<br/>　　<br/>　　    王惊蛰看差不多也到饭点了，就找了一艘渔船划到了河中，船家停好船以后就问王惊蛰喝不喝酒，然后把一篓子啤酒拴上放到了河里。<br/>　　<br/>　　    “哗啦”船家甩手洒出一张渔网，等网落水之后就快速的收拢然后往船上拖拽着。<br/>　　<br/>　　    “呦，您今天的运气不错，打上来一网好鱼呢·······”<br/>　　<br/>　　    船家把渔网拽上来后，里面有两条活蹦乱跳的大鲤鱼和一堆河虾，王惊蛰笑道：“都说这渭河鲤鱼鲜嫩的很，这就宰了吧”<br/>　　<br/>　　    离这条渔船不远，飘着一条木舟，舟上一个女子坐在竹凳上手里握着一根鱼竿，旁边有个年纪不大的姑娘眼神幽怨的看着王惊蛰那条渔船说道：“小表姐我们都钓了大半天了，一条鲤鱼都没钓上来，你看人家一网下去就见鱼了，这什么命啊？”<br/>　　<br/>　　    钓鱼的女子瞥了那边一眼，正看见王惊蛰在那手舞足蹈的喊着杀鱼吃肉呢，这女子忽然就愣了，这人天真的好像一个孩子，笑的很干净。
