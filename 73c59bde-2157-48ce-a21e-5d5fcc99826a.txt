        &lt;h3&gt;第三卷第一百八十章  我结婚了，你怎么办？&lt;/h3&gt;
<br/>
<br/>        江俞轩从张家出来，并没有在张家多做停留，他和张倩楠的婚事，两家大人不反对，但是好似也不是很支持，尤其是今天的状况感觉很诡异，现在都是二十一世纪了，哪里能像原来那样端杯茶就算结婚了？
<br/>
<br/>        如果说大操大办是铺张浪费，那么小打小闹，请自家的亲戚朋友吃顿饭也可以啊，现在就他们家里几个人吃顿饭就算结婚了？
<br/>
<br/>        江俞轩自己也想要一个像样的婚礼，谁说男人就不渴望有个风风光光的婚礼了？何况他还是江家大少。
<br/>
<br/>        他和张倩楠的婚事大多数人并不知情，就如寒伯安一样，他们的女朋友和婚礼都是备受瞩目的，为了不给张倩楠带来困扰，所以他从来都是独自一人出面。
<br/>
<br/>        到现在外面还在流传着：柳下惠江大少，多才多艺又多金，冷面冷心又冷情，不知其心属何人。偶尔有一些能知道真相的都明白其中的道理，三缄其口，谁也不说破。但是像夏颜那种不知死活的真的是只此一枚。
<br/>
<br/>        从张家出来的时候，大家都一直沉默着，尤其是张函更是眉头紧锁，他和落白在书房里说了一会话，出来就一直愁眉不展。
<br/>
<br/>        江俞轩感觉到不对劲，却也不知道该怎么开口，他是一个生意人，对各种世事的观察力还是很强的，在社会上摸爬滚打了这么多年，有些事情大家看的很明白，只是不挑破而已。
<br/>
<br/>        张倩楠说要和他一起回广州，他并没有接话，不是他不想，而是要等张倩楠回去和他父母亲商量。
<br/>
<br/>        江俞轩父子在张家少坐了一会，提出要在附近走走，便告辞回到了酒店，这个酒店是他们旗下的酒店，父子俩个人每个一个总统套房，只是江俞轩在陈子昂的公司上班的时候，从来都不暴露身份，默默的跟在陈子昂的身边，家里的福利也不曾用过，如今不用隐瞒身份了也就大大方方的住回到了自己的酒店中。
<br/>
<br/>        “俞轩，你心里是不是有些不舒服张家今天的态度？”江爸爸问江俞轩。
<br/>
<br/>        “爸，我觉得婚礼也不能这么草率。”江俞轩郁闷的说。
<br/>
<br/>        “落白在张家。”江爸爸对江俞轩说。
<br/>
<br/>        “张函今天说落白是张家二叔的孩子。”
<br/>
<br/>        “落白的妹妹要嫁给吴家，在此之前，吴家断了左膀右臂，很多资金流都进入了落家的手中，当然，寒家也捞到了一部分好处。”江爸爸紧紧的盯着江俞轩。
<br/>
<br/>        “这个问题太复杂了，有消息透露落雪和吴家大少明年4.18号结婚。今天老爷子让我们明年下半年再举行婚礼，也就是说不准我们现在公开我和张倩楠的恋情。”
<br/>
<br/>        “所以，你和倩楠还是低调为好！”
<br/>
<br/>        “我知道了。”江俞轩心思开始变得沉重起来，张倩楠好说，可是还有一颗定时炸弹埋藏着，不知道何时能拨掉，年后回到西城是要加紧步伐了。
<br/>
<br/>        “轩轩，我们只是生意人，你只要记得照章纳税，合理取得利润，做一个合格的商人就好，何为合格的商人？“江爸爸语重心长。
<br/>
<br/>        “爸爸，一个合格的商人就是：要坚信人之初，性本善，人之初，性本上，要善良上进。要坚持社会公理，要爱国、爱民、爱他、爱己！照章纳税，合理取得利润！”江俞轩对着父亲坚定的说。
<br/>
<br/>        “你这几年在外面闯荡着，我也一直关注着你成长的经历，在陈子昂的公司你们做的很好，能在困难的情况下保持初心，勇往直前，这是我欣慰的，也是一直让你在陈子昂的公司里呆下去的原因，你和陈子昂算是相辅相成，俩个人都拥有一颗坚强而勇敢的心，如果说最合适的伴侣，那么非你们俩个人莫属，但是世事造化弄人，你们只能做朋友，你能够不纠缠，远关注，这份感情让你老爸我都很佩服。轩轩，爸爸希望你不被眼前的迷雾挡住了你明亮的双眼。”
<br/>
<br/>        “爸，今天大概是你这么多年给我说得最多的话了吧，哈哈哈哈。”江俞轩似乎是被人揭露了心思一般有些慌张。
<br/>
<br/>        “爸，我今天已经和倩楠结婚了，虽然没有举行婚礼，我们证也领了，我已经决定此生就爱她了。明天就带她回广州吧！”
<br/>
<br/>        “明天看情况吧，也不一定倩楠能回去。”江爸爸看着自己的儿子，他心里那点小心思怎么能瞒得过他？第一次取消婚礼的时候，他发现他竟然感觉莫名的轻松。
<br/>
<br/>        ........
<br/>
<br/>        张家送走了江俞轩父子，张函一家和张倩楠一家在一起商量起张倩楠的婚事。
<br/>
<br/>        张沛尧：“二叔今天不让倩楠和江俞轩公开他们的恋情怕是有一定的原因啊。”
<br/>
<br/>        张雪云：“落家那姑娘和吴家联姻也不知道是好是坏，小函，你今天见落白怎么说的？”
<br/>
<br/>        张函：“落白说他摸不清落雪的心思。”
<br/>
<br/>        张雪云：“如果落家和吴家联姻，虽然有落白这一层关系在，但是我们张家怕更是会在夹缝中生存啊。”
<br/>
<br/>        张沛尧：“按照二叔说的吧，倩楠和江俞轩的关系暂时不要公开。”
<br/>
<br/>        张雪云：“二哥，陆玉这两年基本都不回来了？”
<br/>
<br/>        弥青云：“她不回来还好，回来了怕是事情更多。”
<br/>
<br/>        张沛尧：“小函今天和落白有没有谈论孩子的事情？”
<br/>
<br/>        张函：“他说可以先把孩子送出去，但是这不是长远之计，如果有人真的想找事情，怕是捂不住的。还有一点，江俞轩曾经说过俩个孩子是他的，他们俩个人现在是公众人物，陈子昂的公司马上面临着上市，如果事情不解决，估计以后会引起两家公司的震荡。”
<br/>
<br/>        张雪云：“其实这些事情最不应该牵扯进来的就是江家那小子。你们兄妹俩个人真是太荒唐了，要让人家给你们当卧底。”
<br/>
<br/>        张倩楠：“父亲，我是真心爱江俞轩的。”
<br/>
<br/>        张雪云：“你闭嘴，当初要不是你主动的去接近江俞轩，套取什么情报，那有今天这些事情？”
<br/>
<br/>        张沛尧：“三弟，你早就知道他们的情况？”
<br/>
<br/>        张雪云心情有些烦躁的说：“你以为我像你一样，只想往上爬？”
<br/>
<br/>        张沛尧和弥青云相视一眼，各自叹口气。
<br/>
<br/>        张倩楠：“我明天随俞轩去广州。”
<br/>
<br/>        张雪云：“你那里都不要去了！就给我乖乖的呆在家里，如果你是为了江家好的话。”
<br/>
<br/>        张倩楠：“爸！”
<br/>
<br/>        张沛尧：“倩楠，我们张家只有保持中立，不和任何势力接触，我们才能安稳生活！”
<br/>
<br/>        张倩楠转身走到了院子里，她以为她能和江俞轩相守一生，怕也只是梦一场？当初为了哥哥接近江俞轩，这样能帮哥哥了解陈子昂的一切，谁知道自己竟然慢慢的对江俞轩动了情。
<br/>
<br/>        这么些年江俞轩任劳任怨的跟在陈子昂的身后，她也能看出来江俞轩的心思，但是看着自家哥哥那么痛苦，她宁愿用一生去维护他的爱情。
<br/>
<br/>        北京的冬天实在是寒冷，张倩楠在院子里坐了一晚，她感冒了。
<br/>
<br/>        江俞轩第二天来接张倩楠的时候，张倩楠卧床不起，江俞轩无奈只得随着父亲飞回广州。
<br/>
<br/>        在江俞轩飞回广州的晚上，他在电视里看到了寒伯安向安男求婚的场景，他热泪盈眶，拿起电话给陈子昂打电话；“陈子昂，你看大哥要结婚了，我结婚了，你怎么办？”
<br/>
<br/>        电话里面只有轻微的抽泣声，不知是悲是喜！
