　　狼牙基地。
<br/>　　尚九娣满脸忧愁的坐在床边，楚涵面上的苍白从送回来到现在就没有变过，冰凉的体温若不是能感受到心跳，恐怕谁都会认为躺在这里的是个死人。
<br/>　　负责第一时间护送楚涵回基地的那名飞行员为军法处的人，当下已经被严加看管起来，虽说所有人都很清楚此人既然被何峰选中执行这次任务，定然不会暴露任何有关楚涵的事情，但尚九娣还是谨慎无比，好在那名军法处的成员也理解此时的特殊情况，全称配合。
<br/>　　于是就这样，楚涵被送回狼牙基地之后的好几天内，便一度消失般的一点信息都没有传出，安静的让不少人都人心惶惶。
<br/>　　尚九娣伸出手，轻轻一抚楚涵的额头，声音响起带着冷漠：“你倒是消失的够久。”
<br/>　　尚九娣是在跟谁说话？
<br/>　　这间屋子里明明是被她设为的禁地，除了她本人禁止任何人靠近，并且这两天楚涵的一切都是她在亲自打理，不可能有人进的来！
<br/>　　——————
<br/>　　当下的狼牙基地已经接近于一个空基地，但凡有用的东西都已经搬移到了段氏基地，此时还留守在这里的也就几个离不开需要时刻运转的部门，以及研究部的袁熙叶。
<br/>　　作为楚涵的老妈，一苏醒就立即参与到研究中的袁熙叶，将所有的心神都放在了研究上，这段时间她无数次的想起那个让她可以吐槽个遍的丈夫。
<br/>　　“要是那家伙在就好了，这东西早三个月就能研究出来。”袁熙叶望着手中的这支试管，其内透明的液体中隐隐闪着星星点点的金光。
<br/>　　袁熙叶已经有三个月没有见过阳光，这段时日·她没日没夜的待在研究室内，就是为了那支药剂的研究。
<br/>　　推开门走出，袁熙叶眯起眼睛适应了一下此时正当下午的明亮光线，而后环顾了一圈空空荡荡的基地，偶尔有一两个人路过此处，见到袁熙叶的身影都纷纷露出惊讶的目光，而后立即调头狂奔。
<br/>　　袁熙叶知道，这些人是去向上级汇报她出关的消息。
<br/>　　而她的出关，意义非比寻常……
<br/>　　“都搬空了，这几个月内又打了多少场仗？”袁熙叶自言自语的说着，轻皱着眉头。
<br/>　　没有等待那些汇报的人离开后再回来领路，袁熙叶忽有感悟似得，迈开脚步朝着一个方向走去。
<br/>　　——————
<br/>　　此时在楚涵昏迷静趟的小木屋内。
<br/>　　“一直关注着。”一个声音从角落里响起，回答了尚九娣问出的话，同时一个身影也从屋内的阴影内走出。
<br/>　　尚九娣起身回头，看着不知什么时候进入屋内的人，带着一丝淡淡的敌意：“我陪了他两年，而你呢？一去不回，现在他昏迷不醒，说不定就跟白家脱不了关系，你这时候来做什么？”
<br/>　　这位从角落里走出的来者，正是白允儿。
<br/>　　狼牙基地已空，但因为还有袁熙叶这一重要人物，近期也有昏迷的楚涵被送回，所以基地的防守虽说大不如前，但也绝对称得上很严。
<br/>　　白允儿能够悄然无息的来到此处，并且没被任何人发现，足以证明其神秘家族嫡系的能力之恐怖。
<br/>　　她所能代表的，很大意义上就是白家白忧的同等，甚至更高战力！
<br/>　　白允儿依旧一身白衣，黑色的长发及腰，带着冷然的面容看上去与两年前没有任何差别，无论这个末世纪元发生了什么，无论此时此刻的末世已经进入到哪个阶段，她都没有变化。
<br/>　　好似依旧还是当年那个在孤儿院里，蹲在一堆丧尸头颅前方的十七岁女孩。
<br/>　　白允儿用带有疑惑的目光，先是看了一眼尚九娣，再转移到了床·上昏迷中的楚涵身上。
<br/>　　她道：“我不记得，我以前住在过这里么？你用了‘回’这个字。”
<br/>　　尚九娣的瞳孔一缩，错愕的看着被她视为最大轻敌的白衣少女：“你……”
<br/>　　话到嘴边，尚九娣却不知如何开口，白允儿一向特别，无论是她本身，还是在楚涵心里，至始至终都是。
<br/>　　但眼下的情况，显然是白允儿遇到过什么事，让她失去了记忆？
<br/>　　“我的记忆只有最近的这一年，他们给我灌输的思想，是我生来就在白家，是白家的大小姐。”白允儿伸出手挡住旁边窗户射来的阳光，脸庞上的神色朦胧又迷茫：“我的记忆没有了，感觉却还在。”
<br/>　　话落，白允儿忽的将白·皙如青葱的手放下，利落的动作后刹那间握成圈。
<br/>　　“我对白家的所有人都有着陌生感，却对眼前这个昏迷不醒的人一见如故。”白允儿说着，眯起了眼睛：“眼见不一定为实，但感觉一定是真的。”
<br/>　　尚九娣的瞳孔再次剧烈收缩，当下的她竟一句话都说不出来，一种无尽的挫败感让她无法开口。
<br/>　　白允儿继续开口道：“白家的家主不是重伤垂死闭死关，而是为了躲避制裁者进入了白家禁地的密室，在找到能避开制裁者追杀的办法之前，他终年都不会出来。”
<br/>　　“白家最强的不是白忧，而是那名家主。”
<br/>　　说到这里的白允儿语气一顿，也让听着的尚九娣心脏骤然一停。
<br/>　　紧接着，白允儿看着床·上的楚涵，继续道：“白家的事我大多不知道，他们并不让我参与，高家的秘密也不少，高少辉同样不知道，高家的家主从未出现过在人前，而当下除却身份地位不同寻常的高漫秋，还有位比高家几个嫡系战力更强的人，是高家家主年轻时的一位老奴。”
<br/>　　“肖家已灭族，但还有最后一位成员肖七，她是返祖者并且活的很隐秘，没有大事发生不会出现。”
<br/>　　“至于罗家……”说到最后一个神秘家族的时候，白允儿忽的嘴角轻轻勾了勾，静静的吐出了四个字：“不足为惧。”
<br/>　　罗家离开华夏太久，远离阴阳谷生活在海洋对面的大陆长达几百年，与另外一直将本家设立在阴阳谷不远处的三家相比，的确不足为惧。
<br/>　　一直听着的尚九娣心绪复杂，她定定的看着白允儿，尤其是在最后一句话顿住的时候，尚九娣不知怎么忽然觉得那一瞬间的白允儿，神态与楚涵太过于相似！
<br/>　　那是一种站在顶点，对裙下之臣的俯览。
