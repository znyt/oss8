夏夕绾手脚冰凉，爷爷刚苏醒，今天本来是团圆的日子，但是她没有想到爷爷竟然中毒了，团圆也成了永别。
<br/>
<br/>        所有的欢喜瞬间被淋灭，夏夕绾紧紧的握住老爷子的手，眼眶迅速变得湿润，“爷爷，别走，求你了，你是我唯一的亲人了。”
<br/>
<br/>        夏老爷子不舍的摸着夏夕绾的脑袋，然后看向了夏振国。
<br/>
<br/>        夏振国“扑通”一声在床头跪下了，“爸！”
<br/>
<br/>        夏老爷子说了最后的遗言，“绾绾是我们夏家的小小姐，整个夏家都必须听命于她，包括帝都城的夏家，你二叔他们，我走后帝都城的夏家必然有人来接你，你没什么出息，就不要去了，留在海城混混日子吧。”
<br/>
<br/>        夏振国怎么也没有想到老爷子对他的遗言就是“混混日子”，这四个字不知道是老爷子的心酸还是夏振国的心酸了。
<br/>
<br/>        这时夏老爷子缓缓闭上了双眼，面容安静祥和的走了。
<br/>
<br/>        夏夕绾被泪水模糊了双眼，“爷爷！爷爷！”
<br/>
<br/>        夏振国成了没爸的孤儿，也哭倒在了床前。
<br/>
<br/>        ……
<br/>
<br/>        整个医院都笼罩上了一层离别的悲伤，夏夕绾坐在医院冰冷的长椅上，哭的双目红肿，爷爷的离开带走了她对这座海城最后的留恋与不舍。
<br/>
<br/>        叶翎将一杯热水递了过去，“绾绾，老爷子走的很安详，你也节哀顺变吧，现在最重要的是查清楚谁给老爷子下毒的，究竟是谁害死了老爷子！”
<br/>
<br/>        夏夕绾将脸上的泪珠擦干净，是啊，她现在没有时间悲伤，她还有很多事情要做。
<br/>
<br/>        夏夕绾扭头看向身边的夏振国，夏振国也哭成了一个泪人，“我不是你的亲生女儿，你为什么不告诉我？”
<br/>
<br/>        夏振国收起了心里的难过，胡乱的擦了擦泪，“这件事老爷子不让说，我哪里敢告诉你？”
<br/>
<br/>        夏夕绾突然觉得释然了，其实她一直不明白为什么夏振国不爱她，原来她根本就不是夏家的女儿。
<br/>
<br/>        她有自己的爹地，只可惜她并不知道自己的爹地是谁。
<br/>
<br/>        “帝都城也有夏家吗，我们有二叔吗，我从来没有听爷爷提起过。”
<br/>
<br/>        夏振国想了一下，“我对这些也不太清楚，老爷子在世的时候并没有跟我说过，不过我知道我们海城这个夏家只不过是夏家分出来的一条支线，是当年你妈咪来海城时带出来的，真正的夏家在帝都城。”
<br/>
<br/>        “这些年我们和帝都城的夏家并没有联系，这应该都是听命于你妈咪，想斩断和帝都城那里的联系。”
<br/>
<br/>        夏夕绾点了一下头，她本来以为爷爷苏醒了，她就可以知道全部的真相了，但是没想到她反而越来越迷糊了。
<br/>
<br/>        她不知道妈咪究竟是怎样一个人，妈咪身上好像有很多谜。
