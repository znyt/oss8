　　　　涂小安已经不在需要白楚燕了，她的带路任务已经完成，现在黄金瞳就在眼前，他只要等待时机，将它带走就行了。
<br/>　　事情很简单，不需要惊动任何一个人，悄无声息。
<br/>　　白家祖奶奶就这样站在笼前，欣赏猫的雄姿，不时一伸手，庭院中一颗鲜艳欲滴，很饱满，蕴含灵气的樱桃像是长了翅膀，自动飞了下来，而这位美妇人，一张嘴，便眯着眼吃了下去，鲜红的果汁溢到嘴角。
<br/>　　如今的夜慢慢的深了，涂小安只能静观其变，美妇人是一位他系统都洞察不出的强者，而且看样子还有点邪乎。
<br/>　　他只能等对方就寝，才能下手，只是这旗袍女人愣是没有准备去睡觉的意思，反而坐在了一张椅子上，坐着下巴，看着笼中的黄金瞳发狂，越发狂她好像还越高兴。
<br/>　　这将涂小安弄的非常无语，另外，他特别留意过，白家的别墅内，并没有什么蛇类的气息。
<br/>　　成年的金蚕蛊蛇到底是否真实存在。
<br/>　　莫约过了一个小时，凌晨十二点，这位邪乎的祖奶奶总算坐不住了，站了起来，拿起一块黑布，冲着里面的猫说：“小家伙，明天之后，你就自由了，到时候祖奶奶我会重新给你觅得一个配的上你的主人”
<br/>　　黑布该盖下，黄金瞳安静了下来。
<br/>　　她款款莲步的朝着内屋走去，应该是去休息了。
<br/>　　十分钟之后，一条紫金蛇悬浮了出来，涂小安看着盖着黑布的笼子，微微的思忖一下，到也不急。
<br/>　　他怕贸然动手，黄金瞳发出吼叫，又将那位好不容易走了的美妇人给惊动了。
<br/>　　旋即，涂小安飞到了庭院，看了看那株樱桃树，眼睛亮了起来，这樱桃远看鲜艳欲滴，但是近看每颗其实都是深紫色，足足有一块硬币那么大，很饱满，灵气浓郁，甚至是扑面而来。
<br/>　　刚来到白家，其实涂小安就发现了白家这栋别墅灵气很丰盈，很明显，甚至可以跟古学院比。
<br/>　　像外面城市内，正常来说，是感受不到什么灵气的，只有一些个别的地方，灵气会浓郁点。
<br/>　　尤其是这株樱桃树，灵气汇聚，每一颗樱桃都是自带灵气的。
<br/>　　但涂小安突然流露出了古怪之色，这樱桃虽然灵气浓郁，可散发出来的却不是鲜果的香味，而是一种酸酸的腥味。
<br/>　　没错，这樱桃浓郁灵气下，还伴随着腥味，虽然很淡，但还是被他察觉出来了。
<br/>　　“吃吃看吧”他蛇口张开，一口吃下樱桃。
<br/>　　顿时间，涂小安浑身轻飘飘起来，很浓郁的能量，吃了让人感觉充满了力量。
<br/>　　“叮...宿主获得100进化能源”
<br/>　　吃下，获得能量，系统数据顿显，涂小安一下子心喜了起来，这一颗小小樱桃抵得过一块灵石呀。
<br/>　　怪不得那个祖奶奶让白楚燕多吃樱桃，那丫头也是傻，生在福中不知福，这么好的东西，却就是不吃。
<br/>　　普通人每日吃一颗樱桃，身上素质不断得到改造，很容易就能觉醒。
<br/>　　这整整一颗樱桃树，遍布茂密的樱桃，略微的估算，足足不下上百颗。
<br/>　　也就是说，这颗樱桃树挂着等于上百颗的灵石。
<br/>　　既然是这样，那么本蛇王就不客气了。
<br/>　　涂小安的灵石没了，正愁没地方弄，这樱桃很好的解决了这个问题。
<br/>　　如果他将这上百颗樱桃都吃了，差不多能提升一万多的进化能源。
<br/>　　涂小安现在要证道，必须先要获得十万的进化能源封顶，这可不简单，先累计一万是一万。
<br/>　　就这样，一条紫金蛇悬浮在樱桃树前，一口一颗的吃着樱桃，进化能源数据在不断跳动。
<br/>　　这种获得进化能源的方式，让涂小安非常的惬意跟喜欢，吃着水果，就把实力给提升了。
<br/>　　一颗二颗三颗，当涂小安刚吃下第十颗樱桃，他整个小小蛇躯一僵，樱桃树上突然有一双双冒着幽然的眼睛浮现了出来。
<br/>　　就好像黑暗中，树上亮起了一盏盏的灯笼，明亮而闪烁着。
<br/>　　这是蛇眸。
<br/>　　涂小安诧异不已，原来樱桃树内，藏匿着毒蛇，是什么蛇，它们通体金黄色，体形似蚕。
<br/>　　“是金蚕蛊蛇”
<br/>　　一条两条三条，仅仅一眼，涂小安就发现了樱桃树上，有不下十几条的金蚕蛊蛇，每一条都比余子童身上的蛊蛇要大的多，恐怖的多。
<br/>　　它们盘踞在树上，由于涂小安的小偷行为，将它们给惊动了。
<br/>　　也好在是涂小安，要是寻常人来偷樱桃，看到这一幕，三魂都不见了七魄，吓都被吓死。
<br/>　　“叮...金蚕蛊蛇，觉醒蛇类，等级15级，吞噬可获得300的进能源”
<br/>　　“叮...金蚕蛊蛇，觉醒蛇类，等级18级，吞噬可获得500的进化能源”
<br/>　　“叮...金蚕蛊蛇，等级20级，二阶精怪层次，吞噬可获得1000的进化能源”
<br/>　　一道道数据信息扑面而来，吞噬金蚕蛊蛇？
<br/>　　涂小安貌似很久没有吞噬同类了，当初佘山蛇谷，他吞噬了变异吸血蛇登基坐殿，之后在没吞噬什么蛇种。
<br/>　　也是后来，涂小安没有碰到什么厉害的蛇。
<br/>　　这金蚕蛊蛇，是用特殊手段创造出来，非比寻常，远远要比一般的觉醒蛇种厉害。
<br/>　　按照蛇博士所做，弄一条金蚕蛊蛇的非常的繁琐跟麻烦，可这颗樱桃树上，整整有十几条之多。
<br/>　　而且有精怪级别，那是属于成年了？
<br/>　　不对还没成年，气息不对，涂小安对蛇类是非常的敏感，他能感受的到，就算树上有精怪层次的金蚕蛊蛇，但是依然还没真正的成年。
<br/>　　不得不说这种用特殊手法创造出来的蛇种很变态，都达到了精怪层次，还没有成年，潜力非常的大。
<br/>　　涂小安这条悬浮的紫金蛇，就这样被一双双冒着幽然冷光的眼睛盯着，他没有不寒而栗，反而有点兴奋起来。
<br/>　　看来今天晚上，来白家是来对了，上百颗如同灵石的樱桃，十几条金蚕蛊蛇，最低的等级都是十五级。
<br/>　　进化能源几百起步，毫无疑问，这都将是涂小安今晚的收获，统统的吞噬了，他的证道日期也就越来越近。
<br/>　　他证道，异于常人，必须厚积薄发，难度很大，当然，他现在本身的战斗力都比一些证道的三阶生灵要强。
<br/>　　什么事情都有两面性。
