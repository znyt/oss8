　　哗！
<br/>　　叶尘只感觉眼前一花，
<br/>　　下一刻，
<br/>　　出现在一个恢弘、古朴的大殿之中。
<br/>　　“哼！”
<br/>　　叶尘才刚刚站稳身形，一道冰冷而苍老的声音，陡然响起在大殿中，
<br/>　　“好大的胆子！不但打伤本帝的孙儿，竟然还敢擅闯本帝的宫殿！”
<br/>　　叶尘循声望去，
<br/>　　只见那大殿的尽头，一张宽大的金色王座之上，居然盘踞着一头无比庞大的巨蟒！
<br/>　　不！
<br/>　　更准确的说，不是巨蟒，而是一个头半人半蟒的怪物！
<br/>　　因为这头巨蟒，有一半的身体，却是一个满头白发的老者，穿着一件赤红色的长袍。
<br/>　　而此时，那幽冥炎蛇正犹如一只小鸡般，被这老者擒在手中，无法动弹分毫。
<br/>　　至于那吞天蟒，在盘踞在王座一下，已经奄奄一息。
<br/>　　看到这一幕，叶尘的脸色瞬间大变，
<br/>　　因为他发现，眼前这个半人半蟒的怪物，气息之强大，绝对是他重生以来，遇到的最恐怖的一位，恐怕比那造化天君生前，还要强大许多，他竟然完全看不透！
<br/>　　以他如今的修为，恐怕还远远不是眼前这个怪物的对手！
<br/>　　在这么一个小小的五级星域，怎么会有如此恐怖的存在？
<br/>　　叶尘的一颗心，顿时凉了大半截，同时已经开始暗暗运转体内真元，一旦对方出手的话，他也能舍命一搏了！
<br/>　　然而，
<br/>　　等了好一会，那怪物并没有动手，而是在叶尘身上上下打量起来。
<br/>　　紧接着，两眼一瞪，满脸惊色，道：
<br/>　　“咦！你的身上……怎么会有主人的气息？小子！你到底是什么人？从哪里来？”
<br/>　　叶尘顿时听愣了，
<br/>　　主人？
<br/>　　难道他说的是造化仙君？
<br/>　　不对！
<br/>　　这怪物的修为已经如此厉害，它的主人应该更加厉害才对。
<br/>　　恐怕至少也得是合道仙君，甚至和前世的他一样，是一位渡劫仙帝！
<br/>　　更何况，如果它是造化仙君的话，不应该不认识幽冥炎蛇。
<br/>　　可是叶尘才刚刚踏足修真界不久，并没有接触到那个层面的人物啊！
<br/>　　这怪物说，自己身上有他主人的气息，却又是怎么一回事？
<br/>　　叶尘搞不清楚状况，依旧默然不语，同时暗暗留意周围的情况，寻找可以逃离的路线。
<br/>　　没办法，眼前这个怪物太强大了，而且他还身受重伤，肯定不能与之硬拼。
<br/>　　“砰！”
<br/>　　就在叶尘暗暗寻思之时，那半人半蟒的怪物，随手将手中的幽冥炎蛇丢到一旁，然后身体一晃，便已经冲下王座，来到叶尘十米之前。
<br/>　　“嗯？”
<br/>　　叶尘还以为对方要出手，顿时眉头一凝，险些就要强行催动逆生百转的秘术。
<br/>　　不料这时，
<br/>　　那怪物却突然开口道：
<br/>　　“小子！你不要紧张，本帝有一些问题，想要问一问你，只要你能如实回答，本帝便饶了你和你的灵宠，如何？”
<br/>　　叶尘听到这话，顿时眉头微微一挑，
<br/>　　虽然他不清楚，这怪物要问什么，可是眼下的情况，自然不能轻易动手，否则他今天恐怕就真的要栽在这里了！
<br/>　　“可以，那你问吧！”
<br/>　　叶尘决定先静观其变。
<br/>　　那怪物微微摆了摆硕大的蛇尾，沉吟了片刻，眼眸中泛起一抹精芒，死死盯着叶尘，道：
<br/>　　“你是什么人？从哪里来？”
<br/>　　这么一个简单的问题，顿时把叶尘给问住了，
<br/>　　这个世界的人，应该并不知道地球的存在，他就算说出来，这个怪物只怕也不会信。
<br/>　　那怪物似乎也看出了叶尘眼中的犹疑，于是又加了一句，厉声道：
<br/>　　“我最好实话实说，不然可不要怪本帝不客气！”
<br/>　　叶尘眉头一皱，淡淡道：
<br/>　　“并非我不愿意告诉你实情，我来的那个地方，我就算告诉你，你也未必知道，甚至还会以为，是我故意撒谎骗你！”
<br/>　　那怪物听到这话，两眼顿时微微一眯，脸上竟然浮现出一抹笑容，道：
<br/>　　“你尽管直说就行，是真是假，本帝自会判断！”
<br/>　　叶尘想了想，
<br/>　　其实他的来历，也并非是什么天大的秘密，而且他也很想知道，地球那一方宇宙，和修真界之间的关系。
<br/>　　眼前这头妖兽，估计早已经活了无数的岁月，说不定可从它的口中了解一些情况，甚至还能找到返回地球的办法！
<br/>　　一念及此，叶尘便不再纠结，当下便把自己的来历，一五一十的讲了出来。
<br/>　　当然了，自己用月光宝鉴重生，这是天大的秘密，叶尘自然不会泄露，他只说自己本身地球上的修真者，意外找到了通往修真界的入口，然后便来到了这里……
<br/>　　怪物老者听完叶尘对地球的描述之后，当即又问了一些地球上的历史，听完之后，顿时激动的难以自持，
<br/>　　“没错！不会错了！难怪你身上，会有主人的烙印，原来你真的是‘昊天界’之人啊！哈哈哈！我找到了，我终于找到了！！”
<br/>　　“昊天界？”
<br/>　　叶尘顿时一愣，连忙追问道：
<br/>　　“什么是昊天界？”
<br/>　　那怪物老者又大笑了一阵，方才渐渐平复下激动的心情，又沉吟了片刻，方才缓缓道：
<br/>　　“小子，我来问你，你应该清楚，作为修真者，一共有炼气、筑基、金丹、元婴、化神、返虚、合道、渡劫八大境界，可是在渡劫成功之后，又是何种境界，你知道吗？”
<br/>　　叶尘再次一愣，虽然不明白怪物老者，为什么会突然问起这个，不过还是把自己知道的回答了出来，
<br/>　　“传言说，一旦渡劫成功，便可成就混沌仙体，历万劫而不灭，只不过，所有渡劫成功的修真者，不久之后便会从这个世界消失，具体去了哪里，就没有人得知了……”
<br/>　　怪物老者听完这话，顿时眉头一挑，脸上露出一抹惊愕之色，
<br/>　　“咦！没想到你知道还挺多！不错不错！”
<br/>　　“渡劫仙帝，一旦渡劫成功，便是混沌仙体，由于力量过于强大，会被法则之力所不容，排斥出这个世界之外，进入无尽的混沌之中……”
<br/>　　怪物老者说的言之凿凿，倒是让叶尘震惊不已，
<br/>　　要知道，哪怕是前世的叶尘，登临修真界之巅，对渡劫之后的事情，也只是知道一鳞半爪。
<br/>　　这家伙到底是什么身份，居然知道这等无上辛秘？
