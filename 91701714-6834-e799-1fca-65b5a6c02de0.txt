        &lt;h3&gt;第三卷第二百零七章  夏颜到北京&lt;/h3&gt;
<br/>
<br/>        夏颜如愿和冯睿一起到了北京。
<br/>
<br/>        他们在酒店住下之后，就等着蔡少芬的接见，俩个人等到下午，蔡少芬姗姗来迟，恰好也是饭点。
<br/>
<br/>        冯睿：“蔡姐，这个就是我的干女儿夏颜，说了带她来见见你的，她也一直想要见你，听了你的为人处世简直就是对你佩服得五体投地。”
<br/>
<br/>        夏颜看了一眼冯睿：“天，你啥时候给我讲了她的为人处世事？”但是她嘴上却说：“蔡姨，冯妈说您人好又漂亮，这一见果然，要不是您的辈分在这里，我怕是要喊你大姐了。”
<br/>
<br/>        “就你这小妮子会说话，难怪冯睿喜欢你，好了，你这个干女儿我也认了。”蔡少芬被夏颜的话愉悦了。
<br/>
<br/>        “谢谢蔡妈，哎呀我真的好幸福，有好几个妈妈疼爱我。”夏颜从包里拿出来一块女款手表。
<br/>
<br/>        “蔡妈，第一次见面，这个是我孝敬妈妈的，你就不要嫌弃了，我也是前两天才买的。”夏颜二话不说就将手表戴在了蔡少芬的手中。
<br/>
<br/>        这块手表确实是前两天才买的，一共买了两块，准备孝敬给蔡少芬和弥青云的。
<br/>
<br/>        蔡少芬是什么人，眼睛一搭就能看出来这块手表的价值:“这不行，太贵重了，你看你是孩子我什么都没有给你，怎么能拿你这么贵重的礼物？”
<br/>
<br/>        “再贵重的东西都比不上蔡妈在我心中的分量，您就不要推辞了。”夏颜轻轻的拉着蔡少芬的手，这个女人保养的不错，后期可以拿一些高档的化妆品。
<br/>
<br/>        “你这孩子就是会说话，那我就收下了，以后有什么事情你给我说一声，我能帮的一定会帮你。”蔡少芬也不再推辞，这些人的钱来得很快，随便一个项目都是不少的钱，他们也不在乎那百八十万的。
<br/>
<br/>        “谢谢蔡妈，晚上我们一起吃饭？”夏颜顺顺利利的又认了一个干妈，再多坐一会，拉拉家常，促进一下感情，后期来往也就不愁了。
<br/>
<br/>        蔡少芬刚好也有事情要问冯睿，便也不推辞，三个气味相投的女人坐在了一起。
<br/>
<br/>        “冯睿，上次你说陈子昂孩子的事情你再给我讲一下。”三个人坐下后，蔡少芬就开始问陈子昂的事情。
<br/>
<br/>        “陈子昂？是我的那个同学吗？”夏颜有些不明所以的问冯睿。
<br/>
<br/>        “就是，也就是我的那个小姑子。哎，家门不幸啊。”冯睿叹了一口气。
<br/>
<br/>        “蔡妈，我知道陈子昂，她和我是同学，在高中的时候就不好好学习，天天和男生谈恋爱，导致大学都没有考上，后来又跑到酒吧，认识了张函，和张函在一起呆了三年，听说怀了张函的孩子之后被张函抛弃，然后在怀孕期间又搭上江俞轩，现在那俩个孩子就是张函的，不过江俞轩不知道傻傻的背着锅而已。”
<br/>
<br/>        “啊，你说的是真的？”蔡少芬差点一口菜噎住了。
<br/>
<br/>        “我们在高中呆了三年，也算是闺蜜，她有什么话都给我说，蔡妈，我当您是妈妈，这个话您听听就好，千万不敢说出去，我求您给我保密，我和子昂的关系非常好，她在西城的项目大多数都是我帮着她的。”夏颜为蔡少芬夹了一筷子菜，放在蔡少芬的碟子里。
<br/>
<br/>        “按照你的说法，孩子是张函的？”蔡少芬简直是惊掉大牙，张函那个方面不是不行吗？怎么会有孩子？
<br/>
<br/>        “是啊，陈子昂亲口告诉我的。她还说了，和张函在一起的那几年是她最快乐的时光，所以她舍不得孩子，和江俞轩在一起无非就是看着江俞轩能帮助她，况且她和江俞轩没有未来的，如果可以，她倒是愿意和张函从头再来，而且她也一直在寻找机会，现在她努力的强大，等到了一定的时候就可以和张函站在一起了。”
<br/>
<br/>        “唉，是我们陈家家门不幸，老陈家就是农民，没有什么知识，教育不好子女，真是让我难堪啊。”冯睿深吸一口气，泪眼婆娑。
<br/>
<br/>        “我们都被他给骗了。”蔡少芬叹了口气，这几年张家和他们家按说是在同一条船上的，只是两家总是面和心不和，她也一直认为张函那方面不行，现在孩子都有了，弥青云从上次见面之后，很少和她联系，莫非他们家都知道这个事情？知道这个事情还要瞒着他们陆家？哼，看来不给他们一点点颜色，不知道兔子急了还会咬人。
<br/>
<br/>        冯睿和夏颜相视一笑：“也不算是骗吧，张家有可能不知道呢，我也准备将这个事情给弥姐也说一声的。”
<br/>
<br/>        “你不要说了，我会看情况的，这个关系到我们家陆玉的幸福，我来处理，你们俩个当什么事情都不知道。当然以后有什么情况要及时给我说。”蔡少芬叮咛冯睿和夏颜。
<br/>
<br/>        “行，我们听您的，您说怎么办就怎么办。”夏颜心花怒放，目的达到了。
<br/>
<br/>        “夏颜，你以后有什么要求给我说，我虽然不在西城了，但是还是有点作用的，你年轻能干，好好干，以后也是玉儿的左膀右臂，等玉儿回来了我介绍你们认识。”蔡少芬开始给她的女儿找帮手了。
<br/>
<br/>        这个夏颜年龄也不大，情商智商都不错，是个可造之材，好好培养，以后也是有用处的。
<br/>
<br/>        “谢谢蔡妈，我也没有多大的抱负，就守着西城那一亩三分地就好了。”夏颜谦虚的说。
<br/>
<br/>        “那可不行，我蔡少芬的女儿可不是那种窝囊废，你回去好好干，等我消息。”蔡少芬对夏颜吩咐着，俨然就是一对母女。
<br/>
<br/>        “嗯，我听蔡妈的。”夏颜努力的藏着心中的得意和喜悦。
<br/>
<br/>        “时间不早了，我该回去了，明天我安排司机带你们出去转转。你们等我通知。”蔡少芬站起来准备走。
<br/>
<br/>        蔡少芬一走，冯睿：“颜颜，你说的这些话都是真的？”
<br/>
<br/>        夏颜：“只要蔡少芬当真就行了，你管她是不是真的。”
<br/>
<br/>        冯睿：“可是你为什么要这么说？”
<br/>
<br/>        夏颜：“我还不是为了你，你看看你现在过的是什么日子？大过年的一个人孤苦伶仃的守着偌大的房子，陈家人当你是人了吗？陈子昂不是陈家的骄傲吗？那就让他们的骄傲跌落尘埃。”
<br/>
<br/>        冯睿：“你这么恨陈子昂？”
<br/>
<br/>        夏颜：“恨不恨不重要了，况且我也没有理由恨她是吧。我是真的为你打抱不平，你对我这么好，我难道不为你做点什么吗？”
<br/>
<br/>        “哎，还是女儿贴心，颜颜，也只有你能为我着想了。”
