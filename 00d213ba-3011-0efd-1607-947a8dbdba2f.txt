　　浩大的山脉之中，兽吼震天，惨嚎不断，血气染红了苍穹。
<br/>　　方圆万里之内，仿佛都化为了一片血色的世界。
<br/>　　苏莫的身形，在山脉上空不断的移动，大量的兽魂挣扎咆哮着向他汇聚而去。
<br/>　　他疯狂的吞噬，不顾一切的吞噬，数之不尽的兽魂被他吞噬。
<br/>　　“苏莫，汝不是要给吾一些魂力的吗？”
<br/>　　飞行之中，阎魔鬼圣的意念之音，传进了苏莫的脑海中。
<br/>　　他一直在等待苏莫给予他吞噬的魂力，这是让他恢复的良药。
<br/>　　“那就给你一些！”苏莫沉声道，随即他将吞噬的几十只王级凶兽的兽魂，魂力向阎魔鬼圣的身上涌去。
<br/>　　霎时之间，那些魂力便涌入了炎魔鬼圣的魔魂中，被对方完全的吸收了。
<br/>　　因为阎魔鬼圣在吞噬武魂之中，苏莫能清晰的感觉到，对方魔魂的实力，有所恢复。
<br/>　　虽然恢复的极为有限，但的确是变的强大了一丝。
<br/>　　苏莫立刻不再给对方魂力，免得对方实力太强，逃脱他的掌控。
<br/>　　“这……苏莫，能否再来一些？”阎魔鬼圣充满渴望的说道。
<br/>　　就只是刚才一瞬间的魂力，就让得他恢复了不少，抵的上他十年静养，这让他如何不渴望！
<br/>　　“鬼圣前辈莫急，待我突破到武帝境界之后，定然会再帮你恢复的！”苏莫轻笑着说道。
<br/>　　阎魔鬼圣闻言心中暗叹，他也知道苏莫不可能让他恢复的太多，现在强求也无用。
<br/>　　只能循环渐进，慢慢的来了！
<br/>　　随即，阎魔鬼圣便不再多言，陷入了沉寂之中。
<br/>　　苏莫继续吞噬兽魂。
<br/>　　轰隆隆！
<br/>　　就在此刻，一阵地动山摇，前方的山林中出现了一只巨大的蜥蜴。
<br/>　　此蜥蜴全身呈青紫色，身长达百余丈，一双巨大的眼瞳中，爆射出血色的红光。
<br/>　　嗖！
<br/>　　蜥蜴飞上天空，直接向苏莫冲了过来，凶残暴戾的杀机铺天盖地。
<br/>　　轰隆隆！
<br/>　　还未接近苏莫，蜥蜴巨大的尾巴一扫，如同一根巨大的长鞭一般，碾压一切，横扫虚空。
<br/>　　苏莫见此不惊反喜，因为这头蜥蜴，居然是一头相当于武帝境二重武者的八级二重凶兽。
<br/>　　他在这山脉中已经不短的时间了，还是遇到的第一头八级二重凶兽。
<br/>　　当然，他目前所在的这片山脉非常的巨大，他只是在外围区域而已。
<br/>　　“断！”长剑在手，苏莫一剑狠狠的斩击了出去。
<br/>　　咻！
<br/>　　犀利的剑气破空杀伐，锋锐无敌，只一剑便将蜥蜴的尾巴斩了下来。
<br/>　　吼！
<br/>　　尾巴被斩断的蜥蜴，不仅没有丝毫的惧怕，凶威更甚，血盆大口张开，一口向苏莫咬了过来。
<br/>　　腥风扑面，恶臭扑鼻，血盆大口瞬间便临近了苏莫的身前。
<br/>　　“死吧！”苏莫冷喝一声，爆发全身实力，一剑狠狠的斩在了蜥蜴的血盆大口上。
<br/>　　哧啦！
<br/>　　剑光闪过，巨大的蜥蜴，大嘴被撕开，随即身体也被一分为二，鲜血如大海浪涛，冲起百丈之高。
<br/>　　这只蜥蜴的兽魂，立刻便被苏莫吞噬了。
<br/>　　八级二重的凶兽，实力比之那廖云剑都远远不如，根本不是苏莫一合之敌。
<br/>　　苏莫的身形辗转在山脉之中，不停的杀戮，不停的吞噬，他也不往山脉深处前进，因为如此巨大的山脉，定然有非常强大的凶兽。
<br/>　　就这样，苏莫在这座大山脉中，整整待了十日，吞噬了数不清的兽魂，甚至有几名在山脉中历练的武者，都被他吞噬了。
<br/>　　十日的时间，他的武魂依旧没有晋升到天级九阶，这时候他离开了山脉，向途山镇返回而去。
<br/>　　时间已经来不及了，他必须回去了，只希望苍穹神宫众人，能为他抓到足够多的兽魂。
<br/>　　十几个时辰之后，苏莫回到了途山镇，苍穹神宫的众人，早已先他一步回来了。
<br/>　　所有人都将得到的兽魂交给了苏莫，还有驱使途山镇其它势力获得的兽魂，足足有上千万只兽魂，而且全部是六级以上的兽魂。
<br/>　　因为兽魂不好存放，在出发之前，他们专门到附近的大城池中购买了海量的魂晶。
<br/>　　上千万枚魂晶，交到苏莫的手中，让得他心中大喜。
<br/>　　……
<br/>　　玉镯空间中。
<br/>　　千万魂晶堆积如山，其内各式各样的兽魂，全部都被压缩在了微小的魂晶空间中。
<br/>　　站在魂晶大山之下，苏莫面色凝重，成败在此
<br/>　　“碎！”
<br/>　　苏莫意念一动，玉镯空间中的空间之力一阵挤压，千万魂晶顿时破碎。
<br/>　　刹那之间，千万兽魂纷纷冲出，震耳欲聋的咆哮声充斥在整片空间中。
<br/>　　数之不尽的兽魂，遮盖了天穹，覆盖了大地，场景震撼人心。
<br/>　　“吞噬！”
<br/>　　苏莫直接释放出了吞噬武魂，开始疯狂的吞噬了起来。
<br/>　　无尽的兽魂仿佛万流归宗一般，在惊恐的怒吼声中，急速涌进吞噬武魂之中。
<br/>　　时间缓缓流逝，一息又一息。
<br/>　　仅仅是片刻的功夫，千万兽魂便被苏莫吞噬了九成。
<br/>　　而这个时候，他的吞噬武魂终于没有让他失望，一阵剧烈的颤动之后，其上出现了第九道光环。
<br/>　　天级九阶武魂！
<br/>　　这一刻，吞噬武魂，终于达到了武魂的最高等级。
<br/>　　“哈哈！”苏莫心中狂喜，忍不住大笑了两声。
<br/>　　少倾之后，他将所有兽魂吞噬一空，心中的喜悦也暂时的压了下来。
<br/>　　接下来，他将要冲击武帝之境。
<br/>　　随即，苏莫询问了一番阎魔鬼圣，突破武帝有什么注意的事项。
<br/>　　对方是过来人，虽然不是人族，但想必也有很多了解。
<br/>　　从阎魔鬼圣的口中，苏莫得到了很多有用的信息，对于突破武帝之境，也有了详细的了解。
<br/>　　随后，苏莫也没耽搁时间，立刻便开始尝试突破。
<br/>　　他屏弃了心中所有杂念，心神沉入体内，抱怨守一，以精神意念沟通武魂，尝试着将吞噬奥义和自身神魂，融合进武魂之中。
<br/>　　苏莫首先是将吞噬奥义和武魂融合，因为这一步较为简单。
<br/>　　吞噬奥义和吞噬武魂，本就非常的契合，可谓是同根同源，融合起来难度不大。
<br/>　　仅仅是一个时辰的时间，两者便完美的融合在了一起，让得吞噬武魂充满了玄妙的大道气息，威力更甚。
<br/>　　接下来，就是最难的融合神魂了，成功，他便能晋升武帝之境。
<br/>　　这是最难的一步，亦是最关键的一步！
