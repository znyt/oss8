　　怪物移动的动作顿时慢下来，随后大吼一声，身体抖了一下，将三道火蛇挣断，放弃了毛小方，转身朝妙心扑过来。
<br/>　　妙心早有准备，当下身体一侧，躲了过去，从包里摸出一把长长的木质的法器，在叶少阳看来，很像是哈里波特用的法杖。
<br/>　　妙心向后退了几步，就不再移动，面对着再度扑过来的怪物，表情虽然有点紧张，但更多的还是镇定。
<br/>　　“小心！”叶少阳离得远，也帮不上忙，大声对妙心喊道。
<br/>　　怪物一步冲到妙心面前，抬起了一只爪子，刚要去抓妙心，妙心突然又念了一波咒语，面前突然升起了几道青光，奔着怪物过去，瞬间将它裹住，好像刀子似的，在它浑身上下切割起来。
<br/>　　怪物不得不放弃了妙心，蹲坐在地上，身体用力扭动起来，显然是那几道青光对它造成了一定的伤害。
<br/>　　叶少阳等人看见这一幕，顿时惊呆了，朝那怪物的脚下望去，看到了数道青色的刻痕，在怪物身下不断闪现出来。
<br/>　　因为怪物的身体就在上面，压制了这些刻痕的大半，不过叶少阳还是立刻就判断出，这是一个用不知道什么材料写在地上的符印，顿时恍然大悟，不用说，这符印自然是妙心画在地上的，怪不得她有恃无恐，原来早就做好这一手准备，只是之前大伙慌乱，注意力都在怪物的身上，都没有看到妙心是什么时候在地上画下符印的。
<br/>　　不过，她这一手，却是让叶少阳内心生出几分赞赏。这个妙心，不愧是地师传人。
<br/>　　除了妙心，还有一个人反应也是极快：梧桐。
<br/>　　在妙心暂时用符阵困住了那只怪物之后，梧桐立刻从包里摸出了一件法器，对着怪物丢了出去，同时双手合十，念咒道：“金莲常转，佛法无边，一点灵光，造化珈蓝！”
<br/>　　叶少阳听到“金莲”两个字，内心震动了一下，再看那件法器，却是飞到了怪物的头顶，在空中悬浮起来，一边旋转着，放出了淡金色的光芒，如同灼热的火焰一般，一浪一浪地朝那怪物身上卷过去。
<br/>　　金莲业火！！
<br/>　　没有错，这个悬浮在怪物头顶的莲花形状的法器，就是峨眉山的五宝金莲！！
<br/>　　叶少阳内心一下子就震动了，望着大杀四方的五宝金莲，口中忍不住喃喃吐出一句话：“不会这么巧的……”
<br/>　　是的，没有道理这么巧。
<br/>　　五宝金莲……在自己所在的那个时代，是芮冷玉的贴身法器……虽然原本不是她的，但还是因为一场机缘，最后成了她的专属法器。
<br/>　　而时隔九十多年的今天，五宝金莲，却是出现在了一个跟她长的一模一样的姑娘手中，这……绝对不可能是巧合。
<br/>　　就算梧桐是峨眉山弟子，手上有五宝金莲也是应当，叶少阳还是不相信事情会这么巧，而且……他想到了在早前的时候，不止一个人跟芮冷玉说过她有佛缘，最最重要的是，芮冷玉差一点就在峨眉山出家当了尼姑……
<br/>　　这一切的一切，如同一条条的线索，浮现在了叶少阳的心中，在这种突如其来的情绪之中，叶少阳突然想到了一个只会在小说中出现的词汇：重生……
<br/>　　这梧桐，不会是芮冷玉重生到这个世界的吧？
<br/>　　但是立刻，叶少阳又摇了摇头，这个解释……真的是太扯淡了。就算芮冷玉在自己那个时代已经……丧命了，也没道理会重生到这个时代来吧？
<br/>　　不过，既然自己都能穿越，那么重生似乎也不是不可能发生的事。
<br/>　　但是尽管如此，叶少阳还是相信梧桐不是芮冷玉重生的，有很多原因，其中最重要的一点，就是他相信芮冷玉一定不会死。
<br/>　　后卿，一定会好好保护她的，虽然这个念头让叶少阳心里有点酸，不过他还是相信后卿的实力，毕竟他背后还有着一个强大的尸族……他绝对保护得了芮冷玉。
<br/>　　那么，为什么梧桐和芮冷玉之间会有这么多的巧合呢？
<br/>　　叶少阳心中思绪翻飞，想到了这样那样的可能，一时间竟然忘记了自己的处境，直到毛小方在他肩膀上拍了一把，“你掉魂啦！”
<br/>　　叶少阳猛然回过神来，定睛看去，卢晓清、陈晓宇和曹雨兴都已经对怪物出手了，各自祭出法器，彼此间用灵力连成了一个结界，将怪物封印在了中间，一时间动弹不得，只能被法器不断地消耗，身上已经皮开肉绽，暗红色的血液不断从伤口流出来，偌大的墓室里充斥着刺鼻的血腥味。
<br/>　　终于有工夫好好打量这只怪物了。
<br/>　　叶少阳定睛望去，嘴角顿时抽搐起来，口中忍不住爆出一句话：“尼玛恐龙啊这是！”
<br/>　　“尼玛恐龙，那是什么？”毛小方很正经地提问。
<br/>　　叶少阳昏了，差点一口血喷出来，瞪了他一眼说道：“恐龙！跟尼玛两个字没关系！”
<br/>　　“哦，那你为什么要说尼玛？”
<br/>　　“感叹词啊大哥！”
<br/>　　“哦，那恐龙又是什么，一种邪物的名字？”毛小方锲而不舍。
<br/>　　叶少阳真要昏了，转念一想，这才明白，这个时代好像是没有“恐龙”这个名词的，就算有，普通人肯定也不知道。
<br/>　　再细看那个怪物，体型巨大，有两三米高，长长的脖子都快顶到墓室的顶部了。不过，在一番打量之后，叶少阳觉得说它是恐龙有点不准确，确切说，是一只巨大的壁虎……不，蜥蜴。
<br/>　　体形上几乎完全一样，只是两只眼睛吊上去，嘴巴撇下的线条也更大，不过最大的差别，还是这怪物的嘴角下面有两道胡须，这模样，看上去又有点像是龙。
<br/>　　想到水井边上的那行字，叶少阳猛然间想到了这怪物的名字：“蝾螈，我知道了，这是蝾螈！”
<br/>　　大伙本来在聚精会神地对付这怪物，听到叶少阳的喊声，都惊了一下，然后脸上露出了不可置信的表情，不过这一次，曹雨兴和陈晓宇，都不得不认同叶少阳的猜测。
