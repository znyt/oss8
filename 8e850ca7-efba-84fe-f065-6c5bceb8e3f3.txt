　　“叶少阳那边的关系就不用说了，这林三生等人都是阴司正神，不久前咱们才喝过阴阳司的酒，马面还被小青那货灌醉了，拜了把子，你好意思动手啊？”
<br/>　　马面脸上一红，“说的你老八没吃过橙子公主的桂花糕似的，让马爷我出头？”
<br/>　　白无常道：“够了够了！打是要打的，只是切莫伤了他们，抓捕即可。”
<br/>　　黑无常道：“抓了叶少阳这小子，再抓转世鬼童，等把鬼童交上去，他也无奈何了，自然不会再闹。大家朋友一场，也算是帮他一把。”
<br/>　　这番话说得连他们身后那些功曹鬼差也一起点头。
<br/>　　于是白无常一声令下，一起冲了过去。捉鬼联盟的人一起抵挡。双方都是熟人，也都抱着不伤害对方的目的，一方想拖时间，给叶少阳时间去找芮冷玉，一方也是只想抓人不想伤人，于是战斗跟演习似的，一点也不激烈。
<br/>　　道风撇下了杨宫梓等人，自己又回到山谷，看见这奇葩的战斗，立刻就知道怎么回事，也不管他们，从上空飞过，去找叶少阳。
<br/>　　在山谷深处，叶少阳总算找到了芮冷玉，芮冷玉是凭着本能在逃，发现前面没路了，立刻折返。叶少阳上去抓她，芮冷玉却挥剑刺来。
<br/>　　两把剑一交锋，叶少阳立刻感到了强大的压力：眼前的芮冷玉，比之前强大了不知道多少倍！
<br/>　　怪不得能够解开定魂符，从小九背上逃脱了！
<br/>　　叶少阳恍然明白过来，唤了几声，芮冷玉完全不理，似乎把他当成了追兵，一个劲地进攻。叶少阳只能抵挡，寻找机会想把她抓住。
<br/>　　“小玉，你真的不认识我了吗！”叶少阳一边出招，一边怔怔地望着她，怔怔地说道。
<br/>　　山谷外面，笛声再起，婉转凄凉。
<br/>　　摇光仙子又吹起了她的长笛。
<br/>　　音符不断飞出，如风一般从面前那些僵尸身上抚过，一路下去，那些僵尸血肉尽数化去，只剩下一副骷髅还站立着，之后陆续倒下。
<br/>　　面对逆境，这个十二金仙中真正的最强者，也被激发出了强烈的战意。不过尸兵实在太多，层层叠叠，情况越来越危急。摇光仙子靠着长笛吹出的仙音，也只能勉强抵挡，但她修为毕竟是有限的……谁知道援兵什么时候来，摇光仙子很怕自己撑不住了。
<br/>　　就在这时，从对面树丛里冲出一队人马，摇光仙子定睛看去，立刻认出领队的那个翩翩长裙的姑娘，眼前一亮，停止吹笛，说道：“十娘！”
<br/>　　来的正是巡游司的现任司主，十娘。自从鱼玄机飞升修罗之后，她继承成为司主。
<br/>　　不过，她还有一个身份，乃是梨山老母的嫡传弟子。
<br/>　　十娘身后，跟着数百个鬼兵，看到黎山老母，也是眼前一亮，“小师叔，你怎么样？”
<br/>　　“还好，速来帮忙！”
<br/>　　十娘立刻率领手下一众鬼兵，杀入阵中。她亲自冲到摇光仙子身边，帮她分担压力。
<br/>　　摇光仙子喘了口气，问道：“你怎么会来，还带了这么多人，不怕大帝怪罪？”
<br/>　　“是大帝懿旨，让我来救援你等。之前还有大军，牛头马面他们……还没到？”十娘左右看去，没看到一个阴司的人，不禁好奇。
<br/>　　摇光仙子怔了一下，恍然明白，原来先来的那一拨人，不光是为了抓转世鬼童，也是为了救援空界……他们没有出手，倒不是真的见死不救，而是知道后面还有援军，只是故意晒晒自己，为崔府君出一口恶气。
<br/>　　摇光仙子道：“他们去山谷里抓转世鬼童了。”
<br/>　　“哦，我师父呢？”
<br/>　　“在山谷里，你在这挡住，之前女魃进去了，我不放心，进去看看！”
<br/>　　摇光仙子说完，自己杀出重围，冲进了山谷。
<br/>　　“怎么会这样！”
<br/>　　陷入重围的雪魔，真是有一种欲哭无泪的冲动，肠子都悔青了，不知道当初为什么答应叶少阳帮他对抗空界联盟，先被尸兵轮了一波，现在又被阴兵轮……要不是因为天池罗刹被叶少阳收了、自己又撑了这么久，已经损失了大批的人马，就这么自己一个人回去的话，实在是太丢人，不然他早想一走了之了。
<br/>　　雪魔现在只盼着叶少阳能保护芮冷玉离开，自己好回去交差，于是咬着牙继续苦苦支撑。
<br/>　　黎山老母等一些伤者已经退到了山上，没有任何危险，只有苏沫，在调息了一个周天，恢复部分法力之后，仍然留在山谷里，躲起来观战。
<br/>　　女魃带领一群飞僵，围攻空界强者，杀死一些之后，把目光瞄到了在山上观战的黎山老母等人，喝令手下一起上山。
<br/>　　自己飞到半空中，转头看着下方还在缠斗的叶少阳和芮冷玉，杀机顿起，觉得反正也是举手之劳，于是又纵身折返，对着芮冷玉俯冲过去。
<br/>　　出于一种她不愿承认的嫉妒心理，他不想让芮冷玉活下去，不管她是人，还是什么转世鬼童，都是一样的。
<br/>　　叶少阳看到女魃从后面袭击，立刻扑过去，祭出阴阳镜，画符念咒，射出一束金光，挡住女魃。
<br/>　　“叶少阳，我记得你呢。”女魃邪魅一笑，数道血光在身上环绕，形成了一道坚不可摧的结界，顶着阴阳镜射出的灵光，就这么从空中一步步走了过来。
<br/>　　这种实力，叶少阳清楚，根本不是现在的自己所能对抗的，一咬牙要念动剑诀，突然左肩上一麻，转头看去，是一把剑，刺穿了自己的胳膊。
<br/>　　“小玉……”
<br/>　　叶少阳望着一脸狠色的芮冷玉。自己为了救她挡住女魃，她却从背后刺了自己一剑，虽然知道她现在迷失了本心，不过叶少阳还是感到一种哀伤。
<br/>　　“砰！”
<br/>　　一道黑气撞在她太阳穴上，直接打飞了出去。
<br/>　　“小玉！”叶少阳浑身一颤，转头看去，是道风追了过来。
<br/>　　道风二话没说，上前去战女魃。
<br/>　　叶少阳立刻朝芮冷玉奔去。
<br/>　　芮冷玉从地上站起来，弯着腰大口吐血，然后嘴巴里发出类似低吼的声音，周身环绕的煞气愈发猛烈，火一般燃烧。
