　　望着怀中温柔的佳人，慕清霄很难从她现在的模样，联想到昔日里称霸塔戈尔大沙漠的美杜莎女王。
<br/>　　说来也是，现在的她是彩鳞，昔日的美杜莎女王早已经是过去式。
<br/>　　抚/摸着如凝脂一样的肌肤，慕清霄食指大动，忍不住翻身压了上去，随后屋内再度响起一阵婉转低吟……
<br/>　　半个时辰后，彩鳞化作七彩吞天蟒，金色的蛇瞳幽怨的瞪了一眼慕清霄，摇了摇尾巴，盘在了他的手腕上。
<br/>　　“等将斗气全部转化成灵气，本王才会出关，接下来的日此就由仙儿服侍你吧。”
<br/>　　其实，彩鳞的言下之意就是，接下来的一段时间，你就去祸害仙儿吧。
<br/>　　慕清霄苦笑着摇了摇头，将屋内稍稍收拾了一下，转身朝阳台上走去。
<br/>　　此时，小医仙依旧静静的盘坐在原地，周身至寒灵气缭绕，两个时辰过去，她的气息已经暴涨了一大截。
<br/>　　在这一刻，她的气息已在金丹九重巅峰停滞，估计到天亮，斗气就能全部转化成灵气。
<br/>　　收回目光，慕清霄微微闭目，神识释放而出。
<br/>　　神识顷刻间就将整个迦南学院笼罩在其中，随后在外院一座悬崖边定格。
<br/>　　与此同时，迦南学院外院一座悬崖边，一道身影盘坐于此。
<br/>　　黑色巨尺斜靠在巨石上，萧炎嘴中叼着一根狗尾巴草，今日看到斗圣君临之威后，心里感到极度的不平衡。
<br/>　　五年前，慕清霄已经是斗皇境界，他刚刚进行修炼。
<br/>　　五年后，他有药老的帮助，以最快的速度突破了大斗师境界，为此他还沾沾自喜。
<br/>　　但今日，慕清霄的出现，将他的自信打击的体无完肤，五年后已是斗圣，让他有种五年都活到狗身上去了的感觉。
<br/>　　“药老，慕师兄能在五年内突破斗圣，是因为焚决，或者说异火？”
<br/>　　萧炎徐徐开口问道，毕竟焚决可是能够超越天阶的斗技。
<br/>　　话语刚落，戴在他手中的黑色纳戒一亮，旋即一道苍老的身影漂浮而出。
<br/>　　“焚决你也修炼了，就算异火在恐怖，你觉得能让一名斗皇在五年内突破斗圣吗？”
<br/>　　其实，药尘本不想将焚决交给萧炎，因为他清楚慕清霄在收集异火。
<br/>　　可修炼其他斗技，萧炎的修炼速度实在是太慢了，最终他依旧是将焚决教给了萧炎，并且一路收集兽火，让焚决进化。
<br/>　　五年内经历了坎坷，他也看着萧炎成长，本认为速度已经非常的快，但今日的事情，完全将他的世界观颠覆了。
<br/>　　“小炎子，别多想了，世间本就不公平，清霄的资质与你不是一个级别，心中不平衡也没用。”
<br/>　　叹息一声，药尘道：“你修炼了焚决，大陆上种族林立，将兽火全部收集到，焚决未必不能超越天阶，为师不敢说斗圣，令你修炼到斗尊境还是有把握的。”
<br/>　　就在这时，后面的空间清微一阵波动，一道白袍身影已经出现在了萧炎与药尘的身后。
<br/>　　“老师，五年不见。”
<br/>　　清朗熟悉的声音在耳边响起，药尘转过身来，唏嘘道：“是啊，五年不见，为师当真感到欣慰。”
<br/>　　“老师，当年的事情一周后就能准备好，后续的灵药，等我去中州的时候，在为您准备。”
<br/>　　俗话说一日为师，终生为父，慕清霄与药尘相处的时日虽不长，但心中对他依旧是有一丝尊敬之意。
<br/>　　哪怕他现在的修为，早已将后者远远的抛在身后。
<br/>　　“好好，你有这份心就够了。”
<br/>　　药尘眼底激动之色一闪即逝，若是能够复活，谁想成为魂体，有慕清霄一句话，复活已是板上钉钉的事情。
<br/>　　或许，连魂殿都不需要太过忌惮。
<br/>　　慕清霄目光落在萧炎身上，明显能够察觉出后者的情况，虽修炼了焚决，体内也有一股不弱的火焰，但并非异火。
<br/>　　“小子，怎么闷闷不乐？”
<br/>　　萧炎将嘴中的狗尾巴草丢掉，道：“人比人死，货比货扔，只是有些感慨而已。”
<br/>　　慕清霄微微一笑，道：“我这有很多灵药，能够支撑你迈入斗尊，有没有兴趣和师兄做个交易？”
<br/>　　“斗尊……”
<br/>　　话语刚落，萧炎心脏狂跳了几下，但没有几个答应，道：“什么交易？”
<br/>　　慕清霄直言会语道：“你和老师都知道，我在收集异火，想必老师不可能不知道‘陀舍古帝’。”
<br/>　　药尘沉吟片刻，道：“陀舍古帝，是大陆上的传说，各种古籍中都有一些记载，但资料全都零零散散，只记载有一座斗帝洞府，可位置谁都不知道。”
<br/>　　“不错。”
<br/>　　慕清霄轻轻点头，药尘当初是八族其一药族的成员，知道一些关于陀舍古帝的信息也实属正常。
<br/>　　萧炎听的云里雾里，但他不明白，为何与他的交易中会牵扯到斗帝洞府，他只是一个大斗师，怎么可能与斗帝扯上关系。
<br/>　　“萧炎师弟，你手中肯定有一块残缺的碎片。”
<br/>　　“你怎么知道？”
<br/>　　萧炎一愣，将戴在脖子上的碎片取了下来，道：“这是父亲交给我的，听说是萧家祖先传下来的东西。”
<br/>　　望着萧炎手中的残缺玉佩，慕清霄并没有出手抢夺，道：“老师，你对远古八族的存在，应该知晓一些信息。”
<br/>　　“远古八族！”
<br/>　　药尘思索片刻，蓦然一惊，目光落在萧炎身上，失声道：“炎小子姓萧，难道……”
<br/>　　“萧族虽已经没落，但依旧是远古八族其一，萧炎是萧族的后代，而古薰儿，乃是古族的小公主，而他手中的玉佩就是关键。”
<br/>　　萧炎依旧是有些懵，但药尘却抓住了关键。
<br/>　　“你是说，炎小子手中的碎玉与传说中的陀舍古帝有关？”
<br/>　　慕清霄轻轻点头，目光看向萧炎，道：“你的父亲没有告诉你太多的信息，那是因为连他自己都不知道，你们萧家原本是大陆上最强的家族。”
<br/>　　语气一顿，继续道：“而你手中的碎玉便是开启陀舍古帝洞府的关键，其他七族手中各有一块碎片，八块碎玉集中在一起，那就是陀舍古帝玉，也是开启斗帝洞府的钥匙。”
