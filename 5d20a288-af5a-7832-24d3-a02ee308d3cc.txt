        能拥有如此好听的声音的男人，一定配得上一副帅气的皮囊。
<br/>
<br/>        然而，在童筱颖心里，这个男人就是一只恶魔。
<br/>
<br/>        脸上这两巴掌的“痛”，倒是令童筱颖长记性了。
<br/>
<br/>        这辆极其拉风的加长型的豪华座驾的后座车门已打开，但童筱颖却准备绕道走。
<br/>
<br/>        原本以为此事就此了之，童筱颖却被一个穿着西装革履，戴着墨镜的保镖给拦住。
<br/>
<br/>        “小谢小姐，十爷请您上车。”男保镖一丝不苟地说道。
<br/>
<br/>        童筱颖不予理会，转过身去准备离开的一瞬间，手臂上突然传来一阵剧痛，整个身子硬生生地被人给拽住往后拖。
<br/>
<br/>        下一刻，她便被那人强行塞进了车内。
<br/>
<br/>        车内，除了她和司机以及那个塞她进车的保镖，还有另外两个男人，一个男人是十爷，另外一个戴着无框眼镜的男人，应该是十爷的助理或是秘书。
<br/>
<br/>        “你们抓错人了！我不是……”
<br/>
<br/>        “看样子，昨晚的那两巴掌并未让你学乖。”十爷冷漠地开口。
<br/>
<br/>        他仍旧戴着那副金丝假面眼镜，而假面眼镜虽然遮住了他大半面容，但是她仍旧觉得，他的声音和眼睛都和那位连少很像。
<br/>
<br/>        童筱颖本想敷衍了事，没想到话还未说完，就被他给揭穿了，顿时间让她有点儿语塞。
<br/>
<br/>        “脸还疼吗？”十爷接着问。
<br/>
<br/>        童筱颖瞥了十爷一眼，柳眉微蹙，毫不领情地偏过头去望向了车窗外。
<br/>
<br/>        他这算什么？
<br/>
<br/>        打了她后，再给她喂糖？
<br/>
<br/>        童筱颖不想理会这个男人，其实心里更多的是不想再见到这个男人。
<br/>
<br/>        “小谢，十八岁半，原名‘童筱颖’，又名‘诺筱颖’，是童家的人从人贩子手中买回来，交与梁家抚养，所以，你父母不详。你自幼跟梁家的外婆学了一点三脚猫功夫的医术，高中学习成绩不错，但因为你辍学打工照顾你那个梁姓爸爸的缘故，高考失利，现在在一所不入流的大专学校读大一。因童家欠债，你休学嫁人。丈夫是傅昼景，然而，你丈夫却欠了巨额外债，你现在几乎等于又是在辍学打工，帮你丈夫一起还债。”坐在对面的男人，推了推鼻梁上的无框眼镜，手里拿着资料，滔滔不绝地说。
<br/>
<br/>        童筱颖目瞪口呆地回过头来，看了看坐在对面的男人后，又看了看坐在自己身旁的十爷。
<br/>
<br/>        他们居然调查了她！是想做什么？
<br/>
<br/>        “筱颖小姐，我说得对吗？”那个戴着无框眼镜的男人又问道。
<br/>
<br/>        童筱颖屏气凝神地抿着嘴，不吭声。
<br/>
<br/>        这辆加长型的豪车内，非常的宽敞，玫瑰色的装饰，让周遭的空气急剧升温。
<br/>
<br/>        “你们到底想做什么？”童筱颖按捺不住地问。
<br/>
<br/>        “当然是带筱颖小姐去医院！”眼镜男微微一笑。
<br/>
<br/>        童筱颖惊愕地瞪大了眼睛，几乎是本能地往后躲了躲，但这里压根就没有她的退路。
<br/>
<br/>        “筱颖小姐不必害怕，我们只是想带您去做个鉴定。”眼镜男又接着说。
<br/>
<br/>        童筱颖下意识地看向身旁的十爷，弱弱地问道：“那个……我已婚，父母不详，没什么好鉴定的……十爷，你还是放我下车吧！”
<br/>
<br/>        “你再吵一句，我会让我的保镖打烂你的嘴。”十爷忽然冷斥了一声。
<br/>
<br/>        吓得童筱颖不敢再多说一句话。
<br/>
<br/>        到了医院后，护士给童筱颖抽了血。
<br/>
<br/>        那个十爷并未陪同她一起进医院，而这任务一完成，十爷的手下跟医护人员交代了几句后，不管童筱颖便直接走掉了。
<br/>
<br/>        所以，那位十爷，对她就是这种态度？
<br/>
<br/>        对于这种事情，童筱颖倒是想得挺开，也无所谓。
<br/>
<br/>        就在她转过身去的时候，一抹熟悉的身影，闯入了她的视野，她忍不住叫唤了一声：“雅姗？”
<br/>
<br/>        季雅姗准备进电梯的身影一滞，循声望去，看到童筱颖的时候，便只身向童筱颖走了过去。
<br/>
<br/>        “筱颖，好巧啊！你怎么在医院里？你哪里不舒服吗？”
<br/>
<br/>        其实，童筱颖也很想问这句话，但是季雅姗先问了。
<br/>
<br/>        “我是被那个十爷的手下给带来抽血的。”童筱颖忍不住跟季雅姗抱怨起来。
<br/>
<br/>        听到这里，季雅姗脸上的笑容渐敛。
<br/>
<br/>        童筱颖这才想起季雅姗和十爷的关系，尴尬地抿了抿唇，不知道自己接下来该说什么是好。
<br/>
<br/>        “希望，结果出来后，你是阿十的那个过世的妻子的妹妹就好……”季雅姗讷讷地说。
<br/>
<br/>        童筱颖却听得一脸茫然。
<br/>
<br/>        季雅姗深吸了口气后，缓了缓心情，拉起童筱颖的手，微微一笑地接着说道：“走！陪我去点心去！”
<br/>
<br/>        “可是，我还得去买菜！”
<br/>
<br/>        “买菜有我重要吗？我可是你的好朋友！”
<br/>
<br/>        “当然你重要。”
<br/>
<br/>        “那就先陪我去吃甜点。”
<br/>
<br/>        所以，就这样，童筱颖陪着季雅姗，进了一家甜品店。
<br/>
<br/>        昨晚的事情，童筱颖跟季雅姗道了谢，季雅姗却说这是应该做的事情。
<br/>
<br/>        “筱颖，我很珍惜你这个朋友，所以，你不要嫌弃我！”季雅姗忽然拉住坐在桌子对面的童筱颖的手，一脸认真地说道。
<br/>
<br/>        “我没有嫌弃你！”童筱颖也非常认真地回应。
<br/>
<br/>        季雅姗这才眉开眼笑。
<br/>
<br/>        童筱颖见她终于开心了，脸上也露出了欣慰的笑容。
<br/>
<br/>        “阿十他有过一个妻子，五年前死于一场意外，我不知道他妻子的真名叫什么，也不知道他妻子是什么背景，更不知道他妻子长什么样子。不过，我只知道大家都叫他妻子‘婴宁’，而且婴宁还有一个失散多年的妹妹。能找妹妹，是阿十的妻子生前最大的心愿。”季雅姗黯然神伤地絮絮念叨。
<br/>
<br/>        她不是在为自己伤感，而是在为阿十伤感。
<br/>
<br/>        为什么她要为他伤感呢？
<br/>
<br/>        明明那么想逃离他的束缚。
<br/>
<br/>        季雅姗的眼神里夹杂着太多的说不清道不明的感情。
<br/>
<br/>        童筱颖无法去体会季雅姗的情愫，但作为“旁观者清，当局者迷”，她却看得出一点：“你喜欢十爷？”
<br/>
<br/>        “我？怎么可能！我不喜欢他！他比我大了十岁！我才十八岁！我不喜欢他！”季雅姗矢口否认。
