
<br/>　　<br/>　　    晚上九点多，丽景湾酒店的地下停车库里，沈君的三个马仔开了一辆面包车等了有半个多小时了，沈沉鱼的经纪人从剧组那边打听到刘珺瑶拍完自己的戏份之后就被人接走了，不知道几点才会回到酒店，三个马仔就在停车库里，守株待兔的等着了。<br/>　　<br/>　　    又过了半个多小时，酒店外面的一条马路上，一辆黑色的奔驰小跑往地下停车库方向开了过来，副驾驶上坐着刘珺瑶，开着的是个三十岁左右的青年，他俩刚从外面吃完饭回来。<br/>　　<br/>　　    “沈沉鱼今天没有来剧组，说是请假了，我让导演问她经纪人是怎么回事，他们的回复挺含糊的，也没说什么时候在复工，我估计沈沉鱼是真的出事了”刘珺瑶慵懒的缩在座椅上，抻着小蛮腰，曲线凹凸有致，表情看着司机显得异常妖娆。<br/>　　<br/>　　    贵圈很乱，这个圈子里充斥着各种肮脏的交易和尔虞我诈，就拿沈沉鱼和刘珺瑶来讲，两人在表面上是勾肩搭背的好姐妹，外人看见的都认为两人之间的关系已经到了闺蜜那一层。<br/>　　<br/>　　    但刘珺瑶在背地里一直都对沈沉鱼很不满，这全都是嫉妒心在作怪了，沈沉鱼背景比她好，天分比她强，名气也比她大了一点，综合以上因素的话，刘珺瑶的心里一直都觉得，沈沉鱼是她最有利的竞争对手。<br/>　　<br/>　　    开车的青年说道：“我告诉你的办法肯定管用，而且她出了事之后，根本都查不出一点异样来，就算是报警了，怎么查都查不出什么，你放心就是了”<br/>　　<br/>　　    “咯咯，胡胡你对我可真好，我得怎么谢谢你呢？这个沈沉鱼啊，挡了我的路，这部戏要不是她接了的话我就是女一了，哎呀，没办法啊谁让她有个好哥哥呢，呵呵，不过以后可不会有这种事了，她再也没机会和我争了，圈里的人都把她跟我说成是内地今年最红的两朵花旦，两朵怎么能行呢，花还是得一枝独秀啊”<br/>　　<br/>　　    胡胡扭头打量着一脸媚态的刘珺瑶，说道：“那你想怎么谢我啊？”<br/>　　<br/>　　    刘珺瑶眨了眨眼睛，朝着胡胡喷了口香气说道：“晚上我再给你换个新的姿势，一字马的怎么样？”<br/>　　<br/>　　    “啪”胡胡一巴掌拍在刘珺瑶浑圆的大腿上，说道：“再把那个兔耳朵也带上，我最近比较喜欢小动物……”<br/>　　<br/>　　    几分钟后，奔驰小跑开进地下车库。<br/>　　<br/>　　    “我先下车进酒店，你过几分钟再过去，免得让人看见了”车停好后，刘珺瑶跟胡胡说了一声，就推开车门下来了。<br/>　　<br/>　　    “嗯，去吧”胡胡把车熄火，然后点了根烟等着。<br/>　　<br/>　　    明星们的偷腥，基本都是这种路数，两人是不可能一起出现在电梯和酒店里的，被拍到了就不好解释了。<br/>　　<br/>　　    刘珺瑶下车后挎着包就往停车场的电梯那边走，沈君的马仔看见她的身影后，其中一个就说道：“人带走，直接塞到车里，别等她进电梯在干，那边有摄像头，这里是监控死角，你们动作利索点就行了”<br/>　　<br/>　　    “咣当”两个人从车里下来，压低了帽檐，快步朝着过来的刘珺瑶走了过去，当双方擦身而过的时候，刘珺瑶就猛然感觉有人从身后楼上了自己的脖子，她刚“啊”的叫了一声，紧接着嘴巴就被捂上了，两个马仔直接就把她给抬了起来，夹着刘珺瑶的两腿和脑袋快速的朝着自己的车走了过去，顺着打开的车门就把人给扔到了后座上。<br/>　　<br/>　　    整个下手的过程前后没到两分钟，干脆利索，动作连贯，地下停车库里的摄像根本都没有捕捉到这个画面。<br/>　　<br/>　　    监控没捕捉到，但胡胡看到了，他离刘珺瑶被绑走的这个方向距离并不太远，所以她出事的时候，从车里下来正抽烟的胡胡，就正好看见了这一幕。<br/>　　<br/>　　    虽然看见刘珺瑶被两个人带走了，胡胡却没妄动，只是皱眉看着这一幕，他和刘珺瑶的关系说白了就是戏子和嫖客的故事，刘珺瑶贪图胡胡的能力和关系，对方看上的则是她的身材和脸蛋，可能还有今晚的兔耳朵和一字马。<br/>　　<br/>　　    但他们之间，肯定没有爱情。<br/>　　<br/>　　    “唰”一个马仔掏出一把尖刀，顶在了刘珺瑶的下巴上，低声说道：“别动，别吵也别闹，我就是不一刀捅死你，在你的脸上刮了几刀的话，你后半辈子是不是也毁了？你们这帮拍戏的就指望着这张脸蛋吃饭呢，脸没了，还怎么混啊？”<br/>　　<br/>　　    刘珺瑶惊恐的望着车里的人，哆嗦着问道：“你，你们要干嘛，绑，绑架啊？我，我有钱，可以给你们钱，要多少都行”<br/>　　<br/>　　    “啪”马仔抓着刘珺瑶的头发，说道：“抓你，跟钱没有关系，知道么？开车，快走……”<br/>　　<br/>　　    开车的马仔发动车子，驶离了地下停车库。<br/>　　<br/>　　    于此同时，胡胡稍微犹豫了下，就又重新上了奔驰小跑，然后尾随了过去。<br/>　　<br/>　　    胡胡不见得会英雄救美，但总归得搞明白是怎么回事，因为他跟刘珺瑶的关系虽然外界大部分的人都不知晓，但刘珺瑶的经纪人和助理是知道的。<br/>　　<br/>　　    刘珺瑶在这之前是跟自己呆在一起的，如果今晚她出了事的话，胡胡觉得整不好自己也会被连累上，所以就打算跟着过去看看。<br/>　　<br/>　　    奔驰小跑离着三四个车位，不远不近的跟着前面那辆车。<br/>　　<br/>　　    车里面，马仔给沈君打了个电话，告诉他一切顺利人已经被带走了，沈君告诉了他们一个地址，让把刘珺瑶送过去。<br/>　　<br/>　　    二十多分钟后，车开到了一个公司的后院，停在了一栋仓库的门前，刘珺瑶被从车里拖下来后，就送进了仓库里。<br/>　　<br/>　　    胡胡也跟到了这边，在路边停好车后，熄火关灯，扭头看着院内。<br/>　　<br/>　　    他在琢磨着，自己应该咋办，但他绝对没有想现在就冲进去把人给救出来。<br/>　　<br/>　　    戏子无义，玩戏子的人，那就更谈不上这个词了。
