　　　　看着虚空中的女娲娘娘，玄天皱起了眉头。
<br/>　　对于女娘娘娘，玄天心里还是有些复杂的，女娲毕竟是人族的圣母，是她创造了人族，可以说，没有女娲，就没有今天的人族。从这点来说，人族还应该感激女娲。
<br/>　　但，女娲娘娘的做法实在是令人族心寒不已。放任妖族来无情屠杀人族不说，那是因为她夹在两族之间，偏向谁都不好，她不管不问，玄天虽然心中不爽，但还是能够理解的，但，她竟然还要在玄天灭杀帝俊前阻拦他，甚至用圣威来压迫玄天，这就非常过分了。
<br/>　　人族好歹也是你的子女，你就这样在他们面前偏向妖族，这样真的好么？这一刻，他对于女娲娘娘完全没有了好感。
<br/>　　玄天皱了皱眉，凭立虚空，眼中已经恢复了淡然，看不出丝毫的情绪，看着眼前的女娲，他淡淡出声嘲讽道：
<br/>　　“圣人金贵，降临人族，恐怕不是为了来看看人族吧，更何况人族鄙陋，恐怕会污了圣人的圣眼。”
<br/>　　九天之上的女娲娘娘，瑞气笼罩，祥云环身，听闻此言，有些不悦的皱了皱眉头，她看着玄天的眼睛，好似想要知道他的想法，可惜，她失望了，从玄天眼中她看不出丝毫的情绪。再看看下方不断被屠杀的人族，不知为何，她的心在隐隐作痛，内心深处，有一股淡淡的后悔，但是一瞬之后，她便将这股情绪斩断，对着玄天淡淡道：
<br/>　　“玄天道友，可否看在本宫的面上，放妖帝与东皇一马？”
<br/>　　听闻此言，玄天眼中的失望之色更甚了，他心中更是愤怒异常，噌的一下火气就上来了。妖族是你的亲人，人族就不是了吗？为了妖族，你不帮人族也就算啦，如今还来人族替妖族求情，之前帝俊屠杀人族之时，怎不见你来？
<br/>　　“圣母不公！”
<br/>　　下方的人族燧人氏等人族，听到女娲此言后，大声喊道。他们此刻心中对女娲产生了很大的失望，不过，女娲终究是人族的圣母，母亲对儿子不公，难道儿子就要对母亲不利？儿子终究是母亲生的，母亲可以不公，儿子却不可以不孝。
<br/>　　他们强忍着心中的愤怒，对女娲已经不再抱有什么希望了。
<br/>　　玄天深吸一口气，气急反笑，大声道：
<br/>　　“好！好！好！女娲，你贵为天道圣人，之前既然不干涉妖族屠杀人族，现如今又何必来管人族灭杀妖族？妖族是你的亲人，难道人族就不是了吗？你这样做不怕人族对你心寒吗？”
<br/>　　“人族不敢！”
<br/>　　人族中听到玄天此话后，连忙低头大声叫道。
<br/>　　“哈哈哈！你听听，你看看，你的子女们对你多孝顺啊！”玄天对着女娲大声讽刺道，接着对着下方跪拜的人族大声喊道：
<br/>　　“人族，虽然是后天种族，但岂能由他们容辱？做人要有骨气，天行健，君子以自强不息，地势坤，君子以厚德载物！就算我人族死亡，也要抗争。我人族，不敬天，不拜地，不敬鬼神，不拜神佛，不惧魔神，当自强不息，迟早有一天，我人族能够践踏这天地。”
<br/>　　“我们一日是人，永生永世是人，我们将走出我们自己的人道，不属于仙道，不属于佛道，不属于妖道，不属于魔道，不属于神道，而只能是人道。仙欲欺我，我必斩仙，佛要欺我，我必杀佛，妖要欺我，我必灭妖，魔要欺我，我必葬魔，神要欺我，我必戮神！今后，人族，只尊自己！”
<br/>　　“谨记武祖教诲！”
<br/>　　众人族听后，热血沸腾，他们内心深处涌现出一股逆天的勇气，面对神魔，毫不畏惧，反倒是燃烧起熊熊怒火，他们看着武祖玄天，心中更是树立起一道强大的信念。
<br/>　　这股信念，冲霄而起，不止让所有大能震惊，更让一些生灵，甚至包括妖族、巫族、练气士等敬佩，也让女娲圣人为之动容。
<br/>　　女娲看着下方的人族那股逆天的斗志、信念，心中却丝毫高兴不起来，反而有点隐隐作痛，好像有什么东西要离她而去一般。
<br/>　　玄天再次看向女娲，眼中已经恢复了平静，不带任何情绪，他淡淡道：
<br/>　　“女娲娘娘，看着你是人族圣母的份上，帝俊与太一此次可由你带走，就当是人族欠你的，用他们还了。从此以后，娘娘为人族圣母，也只是人族圣母，再也没有别的关系。
<br/>　　但是，若再有下次，那可就别怪我人族不手下留情了。若是真到那时，就算你女娲出手，我人族也不会理会。”
<br/>　　玄天话音刚落，女娲顿时就感受到了人族在自己身上的气运，少了一大半，修为倒是没什么变化，但修炼的速度，却是大大减少了。顿时，女娲看向玄天的眼神中充满冷冽，她心中怒气升腾，好似一个忍不住就要将玄天灭杀。但是，她知道自己不能，不能这样做，若是她当着人族的面真的将玄天灭杀了，那么她身上剩下的那一半人族气运，也会就此消失了。
<br/>　　女娲眼神冰冷，冷冷地看着玄天，玄天却面无表情，他心中对女娲已经不带任何感情了，仅存的那点亲情也早已消失不见，他看着女娲，就仿佛是在看一个普通的仙人。
<br/>　　“哼！”
<br/>　　女娲留在此地也无可奈何，心中愤怒不已，连带着看向帝俊与东皇太一的眼中中也充满了不忿，不过，她终究是没有多说什么，冷冷地看了玄天一样，再面色复杂的看了下方众人族一眼，一个转身，便消失不见了。随之一同消失的，还有帝俊、太一和白泽等剩下的妖族妖神。
<br/>　　玄天看着女娲离去，面无表情，谁也不知道他在想些什么，静立一会儿后，他转身，对着下方的人族，冷声说道：
<br/>　　“妖族已经暂时离去，但对于我们来说，人族危机还远没有解决，因为我们的敌人还有巫族，巫族背信弃义，背板联盟，反杀人族，最是可恶。接下来，将首阳山所有的巫族给我杀了！”
<br/>　　“杀！杀！杀！……”
<br/>　　人族挥动手中的兵器，大声喊道，吼叫声惊天动地，让人听了热血沸腾。接着，他们毫不畏惧地向着那些进攻人族的巫族杀去。
<br/>　　……
