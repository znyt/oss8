　　声音就响起在楚云天前方三米开外的地方，近到似乎都能感受到对方扑面而来的高阶新人类压迫感，只是楚云天无法夜视什么都看不到，不会知道其实站在他正对面的人并非开口说话的人。
<br/>　　两者有两人，一男一女，说话的年轻男子站在女人的后侧方，隐隐透露出一种臣服感。
<br/>　　这两人正是肖七和澹凰，他们没有进入阴谷，而是绕了个道用了最为麻烦的方式，直接从阳谷入口进来。
<br/>　　至于原因，肖七没有说，澹凰也不好问。
<br/>　　而凭借两人的等阶，或许对于其他人来说闯荡很困难的阳谷周边地带，对于这两人来说却不值一提，行走的轻松自如。
<br/>　　只是当她们一路走来到达制裁之地的时候，却没想到在通道的尽头遇到了一个老头，看起来瘦骨如柴饿了好几天的老头。
<br/>　　肖七蒙在面纱后的双眼露出一丝好奇，这里一片黑暗但对她这个等阶的人来说，毫不影响视觉。
<br/>　　楚云天不敢说话，他看不到黑暗里有什么，但他感受得到有人在看着他。
<br/>　　“你知道再往前是什么地方吗？”肖七开口，声音清丽。
<br/>　　楚云天一愣没想到两人中还有一个女人，随后试探性的说道：“圆形悬崖？”
<br/>　　肖七眉头一挑有些讶异，身后的澹凰则是一声惊疑，不知道制裁之地太正常，但是能说出悬崖二字甚至点出圆形，足以说明眼前这个老头的观察能力有多惊人。
<br/>　　制裁之地很大，从上到下的无数个制裁暗门，整个六门所有的惩罚都发生在这里，无数年来需要巨大的空间支持，如果在无法夜视的情况下单靠摸索，应当摸不出是个巨大圆形。
<br/>　　“你很懂几何？”肖七一语中的的问道。
<br/>　　楚云天不知该如何回答，笑道：“我是个研究人员。”
<br/>　　“难怪。”肖七同样嘴角勾起，继续道：“你进来找人？”
<br/>　　楚云天没想到对方的问话这么不走常规路，与之前的罗杰套路完全不一样，但他也并未多加思考便回答道：“我来实地考察。”
<br/>　　“噗——”澹凰差点笑出来，硬生生的强行忍住。
<br/>　　实地考察，考察什么？死人还是活人？普通人还是神秘家族？
<br/>　　肖七则是玩味的看了楚云天一眼，并没有再继续问什么，只是朝着澹凰交代了一句：“带上他。”
<br/>　　澹凰走上前将楚云天扛在肩上，跟着肖七便直接往前走。
<br/>　　楚云天吓了一跳，连忙喊道：“快停下！前面是悬崖啊！”
<br/>　　澹凰并未理会，肖七更是脚步不停，于是在楚云天心惊胆战之际，两人直接朝着前方一个跳跃。
<br/>　　啪嗒！啪嗒！
<br/>　　两声轻响，两人并未向下坠落，而是直接落在了平面上。
<br/>　　楚云天有些愕然，只是还不等他摸索观察，肖七和澹凰便再次起跳，连续的几次起落之后两人停下，声音也消失。
<br/>　　此时楚云天也察觉到，他们已经落地，并且直接由直线过来。
<br/>　　澹凰回头看了眼身后，依然是那处制裁之地的上方空间，只是他们已经到了对面，横跨而过。
<br/>　　肖七却并未停留，直接朝前走去，澹凰带着楚云天快步跟上。
<br/>　　中央的制裁之地的确是个空间巨大的深洞，但却不是尽头，对面依然有通道！只是如何从制裁之地的上方越过去，却极少有人知道，包括楚涵都不知道这条路。
<br/>　　楚云天屏息凝神，想要仔细看清身后漆黑一片，想要探究这两人是怎么过来的。
<br/>　　“五步一个悬空石阶，石阶是看不见的。”这时候的肖七出声，道出了这一秘密。
<br/>　　带着楚云天快步朝前走的澹凰一愣，没想到肖七竟然会跟一个老头说这些，甚至这个老头并没有开口询问，要知道肖七平时的话并不多，除非需要刻意伪装成另一个人。
<br/>　　肖七却没有解释什么，继续的大步朝前走，身披的白色长衫被走路带起的风吹鼓，隐隐露出妙曼的背影。
<br/>　　澹凰在身后看着有些呆，不禁落后了几步。
<br/>　　“跟上。”前方传来肖七冰凉的声音，好似一盆冷水，浇了澹凰一个透心凉。
<br/>　　澹凰连忙快步跟上，扛着楚云天一声不吭。
<br/>　　楚云天却从中观察到了一丝不对劲，这时候忽的前方一抹刺目的光线射来，让人睁不开眼。
<br/>　　狠狠的眯起眼睛，楚云天适应了一下后才发现他们已经走出通道，眼前的视野豁然开朗，早已不是之前那黝·黑的地方。
<br/>　　带着一丝惊奇感，楚云天并未着急观察他遇到的这两人，而是朝后忘了过去，却看到身后并不是他所想象的那样是一个黑黝黝的山洞，而是一个……
<br/>　　门！
<br/>　　方方正正的门，甚至还有木头镶嵌的边纹，上面好几层的浮雕层层叠叠，一路延伸到了好几米高。
<br/>　　肖七和澹凰还在往前走，随着视野的越发宽广，楚云天这才发现这是一个建筑物，或者应该说是……
<br/>　　宫殿！
<br/>　　那道门则更像是这个宫殿的正门，宫殿整个罩在了刚才的通道上面，看不到尽头也看不到高度，顶端没入在云层里，像是将刚才那整个制裁之地都罩住。
<br/>　　而整个宫殿更是呈现出一种气韵轩昂的感觉，只是颜色有些奇怪，并不明亮，而是无比暗沉的灰黑色，像是一座死宅。
<br/>　　楚云天忍不住有种感觉，这里住着一个很厉害的死人，这是他的墓穴……
<br/>　　终于在看清了这座宫殿的大致全貌后，楚云天这次扭头，想要看看将他带上的这一男一女长什么样。
<br/>　　但这时候的肖七却停下脚步，豁然转身面向那座黑色宫殿，面色凝重。
<br/>　　澹凰虽然不知道发生了什么，但却反应很快的立即跟着转身，站在了肖七旁边面向宫殿。
<br/>　　楚云天吓了一跳，先是被这两人的行为吓到，再是被这两人的打扮惊讶到。
<br/>　　尤其是这个女人，穿着也太古怪了，带着异族风情的面纱和雪白色拖地长衫，衣服却干净的纤尘不染，像来自千年前的异族公主。
<br/>　　就在楚云天愣住之际，肖七动了动，她轻轻提起裙摆的一角，而后朝着那座宫殿单膝跪下……
<br/>　　雪白色长衫轻抚在地面黄沙，带出一股尘埃，恭敬又虔诚，像是一个忠实的信徒。
