　　若说之前，苏莫一招击败金衫青年袁烈，众人只是稍稍惊讶。

<br/>　　“的确强大，天赋恐怖啊！”
<br/>　　“貌似，这还不是的巅峰实力！”
<br/>　　众人议论纷纷，惊叹不已。
<br/>　　以前，很多人以为下界是不毛之地，下界的武者，就算再厉害，也不过尔尔。
<br/>　　但是现在，很多人都改观了，看来下界的妖孽，不比神域的差啊。
<br/>　　“怎么可能？”连霆脸色铁青，如同吃了屎一样的难看。
<br/>　　连续两人，纷纷被一招击败，这不仅没有打压的上官昊，还让上官昊的声名大阵。
<br/>　　“一群废物！”
<br/>　　不远处，梁师兄看到这一幕，脸色阴沉如水。
<br/>　　袁烈是废物，这个白展朝同样是废物！
<br/>　　不过，他也知道，这个上官昊的战力的确强大，唯有真神境的修为才能压制。
<br/>　　远处的塔楼上，殿主遥望着这一幕，微微的点了点头。
<br/>　　苏莫的表现，让他很满意，不过，这貌似还远远不够。
<br/>　　在他看来，苏莫的实力，应该远远不止如此。
<br/>　　演武场中央，苏莫面色平静，击败了白展朝之后，便不再关注，再次转头看向连霆等人。
<br/>　　“还有最后一场了！”苏莫淡淡的说道。
<br/>　　连霆脸色阴沉，沉默不语，但是他在用意念，和身边的几人在交流。
<br/>　　“最后一战，不能再败了，你们几个一品真神谁上场？”连霆用意念，询问身边的人。
<br/>　　其实，连霆心中清楚，此战到现在，已经失去了意义。
<br/>　　因为，就算接下来，他们将上官昊击败，也失去了效果。
<br/>　　因为，上官昊已经表现出了强大无比的天赋，就算败给了真神，也依旧是顶尖天才。
<br/>　　这种天赋，足够拥有天九殿的资格了。
<br/>　　但是，即便如此，他们最后一战，也不能再输了。
<br/>　　“连师兄，要不我上去吧，一招就能击败上官昊！”一名紫衣青年，向连霆传音。
<br/>　　“不行，你一品高级真神，修为超过上官昊太多了，赢了也是面上无光！”连霆否决了，紫衣青年修为远超上官昊太多，不宜出手。
<br/>　　“要不……我上场吧！”又一名青衫青年道，不过，他的神情有些迟疑，显得信心不是很足。
<br/>　　“你的实力不能保证必胜啊！”
<br/>　　连霆叹了口气，对方一品低级的修为，战力有不是太强，很难保证必胜。
<br/>　　连霆有些犯愁了，因为他身边的几人之中，恰巧没有一品中级真神修为之人。
<br/>　　“你们到底谁上场？没有人上场的话，那我就走了！”苏莫见连霆等人，迟迟无人上场，朗声说道。
<br/>　　连霆闻言，正欲指派一人上场，就在此刻，异变陡生。
<br/>　　“连霆，你和上官昊一战吧！”
<br/>　　朗朗的声音，突兀的响起，浩浩荡荡，传遍了演武场。
<br/>　　众人闻言一怔，因为，这声音，乃是殿主的声音。
<br/>　　唰唰唰！
<br/>　　众人纷纷转头，遥望远处的塔楼，便见到殿主在塔楼的顶层，遥遥在望。
<br/>　　“这……！”
<br/>　　“殿主居然让连霆上场？”
<br/>　　“连霆可是二品高级的修为啊，两人的修为太不对等了吧！”
<br/>　　众人惊愕不已，连霆的修为，高达二品高级，而上官昊的修为却是只有虚神三变。
<br/>　　这不仅有虚神和真神这一个大境界的差距，更是有六个小境界的差距。
<br/>　　如此大的差距，根本就没有可比性。
<br/>　　连霆和那位梁师兄，也是愣住了，他们完全没有想到，殿主居然会掺和此事，还有如此吩咐。
<br/>　　苏莫眉头一皱，遥望塔楼之上的殿主，朗声喝道：“殿主，若是如此的话，那就我认输。”
<br/>　　对于这种比斗，苏莫完全没有任何的心情，因为完全是无谓的战斗。
<br/>　　不管他胜败与否，都是有弊无利，徒增暴露一切底细的风险。
<br/>　　“殿主，我还是算了，他修为这么低，战胜他我也面上无光！”连霆亦是开口了，遥望着殿主说道。
<br/>　　他比苏莫高六七个境界，实力完全不对等，这种战斗他也不需要。
<br/>　　“连霆，你不想击败上官昊？上官昊，你不想击败连霆？”
<br/>　　塔楼之上，殿主的声音淡淡的响起，略一停顿，继续道：“当然，连霆你的修为过高，我可以将你封印在一品中级的地步。”
<br/>　　“这……！”连霆闻言，面带迟疑之色，若是这样的话，倒是不错。
<br/>　　只不过，一品中级的修为，高了上官昊差不多三个境界，赢了并不值得庆贺。
<br/>　　虽然他的战力很强，输的可能性几乎不存在，但是这个上官昊就算能抗衡他十几兆招，也能声威大震了。
<br/>　　苏莫闻言，却是眉头紧皱了起来，他感觉这个殿主，貌似在故意给他制造强敌。
<br/>　　还是说，殿主是看出了他的底细，想要一探究竟。
<br/>　　然而，殿主却是并没有征求去的同样，轻轻的抬起了两只手掌。
<br/>　　只见其两只手掌虚抱，掌中源力涌动，双掌连连转动，双掌只见的玄力，片刻之间，便形成了一个微型的阵法。

<br/>　　殿主双手一样，源力组成的阵法飞射而出，跨越数万余的距离，直接没入了连霆的体内。
<br/>　　连霆身躯一阵，而后身上的气息急速下降，眨眼之间，修为便暴降到了一品中级真神的地步。

<br/>　　众人神情怔怔，不明白殿主为何如此。
<br/>　　不过，连霆却是深深的吸了口气，大步的向演武场中央走去。

<br/>　　苏莫的脸色很不好看，他知晓，自己一旦全力出手，底细将会暴露出来。
<br/>　　首先是混沌族的身份会暴露，其次，他的逆天的天赋会暴露，甚至是轮回分魂也会暴露。

<br/>　　连霆来到了场中，伫立在苏莫的对面，沉声说道。
<br/>　　“你出手吧！”苏莫没有拒绝，既然答应了比斗三场，那他也不会食言。
<br/>　　“很好！”
<br/>　　连霆见此，微笑了起来，他还担心苏莫不敢呢，既然敢那就好办了。

<br/>　　除非，上官昊为了藏拙，故意战败。
<br/>　　但是，作为一个天才，他相信上官昊不会故意战败。
