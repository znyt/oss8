　　中域。
<br/>　　九域，一向便是以中域为尊。
<br/>　　中域是九域之中最为庞大的一域，也是最为繁华的一域，其中的无数资源，造就了无数的天才，强者。
<br/>　　七大宗门之四，便是位于中域之中，足以看得出中域的强大，繁华。
<br/>　　中域边界。
<br/>　　一道身影，从这虚空之中，猛然飞了出来，这一道身影，正是楚风眠。
<br/>　　在北域之中，楚风眠可以肆无忌惮的飞行，以他的实力，绝对无人敢去阻拦他。
<br/>　　可是如今到了这中域，楚风眠还无法如此嚣张，七大宗门的实力，要比起北邙学院还要强得多，楚风眠在其中贸然行事，很容易引起不必要的麻烦。
<br/>　　在中域的一处山脉，楚风眠的身形，缓缓出现。
<br/>　　他看向中域的种种，目光之中，也是突然露出了几分怀念之色。
<br/>　　天绝峰，便是在中域之中，可以说前世的楚风眠，便是一直都在中域之中长大的。
<br/>　　如今再度回到中域来，也是让楚风眠的心中，有些怀念。
<br/>　　“万年不见，这中域，倒是翻天覆地一般的变化了。”
<br/>　　楚风眠看着周围，俨然已经是觉得极为陌生了，前世的他，也算是走遍过中域了，不过现在楚风眠一眼望去，这周围的景象，却是已经变化了，已经是跟万年之前，截然不同了。
<br/>　　万年时间，已经是物是人非。
<br/>　　“天绝峰，日后有机会，在回去看看吧。”
<br/>　　楚风眠看向远方，这正是天绝峰的方向，虽然来到了中域，但是一时间楚风眠也不好回到天绝峰。
<br/>　　天绝峰毕竟是曾经剑道之主的所在之地，现在很有可能，已经是落入到了七大宗门的控制之中，楚风眠贸然回去，很有可能就会被七大宗门的人发现。
<br/>　　倒时候楚风眠可就有着无尽的麻烦了，以现在楚风眠的实力，去跟七大宗门抗衡，也不亚于是在以卵击石。
<br/>　　“中域，倒是比起万年之前，还要繁华了。”
<br/>　　楚风眠扫视着周围，感叹道，比起北域的荒凉，中域可以说是遍地荒凉。
<br/>　　就在楚风眠面前的这一处平原之上，一眼望去便是有着大大小小数百座城池，每一座城池中都有着上千万人口，单单是这一片平原，就抵得上北域之中的无数国度。
<br/>　　人口众多，资源繁盛，这一切便是代表着中域的强大。
<br/>　　楚风眠感觉到，就连这中域地底的龙脉，都是比起北域要庞大的多，楚风眠来到这中域之中，他身躯之中的神脉似乎沟通地底龙脉的速度都要快得多。
<br/>　　几乎是无时无刻，都可以汲取这灵力，在北域的时候，楚风眠最多是感觉到可以恢复灵力，但是来到这中域，楚风眠居然是感觉到，他的神脉，似乎是足以抽取地底的龙脉之力。
<br/>　　“凝！”
<br/>　　楚风眠闭上眼，凝聚灵力，在他的手心之中，一枚灵石，缓缓的凝聚出来，这一枚灵石，通体晶莹，那是一枚皇品灵石。
<br/>　　在九域的任何一处，灵石都是绝对的硬通货，是极大的财富，这皇品灵石，更是许多强者都需要之物。
<br/>　　本来楚风眠利用他的剑意，化为的吞天神蟒，也只能够是在灵力极为浓郁的地方，才可以凝聚灵石的，不过随着楚风眠现在实力的增加，在加上这中域地底龙脉的强大，令楚风眠几乎是可以在中域的任何地方，都在凝聚灵石。
<br/>　　短短的瞬息之间，在楚风眠的面前便是足足凝聚出了上百枚皇品灵石。
<br/>　　要是被人看到这一幕，只怕是会惊讶的掉下眼睛来。
<br/>　　中域的地底龙脉之力，的确是极为强大，可是想要沟通，谈何容易，寻常武者只怕是一年时间都难以凝聚出几枚灵石来，所以根本无法像是楚风眠这样。
<br/>　　而楚风眠却是拥有神脉，可以说有着得天独厚的力量，沟通地底龙脉，轻松无比，挥手之间，都足以凝聚出无数皇品灵石来。
<br/>　　并且这地底龙脉的灵力，也几乎是无穷无尽，楚风眠也不需要像是在北邙学院的时候那般担心，怕被人发现，就算是楚风眠抽取一万年，这地底龙脉的灵力，都不会减少十分之一。
<br/>　　所以楚风眠便是可以轻轻送送的，随时抽取。
<br/>　　“有着足够的财富，便是有可能炼制真正的圣品灵丹了。”
<br/>　　楚风眠目光突然闪烁起来，想要超越生死，对楚风眠来说几乎是难上加难的事情，他必须要利用一些丹药，诸天奇物等，才能够增加他突破的可能。
<br/>　　不过大部分的丹药，对楚风眠这个级别的武者来说，都已经是没有任何意义了，像是一般的天级丹药，让现在的楚风眠吞下，只怕也就像是糖豆一样，没有任何的意义了。
<br/>　　在所有丹药之中，唯有对他现在还有着极大作用之物，便是传说中的圣品灵丹了，也被称为天级极品丹药。
<br/>　　虽然名为天级极品，但是更多人却是愿意将其称为圣品，因为这圣品灵丹，可是跟天级灵丹截然不同的存在。
<br/>　　圣品丹药，号称为圣，便是几乎对九域之中任何境界的武者，都有着大用的丹药。
<br/>　　曾经楚风眠吞下的九转夺天丹，也正是一枚圣品灵丹，足以夺天工造化。
<br/>　　楚风眠想要突破，超越生死，最好的办法，便是利用圣品灵丹，这才能够让楚风眠增加极大突破的可能性。
<br/>　　只是这圣品灵丹的价值，比起天级灵器都要高得多，足以是跟祖龙至尊剑，战龙宝车等，相提并论了，炼制所需要的每一种灵药，都是九域罕见之物。
<br/>　　可以说没有极为雄厚的财富，想要炼制圣品灵丹，就是一种痴心妄想的行为。
<br/>　　那九转夺天丹，也是当初的剑道之主，位列九域巅峰的人物，才可以寻找材料，请人炼制而来的。
<br/>　　本来楚风眠也没有想要炼制圣品灵丹的想法，但是现在，楚风眠突然之间足以凝聚灵石，近乎财富无限，才让楚风眠冒出来了这个想法。
<br/>　　楚风眠前世饱读古籍，这圣品灵丹的炼制方法，楚风眠倒是还真的记得，只是他当初一没有足够的灵力，二也没有材料，自然无法炼制，不过这一次，倒是可以一试。
