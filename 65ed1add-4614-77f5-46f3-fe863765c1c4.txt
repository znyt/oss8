　　说完又摸出一张灵符，想了半天，却想不起谢雨晴家的门牌号，又想到周静茹和老郭，也是知道他们住在哪儿，但说不出具体地址。老郭的地址他之前是有，不过具体的街道和门牌也忘了，而且丧葬行这个比较晦气，他怕老板或老板的后人嫌晦气，不愿帮忙。想来想去，干脆写下：石城警察局灵异事件调查科谢雨晴收。
<br/>　　这样只要送到任何一个警察的手上，不出意外都会找到谢雨晴，毕竟谢雨晴在警局的知名度很高，是有名的警花，还有一个当局长的老爸，石城警务系统没人不认识她。
<br/>　　写好之后，叶少阳又把鸡血石挂坠从他手中拿回来，把这个写着地址的灵符也卷起来塞了进去。
<br/>　　“这里面有地址，今年是1922年，等到了2016年……”他想了想日期，因为穿越过来有一阵子，具体日期对不上，干脆给了一个大致的日期，于是只好又把挂坠拿回来，抽出刚塞进去的那张灵符，在地址前面加了几个字，成为：请于2016年10月底交给石城警察局灵异事件调查科谢雨晴收。
<br/>　　这才重新卷好，赛回到老板手中。
<br/>　　“老板，你一定要帮我这个帮。一定！”
<br/>　　说完将大洋强行塞给他。十几块大洋，对做小买卖的来说，不是一笔小数目，老板有些动心，但是觉得这件事很匪夷所思，哭着脸说道：“就算我帮忙，可我也活不到九十多年啊。”
<br/>　　“不需要你活九十年，将来你传给你的子孙，让他们去完成就行，对你来说只是一件小事，对我而言却非常重要。”叶少阳握着店老板的手，一脸诚挚。“如果这些钱不够，我还可以给你些，你需要多少？”
<br/>　　“不是钱的事，不是钱的事。我只是……行吧，反正你这么说，我照做就是了。”老板把叶少阳给的大洋也收下了。
<br/>　　叶少阳再三道谢，这才离去，在路边叫上翠云，一起往客栈走去。
<br/>　　对刚才做的这件事，他是不报什么希望的，不过聊胜于无，也算是给自己寻找一个寄托。
<br/>　　店老板站在门外目送着他离开，再看看自己手中的鸡血石挂坠，喃喃自语道：“这可真是个怪人啊，九十年后……九十年后他怎么知道这世上还有谁，又把东西给谁呢？”
<br/>　　他的媳妇一直在蒸包子，之前听见了他们的对话，但是自己忙着干活没工夫理会，这会儿听见丈夫自言自语，说道：“那你还收人家的钱！”
<br/>　　“我当时已经懵了啊，而且这位先生看着也不像是疯子……他给我家店取的名字挺好的，这是怎么一回事？”
<br/>　　媳妇说道：“不知道就别管了，反正钱也收了，你把东西放起来，别弄丢了就是，将来嘱咐孩子一声就是。”
<br/>　　店老板一想也是，反正也不是什么大事，而且人家毕竟给了十几块大洋，既然弄不明白就不管了，把东西帮人家保管好就是，万一回头人家变正常了，来讨的话，自己大不了连钱带东西还给人家。
<br/>　　把东西放进抽屉之后，老板便把这件事放下了，继续盘算做招牌的事，越想“大肉包子”四个字，越是觉得好，让人一想就会流口水。这样的好名字绝对不能错过。
<br/>　　青冥界，战斗还在继续。尸族大军沉寂了几百年，突发发动进攻，自然是一发不可收拾。
<br/>　　之前星宿海一战，不过是整个战争的开始，之后的进攻更加持久而猛烈，因为袭击发动的太过突然，空界被打了个措手不及。
<br/>　　最初黎山老母等人还想借着九尾天狐帮叶少阳的事，对青丘山发难，正好九尾天狐死亡，与东皇钟一起失踪，青丘山群龙无首，正好是进犯的时机，但捉鬼联盟众人觉察到了阐教的想法，进驻青丘山，帮着抵御外敌，后来随着战争的继续，空界诸方势力，不管是阐教的门派还是截教各大妖山，一起都放下了成见，联合起来迎敌。
<br/>　　青丘山跟阐教也和解了，一起对抗尸族入侵。
<br/>　　阴阳司若干成员，因为帮助叶少阳对抗阴司缉拿转世鬼童，后来追究起来，革职但不查办，一个个发配到青冥界去对抗尸族，这还是阴司几个大佬帮他们讲情的结果。
<br/>　　瓜瓜等人也听从安排，跟着一起去了青冥界，每天结伴在战场厮杀，住在青丘山，在这种无止境的战斗之中，一点点提升着修为和战斗意识，同时，等待着他们的老大……叶少阳的归来，带他们在战场厮杀，抢回他们的女主人。
<br/>　　尸族大举进犯空界的战争爆发之后，在鬼域，之前已经蛰伏的太阴山也开始蠢蠢欲动，与阴司的局面再度紧张起来，小规模的战争不断。
<br/>　　风之谷作为两个朝廷的中间地带，首当其冲，承受了相当的压力，道风一干人也只好暂时放弃了别的计划，静下心来对付太阴山。
<br/>　　在这形势紧张的关头，阴司再度开始对风之谷进行拉拢，但都被道风拒绝了，没人知道他打的是什么主意，阴司对他猜忌颇多，开始风传他是想发展势力，成为第二个无极鬼王。
<br/>　　之前的十二门徒在星宿海被杀得只剩下上古邪神一人，于是道风重组十二门徒，并且用无上秘要的法术，给了上古邪神一张脸，让他拥有更大的权力，同时将叶少阳的鱼肠剑继续给建文帝掌管，更加疯狂地对周围的割据势力展开兼并，吸收更多的力量加入……
<br/>　　青云子已经知道了叶少阳被拖入另一时空的事，出人意料的是，他表现的很是沉默，之是告诉道风时机未到，要继续等待，自己去了陶弘景的道观中清修去了，再不过问任何事。
<br/>　　人间，一切在继续着，普通人继续朝九晚五，并不知道在鬼域和青冥界发生着多么惊天动地的战争，一切照旧，忙碌而活。
<br/>　　只是对叶少阳身边这些人来说，他们的生活少了点什么。以谢雨晴为例，只能每天更加疯狂的工作，来忘却烦恼，静静地等待着叶少阳的归来。
