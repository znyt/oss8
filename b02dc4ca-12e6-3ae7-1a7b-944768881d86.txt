　　第163章
<br/>　　“残红水上飘，梅子枝头小。这些时眉儿淡了谁描？因春带得愁来到，春去缘何愁未消？人别后，山遥水遥。我为你，数尽归期，画损了掠儿梢。”
<br/>　　黑石城，城主府。
<br/>　　原来薛半川主政的大厅，响起琵琶的声音。一绿衣女子唱一段，弹一曲，声音软糯，身体却因恐惧而颤抖。
<br/>　　啪！
<br/>　　琵琶乱了，丝弦割破手指，流出血来。
<br/>　　“喂，怎么不弹了？”
<br/>　　赤眉歪坐在椅子上，手里拎着一壶酒，他饮一口酒，合一段曲，显得悠哉悠哉。
<br/>　　绿衣女子是城里青楼的头牌，花名绿樱桃，往日城内的达官贵人想见她一面并不容易，多少人以与她面谈为荣。
<br/>　　绿樱桃跪倒，身体颤抖，豆大的汗珠落下，脸上没有半分血色。
<br/>　　“喂，弹下去，也唱下去，我爱听你弹。”赤眉喝了口酒。
<br/>　　那中断的琵琶再次响起，唱词中时不时伴随着哭腔。赤眉低声哼着，却也不以为意。
<br/>　　赤眉是北蛮部北蛮王的儿子，北蛮王有很多儿子，赤眉也不知道自己在兄弟中排第几。他因天生一撮红眉得名，至于自己母亲是谁，赤眉也不是很清楚。
<br/>　　和一般的戎族子弟不一样，赤眉不爱骑马也并不喜打猎，倒是对帝国的诗书乐理，琴棋书画感兴趣。
<br/>　　如此秉性，在崇尚暴力和野蛮的部落内自然难获尊敬，手下人也不太拿他当回事。
<br/>　　“哥哥！”福德勒跑入大厅，脸上带汗：“咱们还要在这里待多久？”
<br/>　　“这不在等部落把奴隶和物资运回去。”赤眉道：“怎么待烦了？”
<br/>　　福德勒点点头：“我想阿姆。”
<br/>　　“是想阿姆，还是想你的乌力罕。”赤眉笑着逗弄他的鼻子，换来福德勒一阵脸红。
<br/>　　“乌力罕是草原上最美的骏马，只有雄狮一样的男人才配得上她。”赤眉笑：“告诉我，你这次杀了几个人，又抢了多少奴隶？”
<br/>　　“我……”福德勒脸更红了。
<br/>　　福德勒也是北蛮王的儿子，不过怕北蛮王也不知道自己还有这么个儿子。相比赤眉，他更加没有存在感，和部落的普通人没有任何区别。
<br/>　　“傻弟弟……过来，和我一起听曲。”
<br/>　　兄弟二人坐在椅子上，绿樱桃轻弹琵琶，在她的认知中，戎族都是魔王般的存在。可当接近时却也觉得，这些戎族人也只是普通人而已，至少这个赤眉不是很吓人，笑起来的样子还很好看。
<br/>　　“听得懂么？”赤眉问。
<br/>　　福德勒摇摇头：“听不懂，也不觉得好听。”
<br/>　　“哈哈，傻弟弟，等你再长大些，就觉得好听了。”赤眉乐不可支，忽然一指绿樱桃：“好看么？”
<br/>　　“好看。”福德勒呆呆答道。
<br/>　　“这是哥哥帮你抢的，现在送给你了，哥哥帮你问过，她可还没被破过身子？”
<br/>　　“我……我已经有乌力罕了。”
<br/>　　“不敢见血的男人，可不是我北蛮部的勇士，要了她，或者杀了她。”赤眉忽然捏紧福德勒的脖子：“不然，我杀了你。”
<br/>　　那眼底的寒意如此浓厚，竟让福德勒情不自禁打个哆嗦，绿樱桃已吓得跪倒在地，冷汗如雨。
<br/>　　“好弟弟，就在这里成为真正的男人吧！”
<br/>　　说罢，赤眉哈哈大笑，将一柄刀插在福德勒面前。大厅空旷，只留下福德勒和绿樱桃两人。
<br/>　　看着跪在自己面前，曲线玲珑的女人，再看看插在地上的弯刀，福德勒下意识咽了口吐沫。
<br/>　　是杀了她，还是要了她？
<br/>　　……
<br/>　　蛤蟆寨，聚义厅。
<br/>　　林少羽孤零零举起的手，显得那样孤独，也显得那样可怜。
<br/>　　程大雷愣了愣，忽然问：“少羽，你为何要战，是不是心里有什么主意？”
<br/>　　林少羽坐在那里平静道：“为国为民，一死而已，戎族肆虐，我辈只能舍生取义，死，又有何惧。”
<br/>　　大厅内有不屑的笑声响起，和这帮山贼扯国家大义还不如和他们聊今天吃什么。可林少羽却不以为意，目光灼灼的盯着程大雷，一副只有你懂我的表情。
<br/>　　程大雷也很无语，本来还以为林少羽有什么高见，能说服这些山贼。的确程大雷也经常把为国为民挂在嘴边，但他心里却是半个字都不相信的。
<br/>　　如今，山寨五千余人，只有林少羽一个要战，好吧，就算算上程大雷也不过是两个。人心不在战，给他们导弹也打不赢这一战。
<br/>　　眼下，似乎只剩下逃这一条路了。
<br/>　　但程大雷心里真不想逃。
<br/>　　程大雷起身，道：“诸位，还认我这个大当家么？”
<br/>　　众人安静下来，高飞虎道：“大当家想要做什么，我们听大当家的。”
<br/>　　“只要我这大当家还作数就好。五千兄弟聚在一起，不过是为了活着而已。你们喊我一声当家的，也就是把命把交在我手中。但我并没有资格让大家去送死，我只问一句，若我能借来兵，有足够的把握打赢这一战，大家还想一战么？”
<br/>　　“这个时候，去那里借兵，又有谁肯帮我们？”高飞虎忍不住问。
<br/>　　“大当家……”一个喽啰跑进来：“杨龙停从黑石城逃过来了。”
<br/>　　这里聚义议事，程大雷一直派人盯着路上的动静。
<br/>　　他一听这话，瞬间来了精神，道：“诸位兄弟随我出去看看，我不会让我们兄弟陪我一起送死，但若有必胜把握，还希望兄弟们帮我一帮。”
<br/>　　程大雷带着众人离开山寨，只见败兵如蚁。
<br/>　　“截住他们路。”程大雷下令。
<br/>　　山寨喽啰将官道封住，从黑石城逃过来的败兵和平民不多时就挤成一大堆。
<br/>　　“程寨主想做什么？”
<br/>　　几匹快马分开众人，为首正是杨龙停，此刻他模样狼狈得紧，再无上次坐在大撵上威风八面的样子。
<br/>　　“王上，我一直在等你！”
<br/>　　“等我做什么？”杨龙停奇怪道。
<br/>　　“黑石城被占，青牛山已是帝国最后一道关口，若此地再被戎族占了，帝国在北地将无险可守。还望王上放下曾经恩怨，与我合力把守住此地。我青牛山蛤蟆寨愿为此战先锋。”程大雷道。
<br/>　　杨龙停骑在马上，上三眼下三眼的打量着程大雷，难以置信道：“你病还没好么？”
<br/>　　
<br/>(本章完)
