　　第108章
<br/>　　程大雷并没有撒谎，他的隐藏属性就是：一个好人。虽然这个隐藏属性，目前看上去并没有什么用处，看系统介绍，似乎是和魅力值有关，但也没发挥过作用。
<br/>　　高飞虎并不相信程大雷，他用警惕的眼神盯着程大雷，来回打量他好几遍。
<br/>　　“姓程的，你究竟弄什么玄虚？把我们当猴耍么，不要以为整座青牛山只有你一个聪明人！”
<br/>　　无怪高飞虎不相信，包括他在内，其他各山头的首领也都不太相信程大雷口中说得。
<br/>　　“年轻人。”程大雷拖了个长音：“不要年纪轻轻就什么都不相信了，有些成熟并不值得。”
<br/>　　“……”一片雅雀无声。
<br/>　　叮咚，程大雷觉得这个逼装得绝对满分。
<br/>　　“好啦，各位稍安勿躁，我点几个人，和我一起出去干趟活，各家有能打的，也别藏着掖着，毛遂自荐也是欢迎的。”程大雷顿了顿：“毛遂自荐这个词你们听得懂吧？”
<br/>　　“……”群贼。
<br/>　　无论众山首领信或不信，事情却是不能不做。程大雷选了赵子龙、高飞豹、小白狼，张肥随行，刘关留在山寨主持大局。当然，徐神机是必不可少的，这个毒奶法师，决不能把他留在山寨。
<br/>　　“爷，让我陪你一起去吧？”柳芷道。
<br/>　　“这次是出去做事，不是去玩，下次吧。”程大雷揉揉她的脑袋。
<br/>　　“我也可以做事，家里人都出过海，我也会水。而且一路上我也能照顾你。”柳芷道。
<br/>　　程大雷倒是差点忘了这件事，柳家人本来就是水军，操船如水，他们也都是行家。遇到海盗什么的，他们也有经验。
<br/>　　那么，就带上吧。
<br/>　　连夜将辣椒运到港口，一艘大船早已在那里等待着，入城的各方面关系，苏樱已经提前打点好。
<br/>　　一番忙碌，自不必说，最后将辣椒全部装上船。眼看就要出发，程大雷一直没有看到苏樱。
<br/>　　“你家小姐呢？”程大雷问一直操持忙碌的小蝶。
<br/>　　“我家小姐病了，染了风寒。”
<br/>　　“不严重吧？”
<br/>　　“不太严重，只是不能来送你了。”
<br/>　　“那嘱咐她好好养病，我回来再去看他。”
<br/>　　程大雷迈步要登船，见小蝶也跟上船来，他奇怪道：“你跟着做什么？”
<br/>　　“和你一起去呀，我替我家小姐照顾你。”
<br/>　　“咦，都要照顾我，我最近有这么受欢迎么。”
<br/>　　上的船来，水手撑蒿起航。
<br/>　　“走喽！”
<br/>　　水手们扯着嗓子喊了一声，三面帆扯满，顺风而行。
<br/>　　嘟，正义的少年喔，你接受任务，帮青牛山群贼度过寒冬。
<br/>　　今日里，程大雷腰缠十万贯，要乘风下扬州。
<br/>　　……
<br/>　　上船之后，内行和外行就立刻看得出来。像小白狼和高飞豹，在地上凶得狠，一旦上船立刻慌的跟什么似的。相比之下，小小的柳芷却在船上如履平地。
<br/>　　程大雷稍微好些，至少没有晕船。他在船上转了一圈，也没什么自己能帮忙的。如今操船水手都是原先柳家的人，船上的诸多事情内行得紧。
<br/>　　转了一圈，他还是选择回到船舱。刚走进舱门，程大雷就吓了一跳，只见一个小童立在房间里，唇红齿白，粉雕玉砌。
<br/>　　“怎么认不出来了？”小童转了一个圈。
<br/>　　“苏家小姐？”程大雷睁大眼睛。
<br/>　　苏樱捂着嘴咯咯笑，声若银铃般在舱内回荡。
<br/>　　程大雷一时还真没敢认，苏樱此时换了男装，做仆人打扮。可普通的仆人装穿在她身上，勾勒出亭亭玉立的身材。粉唇玉面，眼里像含着秋水。
<br/>　　“你不是病了么，怎么来了？”
<br/>　　“我怎么不能来，你自己又不会做生意。扬州是豪富之地，有很多乱七八糟的规矩，你懂么？”
<br/>　　苏樱又捧出一套崭新的袍子：“扬州人都很势利，你若穿得破破烂烂，他们会看不起你的，根本不会和你做生意。”
<br/>　　程大雷手扶过布料，丝绸质地，入手像扶过女儿的肌肤一样。单单这件袍子，也值七八两银子，搁在前世算是奢侈品了。程大雷自从来到这个世界后，还没穿过这么好的衣服。
<br/>　　“这衣服是你绣的？”
<br/>　　苏樱脸上一红，却没有回答。
<br/>　　千针万线，这样一件袍子，怕是要做上好几个月。
<br/>　　“大当家，大当家……”
<br/>　　一阵黄莺般的声音在外面响起，柳芷大步跑进船舱，在看到苏樱时，脸上笑容瞬间顿住。
<br/>　　苏樱脸上的笑容也消失了，随即脸上浮现另一种笑容。
<br/>　　“柳家小姐，一向未见，进来可好？”
<br/>　　“我现在是爷的奴才，这小姐二字万万担不起。”
<br/>　　“哪里话，我们是幼时相识，这些日子，也亏得你照顾他，我还要多谢你。”
<br/>　　“我本是爷的奴才，随身服侍也是应当，这谢字从苏小姐口中说出不太合适吧。”
<br/>　　苏樱和柳芷早就认识，算是闺中密友，从前在一起聊聊刺绣，聊聊诗词什么的。这件事程大雷早就知道，可看二人今天说话的意思，程大雷怎么觉得有一股火药味。
<br/>　　咦，原来古代也有表面闺蜜哦！
<br/>　　船借风力，风借船行，一路上风平浪静，并没有出什么事端，五天后顺利抵达扬州。
<br/>　　这里是扬州千舟城，还未下船便听到港口人声鼎沸，隔船往下望，港口热闹至极，来往的船只络绎不绝。
<br/>　　果然是个好地方嘛。
<br/>　　程大雷换上苏樱为自己准备的袍子，素红的底子，脖子上围着貂绒，腰间佩着长剑。
<br/>　　这幅装备，让高飞豹羡慕得啧啧称奇，口中道：在幽州，城主也就这扮相了。
<br/>　　可程大雷总感觉自己这幅打扮，像穿貂带金链子的社会大哥。幽州的审美和扬州的审美该不会有什么误差吧。
<br/>　　“都给我听好了，咱们是北国来的世家子弟，眼界气派都得给我有，扬州这边嫌贫爱富很严重的。别都一副乡下人进城的模样，丢了我们的面子。”
<br/>　　“眼界气派，如果咱们没有怎么办？”高飞豹道。
<br/>　　“装！”
<br/>　　苏樱已经和程大雷交待过，扬州是个好地方，只有先把面子装起来，才有够资格的人和自己做生意。
<br/>　　还未下船，繁华已经入眼。
<br/>　　这里便是扬州，千载繁华地，多少才子诗人的故梦，小桥流水比不得佳人美貌。
<br/>　　今日，程大雷来了。
