　　叶少阳目光落在尸体的右手中，身体的右手略微有点压在身下，手里好像拿着一块什么东西，黑咕隆咚的。叶少阳把尸体的手用力得掰出来，惊讶地发现，尸体的手中，居然扣着一只龟壳，巴掌大小，被尸体完全扣在手中，叶少阳仔细看后，发现不只是龟壳。
<br/>　　龟壳之中，还有乌龟的头骨和四肢的骨头，也已经完全钙化了。叶少阳顿时想到，这乌龟当时在她手中的时候，难道是活着的，就这么一直憋到死，然后浑身的肉腐烂分解，变成了现在这个样子？
<br/>　　有点匪夷所思……
<br/>　　叶少阳不敢乱动，在下面几个人的催促下，把在棺材里看到的情况说了一遍。
<br/>　　大伙听完之后，都是沉吟起来。
<br/>　　“啊，你们不要碰那尸体！快点把棺材盖上！”妙心突然喊了一声，声音充满了紧张，甚至还有点恐惧。
<br/>　　叶少阳心中一惊，来不及细问，就知道她一定是想到了什么，不然不会这么紧张，当下寻找棺盖。
<br/>　　棺盖已经被他们扔在地上了。
<br/>　　叶少阳急忙招呼毛小方一起跳下去，一人一头抓住棺盖，提起来之后，重新爬上坛城，想要把棺材盖上，结果就在这当口，陈晓宇那只妖仆“猴王”从他肩上跳下来，也不知道是因为好奇还是感知到了什么，嗖的一下窜到了坛城上，钻进棺材里，好奇地打量着躺在里面的尸体。
<br/>　　“猴王！”陈晓宇叫了一声，“不许盖棺！”
<br/>　　叶少阳跟毛小方都愣了一下，就在这时候，棺材里那个穿着银丝衣的尸体突然咔嚓一声坐起来，双手一合，朝猴王脖子捏过去。
<br/>　　猴妖毕竟是猴妖，反应极快，双腿一蹬，从尸体上弹跳起来，钻出了棺材。
<br/>　　“松手！”叶少阳大叫一声，毛小方心领神会，两人一起松开了棺材盖。
<br/>　　因为尸体坐起来的缘故，本身比棺材还高，棺材砸在她头上，把脑袋砸歪了一下，女尸脖子一挺，把棺材盖顶翻了，人从棺材里跳起来，立刻就朝最近的叶少阳扑过来。
<br/>　　纵然事情发生在仓促之间，不过叶少阳早就做好了准备，解开了勾魂索，这时候见尸体扑过来，立刻将勾魂索甩出去，勾住了女尸的脖子，人就势朝地上跳下去，勾魂索拉直之后，将女尸连带着从坛城上扯了下来，摔在地上。
<br/>　　叶少阳一个箭步冲过去，赶在女尸起身之前，用勾魂索在她身上又缠了一道，用力一拉，女尸虽然力大无穷（僵尸一般力气都大），但是反应不及，无处借力，被叶少阳再度拽倒。
<br/>　　叶少阳一跃而上，三下五除二，用勾魂索将女尸紧紧缠了几道，捆得严严实实的。
<br/>　　叶少阳这才长出了一口气，朝后面退了几步。
<br/>　　女尸被捆成肉粽，双手又不能用，在地上跟个虫子似的蠕动起来，把勾魂索挣得咯吱响，不过叶少阳可一点也不担心，勾魂索虽然不是什么神兵利器，但也是地府高等鬼差的标配，玄铁打造，地狱之火锻造而成，结实程度是完全不用怀疑的。
<br/>　　叶少阳之前还用它捆过更厉害的邪物，也都不在话下，对付一只这样的僵尸实在不算什么。
<br/>　　叶少阳冲妙心挑了挑眉毛，有点邀功的意思。
<br/>　　“快躲开！”妙心却是叫了一声。
<br/>　　什么？
<br/>　　叶少阳一下子就愣住了，别人也都愣住了，就在这时候，看到地上的女尸突然停止挣扎，浑身颤抖起来，幅度越来越大，在颤抖的同时，众人吃惊地发现，她的肚子也迅速大了起来。
<br/>　　因为叶少阳着重缠绕的是她的双臂，肚子上就箍了一道，所以并不影响她腹部的膨胀，眼看着的工夫，女尸的肚子就大到了好像八月怀胎的地步。
<br/>　　要爆了！
<br/>　　没有妙心的提示，大伙本能地意识到了这一点，立刻散开，躲藏在了坛城的另一面，然后只听的噗的一声，听上去好像是球体爆炸一样的声音……
<br/>　　一道绿色的液体，喷射得哪里都是，然后，一股腐败的类似于臭鸡蛋一样的臭味，在墓室里散发开来。
<br/>　　臭到无法形容。
<br/>　　叶少阳当场就要吐了，赶紧用袖子捂住嘴，再看大伙也都是一脸苦逼的样子。
<br/>　　“大家散开，这是尸瘴，吸的太多会中尸毒！”妙心捂着鼻子喊起来。
<br/>　　其实不用说什么尸毒，光是这么臭的气体，就让人受不了，大伙一个个都用袖子捂着嘴，寻找之前的入口。
<br/>　　但因为在坛城后面转了个圈，墓室里的光线也不够，大伙找了一会才找到出口在哪，立刻飞奔过去，钻进了墓道里。
<br/>　　叶少阳最后一个，刚要进去，突然听见一阵噗噗的声音，转头看去，只见被女尸爆体喷出来那些绿色的液体之中，有一些黑乎乎的东西正在蠕动着。墓室里光线昏暗，看不清什么，叶少阳目光寻找到一只鲛油灯，是之前放在地上照亮的，在鲛油灯的附近，能大致看清那些黑点，原来是数不清的小虫子，正在从绿色的液体中一堆堆地爬出来，也不四下乱走，而是聚集在一起，然后……叶少阳之前听见的噗噗的声音，正是这些小虫子爆开自己的身体，从中间飞出一只好像苍蝇一样的东西。
<br/>　　尸蠓！
<br/>　　叶少阳心中一惊，这种好像臭虫一样被法术界认为基本上绝种的东西，没想到会被自己遇上。
<br/>　　所谓尸蜢，顾名思义，就是一种尸体身上长出来的一种虫子，也可以看作是尸虫的一种变异的进化形态。
<br/>　　尸蠓这个词，其实还不是法术界定义的，而是苗医，把它定义成了一种药材，据说尸蠓有蚊子一样的尖刺状的口器，后面连着头部里的一个浆囊，里面贮存的是被尸蜢吸收转换之中的液体，叮人之后，一边吸血，一边会排出这种液体。
<br/>　　这种浆液中带有尸毒，对普通人来说是剧毒，被叮一口就等于注入了尸毒，后果很可怕，扩散之后会变成行尸，但是对得了某些疾病要死的人来说，却能够以毒攻毒，甚至救人一命。
