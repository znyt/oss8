　　山谷中。
<br/>　　鲁桂静静的盘坐在大石上，恢复体内的伤势。
<br/>　　鲁桂，便是这名黄泉魔宗强者的名字，他是黄泉魔宗七大元老之一。
<br/>　　元老，在黄泉魔宗地位崇高，比之四大殿主有过之而无不及。
<br/>　　“这一次真的是得不偿失！”一边疗伤，鲁桂心中一边暗叹。
<br/>　　他们信心满满的前来击杀苏莫，没想到不仅没有杀死对方，他们七人还受了伤。
<br/>　　苏莫已经完全成长起来了，估计就算宗主亲自出手，也不一定能轻易杀死了！
<br/>　　“不知道其他人怎么样了？”鲁桂暗自沉思，随即便准备给其他人传讯。
<br/>　　就在此时，蓦地，万里之外，一道璀璨的金光直冲天穹，耀眼无比。
<br/>　　金光只是出现了一瞬，立刻便消失不见。
<br/>　　“嗯？怎么回事？”鲁桂见此，顿时心中惊疑，随即他立刻身形一闪，便向金光出现之处飞去。
<br/>　　山脉中出现如此异象，他自然要去看看。
<br/>　　鲁桂的速度极快，呼吸之间，便来到了刚才金光出现之处。
<br/>　　他眼眸如电，扫视四方，却并没有发现有什么特殊的情况。
<br/>　　随即，鲁桂灵识扫荡而出，仔细搜索这一片地域。
<br/>　　很快，他便发现一头黑虎口中咬着一副画卷，在向远处疾刺。
<br/>　　“这是什么画？”鲁桂心中疑惑，难道是什么宝物？而后被这头黑虎发现了！
<br/>　　这种事情倒不是什么稀奇之事，苍穹世界山脉众多，遗迹数之不尽，还有一些武者死亡之后，都会遗落物品，经常都有人能寻到宝物。
<br/>　　这头黑虎看来也是运气好，在山脉中发现了这幅画卷。
<br/>　　嗖！
<br/>　　鲁桂毫不犹豫，身形如电，急速向黑虎冲去，而后一招便将黑虎轰杀了。
<br/>　　黑虎死亡，其嘴中叼着的画卷便滚落在地。
<br/>　　“我倒要看看是什么宝物？”鲁桂上前，伸手拿起了画卷，在眼前打量了起来。
<br/>　　这是一幅古朴的画卷，材质呈灰白色，画卷展开，足有大半丈长。
<br/>　　画卷上绘制的是一幅山水画，其上有山川河流、原野草地。
<br/>　　“空间宝物？”鲁桂稍一查看，眼眸中顿时精光闪耀，因为这幅画卷之上，弥漫着浓郁的空间之力，这不是空间宝物是什么？
<br/>　　“有趣！”鲁桂的嘴角勾起了一丝笑意，随便在一个山脉中疗伤，都能得到空间宝物，这气运真是没谁了！
<br/>　　随即，鲁桂便准备好好查看一番这幅画卷，看看其内有什么奥秘。
<br/>　　就在此刻，异变陡生。
<br/>　　鲁桂身前的空间突兀的波动了一下，而后一道犀利的寒光陡然从画卷上闪现，直刺他的胸膛。
<br/>　　“什么？”一瞬间，鲁桂大惊失色，他完全没有想到这画卷中居然暗藏杀机。
<br/>　　这道寒光速度极快，无与伦比，加上出现的太过突然，和他的距离又不足半丈，他完全没有任何躲避的时间。
<br/>　　噗嗤！
<br/>　　利刃刺入血肉的声音响起，寒光瞬间刺进了鲁桂的胸膛，狂暴的玄力如同一柄柄锋利的尖刀，冲进了他的体内。
<br/>　　啊！
<br/>　　一声凄厉的惨叫声响起，鲁桂立刻丢下了画卷，身形急速暴退千丈。
<br/>　　“苏莫！”鲁桂定晴看去，一名青年的身影出现在了原地，正是苏莫。
<br/>　　“你必死无疑！”苏莫收起虚界神图，面色冷漠，眸光如刀，盯着眼前的鲁桂，他的手中拿着十亿魔剑，魔剑上血光耀眼。
<br/>　　为了最大程度的重创对方，他没有动用帝皇剑，而是改用了十亿魔剑，
<br/>　　刚才一击，他不仅重创了对方，十亿魔剑更是在一瞬间，吸收了对方体内八成的精血。
<br/>　　就算是武皇境九重武者，失去了八成的精血，也会虚弱不堪。
<br/>　　“没想到你还是追上来了！”鲁桂脸色惨白如纸，形如厉鬼，他完全没有料到，苏莫会用这种方法来偷袭他。
<br/>　　对方刚才一剑，剑气绞碎了他体内的五脏六腑，和大量的经脉，更是几乎将他体内的精血吸干了。
<br/>　　这对他这个级别的强者来说，虽然不足以致命，却是让他遭受重创，实力降到了冰点。
<br/>　　“死吧！”苏莫懒得和对方废话，一步踏出，身形如离弦之箭，直接向对方冲了过去，十亿魔剑急速此处。
<br/>　　嗖！
<br/>　　鲁桂哪里还敢和苏莫战斗，毫不犹豫，身形拔天而起，急速逃离。
<br/>　　苏莫见此，立刻急速追了上去。
<br/>　　不过，目前的桂遭受重创，经脉受损，速度大减了下来，根本逃无可逃了。
<br/>　　咻咻咻！！
<br/>　　苏莫心神一动，九柄本命灵剑飞射而出，立刻追上了鲁桂，将其包围了起来。
<br/>　　“你逃不掉了！”苏莫冷笑这说道。
<br/>　　“可恶！”鲁桂脸色铁青，眸中杀机浓郁，既然逃不掉那就拼死一搏。
<br/>　　“杀！”
<br/>　　一声厉喝，鲁桂顿时出手了，他的身上突然冲出一道道鬼影。
<br/>　　这些鬼影全部是狰狞的厉鬼，阴森无比，模样一个比一个恐怖。
<br/>　　呜呜呜~~~
<br/>　　厉鬼全部都向苏莫扑了过来，带起阴风阵阵，仿佛鬼哭狼嚎。
<br/>　　“结束了！”苏莫摇了摇头，手中魔剑一挥，剑气挥洒长空，一道道鬼影在剑气之下湮灭。
<br/>　　于此同时，苏莫心神一动，九柄本命灵剑铺天盖地的向鲁桂身上激.射而去，就欲将对方彻底击杀。
<br/>　　不过，苏莫显然小看了鲁桂，对方毕竟是武皇境九重的强者，即便遭受重创，也还有保命的本事。
<br/>　　“天鬼护体！”
<br/>　　眼见九柄灵剑袭来，鲁桂大喝一声，他的身上再次浮现了一只厉鬼。
<br/>　　这只厉鬼高达三丈有余，全身漆黑如墨，非常凝视，其双眸中爆射出幽幽的绿光。
<br/>　　厉鬼将鲁桂守护在内，为其抵御灵剑的攻击。
<br/>　　咻咻咻！！
<br/>　　瞬息之间，九柄本命灵剑纷纷射在了厉鬼的身上，但是却并没有破开厉鬼的防御。
<br/>　　这倒不是灵剑的攻击力不够强，而是厉鬼的身上，蕴含了一种扭曲引导之力。
<br/>　　灵剑乍一接触到厉鬼，顿时偏离了方向，从一旁飞射而过。
<br/>　　九柄灵剑全部都偏离了方向，从厉鬼的体表划过，没有对厉鬼造成丝毫伤害。
<br/>　　“有趣！”
<br/>　　苏莫见此顿时来了兴趣，他倒是从未见过这种防御，可以偏离对手的攻击。
