　　　　说起古学院，涂小安就想起了那个自带笛声的湖泊，如果里面真的镇压一头三阶生灵，那么自己下去将它KO了，是不是自己的等级又可以连升几级？
<br/>　　但是三阶生灵超凡脱俗，涂小安还真没跟这种级别的生物战斗过，心里也没什么底。
<br/>　　但是探还是肯定要探的。
<br/>　　他发现自己的好奇心是随着实力在暴涨。
<br/>　　实力越强，好奇心莫明也在滋生。
<br/>　　如果能名正言顺的进入古学院，未尝不可，说不定还能体验一下大学的生活。
<br/>　　涂小安看着蛇博士：“你那位朋友是谁”
<br/>　　“山野集团的董事长东方野，我要你保护的是他的儿子东方袁”
<br/>　　咦...
<br/>　　涂小安诧异了下，道：“东方袁可不是觉醒者”
<br/>　　“你见过？”陈默也意外。
<br/>　　涂小安点头：“说来也巧，我来这里之前，刚见过，是个大胖儿子”
<br/>　　陈默主动将他后面那个大胖儿子的话忽略，道：“袁儿的资质不错，有可能突破，成为觉醒者，所以董事长给温城大学捐了一个亿，古学院才破例将还未觉醒的袁儿招收进去”
<br/>　　一个亿？
<br/>　　涂小安听的目瞪口呆，他好想咆哮出来，你给我一个亿，我分分钟让那个大胖儿子成为觉醒者。
<br/>　　如果不愧是温城第一龙头集团，给自己儿子上个学，就捐了一个亿。
<br/>　　他是多渴望望子成龙呀。
<br/>　　“就因为袁儿不是觉醒者，就进入古学院，难免会被人欺凌，在里面，他的身份对别人可没有半点的震慑力”
<br/>　　能入古学院的人，来历都不一般，很多都是隐世家族的子弟，这温城第一上市公司董事长的儿子，还真没放在眼里。
<br/>　　陈默跟东方野就担心这一点，如果涂小安没出现，陈默也会物色别人。
<br/>　　但有这条蛇精，他也就不用考虑别人了。
<br/>　　涂小安大致明白了，不得不说，这个大胖儿子的命真好，有个给力的老爸，什么都帮他安排好。
<br/>　　“你有什么要求，尽管提，我能做到的，都答应你”
<br/>　　陈默看着涂小安。
<br/>　　“我没什么要求，我刚好也想去古学院，你能帮我光明正大的进去正好，我可以答应你护他周全，不过...“
<br/>　　“不过什么”
<br/>　　涂小安嘻嘻一笑：“学费都能捐一个亿，我这个保镖总要给点酬劳吧”
<br/>　　“你要钱？”陈默愣了下。
<br/>　　涂小安一头黑线，这叫什么话，他怎么不要钱，他可是要养家糊口的。
<br/>　　上有老妈，下有老姐，不但要养老送终，还要给自家老姐攒嫁妆呢。
<br/>　　压力很大的。
<br/>　　况且他岂能白白给大胖儿子当保镖，他涂小安可是一代登基坐殿的蛇王，就差证道了。
<br/>　　“我让董事长给你一千万当作酬劳，你只要保证他不被人欺凌就行，平时别的事情不用管”
<br/>　　涂小安听到一千万，眼睛猛然一亮，这个钱好赚，说是不被欺凌，其实就是别让东方袁有生命危险就成。
<br/>　　“没问题，什么时候入校”
<br/>　　陈默道：“三天后，那时候我找你，让你跟着袁儿一起入校”
<br/>　　“好，一言为定”
<br/>　　两人一下子达成战略合作关系，皆大欢喜，但陈默依然提醒了一句涂小安。
<br/>　　“温城毕竟不比白镇，你平日能低调点还是低调点，免的惹祸上身”
<br/>　　..............
<br/>　　半个小时后，涂小安从咖啡馆出来，心情大好，中午十分，他刚去银行兑现了五百万RMB，这不，很快又有一千万进账。
<br/>　　这大都市赚钱还真是简单，到时候他也可以市中心买套房，接老妈过来享享清福。
<br/>　　今天也没什么别的事情可做，就等三天后，给大胖儿子当保镖入校，涂小安想了想决定回陈家大院。
<br/>　　但心中还有一事，那么便是白家，白楚艳三番二次在自己的手中吃亏，而且他们白家的蛇王石，落在猫精的手中，肯定不会善罢甘休。
<br/>　　涂小安就是担心白家的人会继续找猫精的麻烦，他们并不知道蛇王石已经被自己这条蛇精融合了。
<br/>　　这是一个问题，如何能让白家的人知道蛇王石现在在自己的手中呢，知道这点，想必他们也就不会在寻找猫精的位置了。
<br/>　　但他也不能傻逼一样的过去主动交代？
<br/>　　问题是涂小安也怕麻烦。
<br/>　　不过猫精现在体内不会散发特殊蛇类气息，也不会招惹群蛇而至，白家的人想找猫精也没那么简单，况且猫精已经被自己整的化形。
<br/>　　小隐隐于市，暂时应该是安全的。
<br/>　　初来乍到，一个隐世修炼家族，涂小安还不想去招惹，别人数百年的底蕴，不是摆设的。
<br/>　　况且他们之间也没什么深仇大恨。
<br/>　　涂小安边走边思忖，来到了一个红绿灯的交叉路口，红灯还没停，绿灯没亮，来往的车辆在穿梭，他却看见一位可爱的小女孩在过斑马线。
<br/>　　这小女孩手中抱着一个洋娃娃低头玩耍，无视行驶而来的车辆，就这样堂而皇之的在斑马线走去。
<br/>　　滴...滴...滴...。
<br/>　　前面一辆黑色轿车快速的冲来，冲近了，才看来一个小女孩挡路，连忙着急的按喇叭。
<br/>　　“快闪开，快闪开”司机疯狂的咆哮。
<br/>　　可惜...发现晚了，踩刹车都止不住...

<br/>　　这五六岁的可爱小女孩，眼前就要出车祸，刹那间，不少人闭上了眼睛，不忍看这血腥的一幕，因为他们仿佛已经看到小女孩飞出去，满身是血的画面。
<br/>　　“该死，这小女孩的家属呢，怎么能让那么小的孩子一个人过马路”
<br/>　　轿车已经出现在小女孩的一米内，无人可救。
<br/>　　说时迟那时快，涂小安一咬牙，顾不得许多，向前踏出一步，身形飘逸恍若幻影，瞬间移形换位，一个眨眼就冲到斑马线中间，将小女孩抱住。
<br/>　　砰...
<br/>　　轿车而至，只见一位清秀少年刚抱起小女孩就被撞飞了出去，飞出了五六米之外，还滚了好几圈。
<br/>　　啊..啊...啊...
<br/>　　附近出现了密集的尖叫惊恐声。
<br/>　　“叮...宿主的人形态损坏百分之五十”
<br/>　　涂小安压根不去理会系统的这道提示音，低头看向怀中的小女孩。
