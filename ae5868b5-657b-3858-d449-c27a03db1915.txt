　　此时此刻，六大王宫的七百名弟子，有一百多名，已经通过了轮回三关。
<br/>　　五百多名，无法通过轮回三关，将不能进入轮回古城。
<br/>　　但是，现如今，唯有苏莫，依旧在人门关之中，既没有通过也没有失败。
<br/>　　“哈哈，段惊天，我倒要看看谁的脸更痛！”褚彦面上露出戏虐般的笑容。
<br/>　　之前，他被苏莫打了脸，现在终于轮到他打脸了！
<br/>　　他心中暗喜，暗暗决定，待会段惊天无论过关与否，他都要狠狠的嘲讽一番。
<br/>　　沁儿眉头微皱，段惊天居然三个呼吸的时间，都无法过关，难道法则之力很弱吗？
<br/>　　玄天神王亦是有些疑惑，这个段惊天是轮回之身，悟性肯定不会差，法则之力怎会如此孱弱。
<br/>　　时间缓慢的流逝！
<br/>　　四个呼吸！
<br/>　　五个呼吸！
<br/>　　很快，便过去了六个呼吸的时间，苏莫依旧在人门关之中。
<br/>　　“怎么回事？”
<br/>　　“已经六个呼吸的时间了！”
<br/>　　“情况不对啊！”
<br/>　　众人议论纷纷，情况似乎不正常，有些诡异。
<br/>　　因为，正常情况之下，如果是法则之力足够，三个呼吸之内，就能通过人门关。
<br/>　　而三个呼吸的时间，都无法通过的话，就会被人门关排斥出去，从而闯关失败。
<br/>　　但是，这个段惊天，已经六个呼吸的时间了，居然还在人门关之中？这就奇怪了！
<br/>　　“怎么回事？”
<br/>　　几位长老，也奇怪了起来，完全不清楚这是什么情况？
<br/>　　这是以往从未出现过的。
<br/>　　玄天神王面色郑重，他心有猜测，这可能和段惊天是轮回之身有关，但是，他却是不会说出来。
<br/>　　所有天才都惊奇不已，所有人都凝视着苏莫，不明所以。
<br/>　　人门关之中，苏莫的确是遇到了困难。
<br/>　　人门关，考验的是法则之力的强弱，他的法则之力虽然特殊，但是并不强。
<br/>　　那股玄妙的奇异之力，在他的体内，不断的和他的法则之力僵持。
<br/>　　最为主要的是，在人门关之中，他的法则之力，无法控制，一切都是那股奇异的力量自行引动而起。
<br/>　　“这下糟了！”
<br/>　　苏莫心中无语，若是失败，连轮回古城都进不去，还拿什么证明自己？
<br/>　　僵持在继续，苏莫的法则之力，在坚持了十个呼吸之后，不断的溃败。
<br/>　　几乎是在半个呼吸之内，法则之力，便被彻底击溃。
<br/>　　而后，那股奇异的力量，瞬间充斥苏莫全身。
<br/>　　他的吞噬元神，就隐藏在丹田深处，无可避免的，被那股奇异的力量碰触了。
<br/>　　霎时之间，那股奇异的力量，如阳春化雪一般，瞬间消融一空。
<br/>　　奇异的力量消散，苏莫获得了自由，身形一闪，便通过了人门关。
<br/>　　他通过人门关，用了整整十一个呼吸的时间。
<br/>　　全场一片寂静，所有人心中都惊疑不定。
<br/>　　十一个呼吸的时间通过人门关，这不符合常理啊！
<br/>　　超过三个呼吸的时间，就会被人门关排斥出去，从而失败。
<br/>　　“哈哈哈！段惊天，这就是你的天赋吗？”
<br/>　　褚彦大笑了起来，虽然他也觉得有些奇怪，但是，他可不会放过这个打击苏莫的机会。
<br/>　　“你的脸痛不痛啊？十一个呼吸的时间，你可是打破了记录啊！”褚彦放声的嘲笑，面上带着不屑的笑容，声音极为刺耳。
<br/>　　沁儿瞥了褚彦一眼，眼神略带厌恶，随即她来到苏莫的身前，疑惑的问道：“段惊天，怎么回事？”
<br/>　　“不清楚！”苏莫摇了摇头，他无法快速的通过人门关，自然是法则之力太弱，但是，最终能通过，肯定是吞噬元神的原因。
<br/>　　但是，里面具体的情况，他就不清楚了。
<br/>　　“很显然，天赋不够呗！“褚彦阴测测的说道。
<br/>　　苏莫瞥了褚彦一眼，这个家伙，还真是够恶心人的。
<br/>　　不过，他也懒得争辩。
<br/>　　轰隆隆！
<br/>　　就在此刻，人门关消失了，轮回古城之中，响起了一片轰隆隆的巨响。
<br/>　　巨响震天，仿佛大地轰鸣，天地齐震。
<br/>　　之间轮回古城的上空，一个庞大的虚影，缓缓升腾而起。
<br/>　　这个虚影，是一个人影，极为庞大，高约数十万丈，仿佛是一个巨人，站立在轮回古城之中。
<br/>　　虚影头顶天、脚踏地，一股苍茫、古老、磅礴的气息，弥漫九天十地。
<br/>　　少倾，虚影渐渐的凝实，成为了一个犹如实质的巨人。
<br/>　　这是一个全身金光的中年人，眸如六道轮回，面容有些模糊，让人看不清具体面容。
<br/>　　不过，一股无形的威压，如同太古神山，如同亿万星辰，狠狠的压在了众人的心头。
<br/>　　仿佛，这是一位主宰一切的帝王，能让所有人为之低头。
<br/>　　“天尊大人！”
<br/>　　“是天尊大人的幻象！”
<br/>　　“好可怕的感觉！”
<br/>　　这一刻，众人不再关注苏莫，所有人的目光，都凝视在了这位巨人的身上，因为这就是轮回天尊的幻象。
<br/>　　这一刻，所有人的面上，都露出了崇敬之色。
<br/>　　轮回天尊，宇宙的传奇，神域的传奇，人族的传奇。
<br/>　　这是一位太古至强者，威震万古，震慑八荒。
<br/>　　就连六位长老，都面色恭敬无比，不敢有丝毫的不敬。
<br/>　　“走，进轮回古城！”
<br/>　　“我们走！”
<br/>　　一众通过轮回三关之人，纷纷身形闪掠，向轮回古城之中而去。
<br/>　　能通过三关之人，修为最低都是一品高级，可以说全部都是强者。
<br/>　　苏莫也没有迟疑，和沁儿等人一起，快速的进入了轮回古城之中。
<br/>　　至于那些没有通过三关的人，就无法进入古城了，只能望洋兴叹。
<br/>　　苏莫进入轮回古城之中，根本看不到任何建筑，入眼处全部是迷蒙的光晕。
<br/>　　只有前方那巨大的轮回天尊幻象，耸入云霄，如同擎天巨峰。
<br/>　　所有人都向轮回天尊的幻象之处赶去，众人的速度极快，片刻之间，便临近了轮回天尊的幻象。
<br/>　　距离近了，更能感受到那浩瀚的威压，这不是气势，但是，却作用在众人的心中。
<br/>　　这是一种骨子里的畏惧与崇敬，任何人面对这位太古至强者，面对着这位人族先祖之一，都会感到心神震颤。
<br/>　　“拜见天尊！”
<br/>　　“拜见天尊！”
<br/>　　“拜见天尊！”
<br/>　　一时间，呼声震天，所有六宫弟子，全部双膝跪地，向巨大的轮回天尊幻象跪拜了下来。
<br/>　　苏莫也跪拜了下来，这既然是他的前世，跪拜也无不可。
<br/>　　嗡嗡~~~
<br/>　　随着众人跪拜，虚空震颤，异像陡起。
