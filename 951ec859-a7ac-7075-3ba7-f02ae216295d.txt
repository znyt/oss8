　　蛇王庙只所以香火旺盛，第一来自信徒的信仰跟崇拜，第二，也有一些化腐朽为神奇的地方。
<br/>　　比如，蛇缠身，蛇怨气，都有办法到蛇王庙来化解。
<br/>　　蛇报复心强，被蛇纠缠过的人自然就信，没遇见事的人，或许不屑一顾。
<br/>　　涂小安将自己的体型变的昆虫般大小，就招摇过市的飞入了蛇王庙中，庙内庙外都是一些来烧香祈福的游客，但却没一人能发现他这条会飞的迷你蛇。
<br/>　　庙的正堂供奉着蛇王，此蛇王雕像非常的巨大巍峨，但却不是一条蛇的样子，而是一位慈眉善目的中年人手握蛇杖坐立的模样。
<br/>　　相传这是蛇王曾经化为人形，在人间游历，做善举时留下的样子，后被人铭记，刻为蛇王雕像。
<br/>　　涂小安无声无息的落到了蛇王雕像上，放眼望去，几位外地的游客正对着蛇王雕像磕头许愿，样子非常的虔诚，就跟拜佛拜拜菩萨是一样一样的。
<br/>　　他留意了一下系统的消息，果然这些人拜蛇王雕像，跟他没半毛线的关系，不会产生一点信仰点数。
<br/>　　他只有取而代之才行。
<br/>　　此时，只见一位莫约快三十岁的少妇朝着蛇王雕像磕了三个响头，然后双手合十，嘴如梦中呓语似的说：“蛇王大人，求你保佑我这个月一定要怀上孩子，我婆婆说了，只要我这个月内怀上宝宝，就给我一百万当奖励”
<br/>　　许完愿，再次磕上三个响头，非常虔诚。
<br/>　　涂小安耳聪目明，方圆几公里内，任何细微的声音，只要他刻意想听，都能听的清楚，更别说眼前这位少妇距离他不过三丈内了。
<br/>　　只是听完少妇的话，涂小安惊奇得如五雷击顶，目瞪口呆，这年头还有求子求到蛇王庙的？
<br/>　　你让一条蛇保佑你生孩子，这是不是有点剑走偏锋了。
<br/>　　隐晦间，涂小安顿时觉得自己压力山大，他要是想取而代之，想要更多的信徒，恐怕要有不少的新“业务”。
<br/>　　这时，一位中年妈妈带着一位十三四岁的孩子进来磕头许愿，只见那男孩看到蛇王雕像，噗通一声的就跪在了蒲团上，样子到是很是虔诚，旋即张嘴便来：“蛇王大人，我可是您的铁杆粉丝，我想求你保佑我玩LOL拿五杀，我玩了几千把了，一次五杀都没拿到过”
<br/>　　旁边的妈妈闻言就一个爆栗敲过去，嗔怪：“胡说什么，给我好好的磕头就是”
<br/>　　“蛇王大人赎罪，小孩子没把门，你别理他，我求你保佑我丈夫不要出去找小三，要是还出去找小三，你就派条毒蛇咬死他得了，我还能继承他的遗产”
<br/>　　中年妈妈偷偷背着自己的孩子说出了这样一番惊天地泣鬼神的话。
<br/>　　暗中的涂小安听的是一头黑线，Y的，怎么没一个靠谱的许愿。
<br/>　　简直是来玩过家家似的，拿蛇王是什么。
<br/>　　求子，游戏五杀，谋杀亲夫....
<br/>　　还真是大千世界，无奇不有，还有人许愿让蛇王派蛇咬死自己老公的。
<br/>　　当然，正常情况下，蛇王庙不应该是这个样子的。
<br/>　　忽然，庙外有三四个人抬着一位脸色苍白，晕迷不醒的男子冲了进来，其中一人便道：“快拿蛇王水过来，我朋友中蛇毒了”
<br/>　　顿时间，庙中一位身穿奇异道袍的中年男子连忙跑了过来，检查了一下中毒者的伤势，发现他的小腿部位早就紫黑紫黑了。
<br/>　　果然是中了蛇毒，一看便知。
<br/>　　“放平，不要碰他了”道袍男子转身离开，旋即手中端了一碗略带浑浊的水而来。
<br/>　　这几个人，一看到蛇王水顿喜，连忙冲着蛇王雕像磕头，：“谢谢蛇王大人赐水”
<br/>　　此时不少游客看到动静围了过来，都听说蛇王庙很灵，其中最让人津津乐道的就是，你要是在白镇被毒蛇咬了，那么可以直接来蛇王庙。
<br/>　　喝下蛇王赐下的蛇王水，立刻康复，非常神奇。
<br/>　　趴在雕像上的涂小安看到这碗浑浊的水，露出了古怪之色，外人或许不知道蛇王水里面的门道，但他可是土生土长的白镇人，而且还是捕蛇人。
<br/>　　这说是蛇王赐下的蛇王水，可以解蛇毒，其实不过是普通的水，在里面加了一些捕蛇人秘制的解毒药。
<br/>　　一是为了提高蛇王的声望，二也是为了做好事。
<br/>　　任何看似神奇的事情，背后都有点故弄玄虚。
<br/>　　不过这“蛇王水”仅仅是对普通毒蛇的毒有作用，要是碰到罕见蛇种的毒，那么就只能抬医院，用抗蛇血清了。
<br/>　　现代医学发展到现在，虽然已经非常发达，但针对毒蛇的咬伤，仍然没有直接可以口服的特效药，只能用抗蛇血清。
<br/>　　另外抗蛇血清也不是乱用，你被什么毒蛇咬的，就要用什么蛇毒研发制作的血清。
<br/>　　比如你被五步蛇咬了，用眼镜蛇的血清，那就牛头不对马嘴了。
<br/>　　蛇王水在前，这几个人稍微的安心了，显然对蛇王是非常的信任，觉得自己的朋友一定无碍了。
<br/>　　唯有端着蛇王水的道袍男子，迟迟没有将水灌入中毒者的口中，而且表情有点难看起来，因为他看出来了，这晕迷者中的蛇毒不简单呀，恐怕不是一碗蛇王水可以解决的了。
<br/>　　必须立刻去医院检查，检查出是被什么毒蛇咬的，在对阵下药。
<br/>　　好死不死，最麻烦的是，今天外来的游客很多，周围都围了不少人过来，静等蛇王水的神奇功效。
<br/>　　他现在有点骑虎难下，要是这碗蛇王水被喝下去，晕迷者依然不见好，恐怕会对蛇王庙的名誉照成重大的损害。
<br/>　　往常，道袍男子遇到这种情况，都会事先检查中毒者的伤口，要是看到伤口不一般，毒性剧烈，就会立刻找一个托词，说今天没有蛇王水了。
<br/>　　要是一般的蛇毒，自然没有问题，免费为了献上一碗蛇王水。
<br/>　　今天游客太多，道袍男子也是忙的失了分寸，一听有人急着要蛇王水，匆匆看一眼伤口，确定是蛇毒，就立刻去拿了。
<br/>　　拿过来后，再看伤口更加的紫黑，才恍然回神，暗叫不好。
<br/>　　“大师，我来喂吧”
<br/>　　一人看到道袍男子楞在那里，连忙准备动手将蛇王水接过来。
<br/>　　“还是我来吧”
<br/>　　道袍男子心中发苦，暗暗祈祷能解毒能解毒，蛇王招牌不能砸。
<br/>　　这蛇王庙可是白镇的文化古迹，金字招牌了。
