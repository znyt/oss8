　　“是你送给我的嘛，是我用实力收服，让它认主”
<br/>　　涂小安冷冷的说，摸了摸脚下银月巨狼的头，银白的毛发很柔顺。
<br/>　　“蛇妖，你到底怎样才会放过我，你说，我能做到的，我都答应你”
<br/>　　罗子雄还在拖延时间，暗中疯狂咆哮：该死，为什么古学院的人还没冲进来，这是怎么回事。
<br/>　　这什么办事效率，救人如救火，不知道吗。
<br/>　　住所外，此时汇聚了大量的学员，他们深深的感受到住所内强大的蛇类气息蔓延。

<br/>　　一大帮正义凛然的学员被高级教官拦在了外面，他们抗议起来。
<br/>　　高级教官道：“这是副院长的命令，谁要是违背，就立刻被逐出古学院”
<br/>　　逐出古学院这几个字一出，让这群学员们一下子立刻安静了起来。

<br/>　　“院长，遵您的指令，住所被全面封锁，也没一个人迈进去过”
<br/>　　“不过蛇妖好像是在里面行凶，里面有学员的惨叫声传来，我们真的不进去救人？”高级教官担忧询问。
<br/>　　副院长早就想好的措辞，道：“这蛇妖气息强大，实力非同小可，不是你们可以对付的，为了避免不必要的损伤，老夫要亲自动手，降妖除魔”

<br/>　　这爆炸性的消息，顿时间让外围被拦下的学员都窃窃私语起来。
<br/>　　古学院的副院长实力，深不可测，很多学员眼睛都亮了起来。
<br/>　　“老夫现在进去，切记，老夫没出来，不准任何人踏进住所半步”
<br/>　　副院长最后吩咐一声，朝着住所内走去。
<br/>　　“哈哈...副院长出手，里面的蛇妖一定要身首异处了”
<br/>　　“我听说副院长早就是三阶生灵级别，乃是我们古学院一等一的强者，很多年没有出过手了”
<br/>　　“那是，古学院高手如云，副院长什么身份，能轻易出手吗，可惜呀，我们不能亲眼看到副院长大发神威，降妖除魔”
<br/>　　“看不到，我们竖起耳朵，听听里面的动静也好”
<br/>　　学员们泛着兴奋，看着副院长走了进去，直到消失。
<br/>　　住所内，涂小安将罗子雄从墙壁上抓下来，又是一顿暴虐，最后骑在他遍体鳞伤的身上，手中握着一把薄如蝉翼的飞刀，抵在他的胸口心脏位置。
<br/>　　“别杀我，求求你了，饶我一命，我不能死”罗子雄双眼惊恐，紧张得汗一股脑儿往外冒，心“扑冬，扑冬’，地跳。
<br/>　　说不定下一秒，他的心脏便要停止跳动。
<br/>　　“青枭，解气没”涂小安转身，看了仙灵孔雀一眼。
<br/>　　青枭在梳理自己的羽毛，不少地方都空荡荡的无毛，让它一双漂亮绝伦的眼睛尽是杀意：“杀了这个丑八怪，人家才能解气，哼哼...”
<br/>　　“好，主人听你的”涂小安邪魅一笑，对着罗子雄说：“你也听到了，你就安息的去吧，下辈子做人，眼睛放亮点”
<br/>　　“不要不要”
<br/>　　罗子雄感觉胸口刺痛，那柄飞刀轻松的刺破了他的皮肉，往心脏部位突进。
<br/>　　“哦，对了，忘记问你，谁让你来招惹我的”涂小安动作微微一停。
<br/>　　罗子雄浑身颤栗，忙道：“是大小姐邪七七发话，说谁抓了你的孔雀送给她，她就跟谁来一场约会”
<br/>　　“邪七七？”
<br/>　　涂小安脑海中浮现一个风华绝代的美女，懂了。
<br/>　　“你去吧，你的银狼，我会帮你好好的养着”
<br/>　　“不要，不要，救命，救命啊，该死，为什么还没人来”罗子雄感觉到飞刀一点一点的突进皮肉，跟他的心脏非常接近了。
<br/>　　慌了，怕了，死神的手几乎已经握住了他。
<br/>　　该死，他要杀就杀，直接刺进去不就行了，非要一点点的突进。
<br/>　　这是最折磨人的残忍方式。
<br/>　　完了，完了，他本是天才的生物掌控者，前途无量，甚至有希望成为三阶生灵，超凡脱俗，活了三五百岁，都不是问题，可现在，就这样被杀了。
<br/>　　不甘心，他真的不甘心。
<br/>　　就在此刻...

<br/>　　一道苍老的声音骤然响起。
<br/>　　副院长到了，看到涂小安将飞刀一点点的刺进罗子雄的心脏，立刻喝止。

<br/>　　“你看看那边”

<br/>　　涂小安拼命的让自己变强，就是不想唯唯诺诺的活着，他想保护自己想保护的人，去救自己可以去救的人，不求有功，但求活的潇洒惬意。
