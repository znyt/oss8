　　青袍老者顿时一愕，这才恍然想起，
<br/>　　他刚刚看到精品品阶的化婴丹，实在太过于激动，一时间竟然忘了这茬。
<br/>　　“敢问叶先生，不知道这一瓶，又是何灵丹？”
<br/>　　青袍老者没有着急打开药瓶，而是陪着笑脸，向叶尘开口问道。
<br/>　　“也是化婴丹！只不过品阶和之前的有所不同！”
<br/>　　叶尘品了一口茶水，随口回答一句。
<br/>　　青袍老者先是一愣，继而明白了过来，心想，
<br/>　　‘之前那两枚化婴丹，可是精品品阶的灵丹，那这个玉瓶里的灵丹，多半应该是普通品阶，又或者是粗制品阶吧？毕竟即便是五品炼丹宗师，想要炼制出精品品阶的化婴丹，也不是一件容易的事情！’
<br/>　　一念及此，青袍老者微微有些失望，不过还是按照规矩，缓缓打开了丹瓶。
<br/>　　然而，
<br/>　　就在他打开丹瓶的那一瞬间，
<br/>　　轰隆隆！
<br/>　　周围空间忽然一阵晃动，简直犹如发生了地震一般！
<br/>　　紧接着，
<br/>　　在青袍老者无比震惊的目光中，一颗散发着金色玄光的丹药，竟然自动从瓶内悬空而起！
<br/>　　呼呼呼~
<br/>　　一股浓郁到令人心醉的丹香，瞬间传遍整个天藏阁！
<br/>　　很快，乃至方圆数里之内，都被这股丹香所笼罩！
<br/>　　“这、这、这……”
<br/>　　青袍老者两眼死死盯着眼前的那颗丹药，身体情不自禁的剧烈颤动起来，已经激动连话都快不会说了，
<br/>　　“这难道……就是传说中的……宝丹？！！”
<br/>　　唰！
<br/>　　唰！
<br/>　　唰！
<br/>　　大殿门口，突然出现三道气息强悍的苍老身影，犹如鬼魅一般。
<br/>　　而这三道苍老的身影出现之后，全都死死盯着眼前的这颗金色宝丹，眼眸中透着浓浓的炙热之色，
<br/>　　“完美品阶的化婴丹吗？”
<br/>　　“没想到时隔十多年，我们天藏阁，又有宝丹出现了！”
<br/>　　“这宝丹，是何人之物？”
<br/>　　……
<br/>　　三名老者进入大殿之后，全都直勾勾的望着那枚化婴丹，纷纷开口道。
<br/>　　青袍老者和那妖艳女子，看到这三名老者，当即躬身行礼，
<br/>　　“拜见三位长老！这宝丹，是这位叶先生之物，想要在我们这里拍卖！”
<br/>　　“哦？”
<br/>　　其中一名老者，听到这话，直接上前一步，猛然伸出手，便朝那枚化婴丹抓去。
<br/>　　叶尘见状顿时眉头一皱，抢先一步，虚手一抓，
<br/>　　嗖！
<br/>　　那两瓶丹药，第一时间被吸入叶尘的掌心之中。
<br/>　　“咦！”
<br/>　　那老者显然没有想到，自己这一抓之下，居然失手了，不由得重新打量了叶尘一番，脸色的惊色更浓，
<br/>　　“明明只有金丹境，竟然可以从老夫手下抢走东西，有意思！”
<br/>　　叶尘冷冷瞥了那老者一眼，
<br/>　　“怎么？你们天藏阁，就是这么做生意的吗？看到好的东西，就想动手硬抢？”
<br/>　　“你！……”
<br/>　　那老者两眼一瞪，胡子都气得吹起来了。
<br/>　　这时，旁边一名高瘦的老者，笑着开口道：
<br/>　　“这位小友，你千万不要误会！老李行事鲁莽，我代他向你道歉！”
<br/>　　“我天藏阁在云丹城，向来是童叟无欺，又怎会抢顾客的东西呢？”
<br/>　　那名被称作‘老李’的老者，也当即跟着道：
<br/>　　“就是！你这枚完美品阶的化婴丹，虽然是不错，但是本长老还没有看在眼里，本长老只是想仔细确认一下而已！”
<br/>　　叶尘见两人倒也不像是说谎，而且眼前这三名老者的气息，都是元婴境巅峰境界，这化婴丹对于他们来说，的确没有什么意义。
<br/>　　那名高瘦老者，见叶尘一脸警惕之色，转而望向那名妖媚女子，板着脸，道：
<br/>　　“既然这位叶先生，要在我们天藏阁拍卖物品，你还不快点取契书过来！！”
<br/>　　那妖媚女子见连坐镇天藏阁的三位长老都出动，哪里还敢耽搁，连忙点头称是，下去准备去了。
<br/>　　这一次，妖媚女子的效率很高，几乎一转眼的功夫，便已经去而又返，手里捧着一张金色帛纸，上面写着数行金字。
<br/>　　叶尘知道，这是修真界通用的契约书，买卖双方约定好之后，还要将自己的精神烙印刻在其中，一旦违约的话，不但会遭到精神反噬，还会在商业圈子里身败名裂。
<br/>　　“叶先生，老夫亲自和你签署契书，这下你总该放心了吧？”
<br/>　　那名高瘦老者笑眯眯的开口道。
<br/>　　他们天藏阁虽然是云丹城第一大拍卖场，可是近些年已经有些衰败，大有被另外几家拍卖场一举盖过的趋势。
<br/>　　如今突然出现了这么一颗宝丹，若是能好好宣传一番的话，必定可以将大大提升他们天藏阁的名气和威望。
<br/>　　所以，他们无论如何，也不能错过这笔买卖！
<br/>　　叶尘略微想了想，便点头答应了下来，
<br/>　　他其实根本不怕天藏阁违约，要是对方真的敢贪墨他的东西，直接灭掉，也就是了！
<br/>　　很快，
<br/>　　双方签下契书，叶尘也将那两瓶丹药，丢给那名妖艳女子，后者接过那两瓶丹药的时候，脸色变得煞白，娇躯都忍不住一阵颤动。
<br/>　　要知道，别看着只有区区三颗丹药，可是却价值数百万灵石，只要是出了什么意外，她就算把自己，恐怕卖了也赔不起！
<br/>　　……
<br/>　　签订了契书之后，叶尘直接被请到拍卖场最豪华的贵宾包厢，站在包厢中的阳台之上，可以俯视整个拍卖场中的情况。
<br/>　　只不过，此时离晚上的拍卖会，还有好几个小时的时间。
<br/>　　包厢之中，装饰的十分华丽，而且各种美酒、美食，应有尽有，甚至天藏阁还为叶尘准备了十名美女，不过却被叶尘拒绝了。
<br/>　　叶尘闲来无事，便在周围随手布置了一道禁制，盘腿进入了修炼状态。
<br/>　　说起来，从他来到中央大星域之后，就没怎么好好修炼过，但是修真界灵气浓郁，即便没怎么修炼，他如今的修为，也已经达到了金丹境八重巅峰。
<br/>　　包厢里布置了高级聚灵阵，灵气倒是十分浓郁，叶尘将吞天神功全力施展开口，周围海量的灵气，顿时犹如泉涌般，灌入叶尘的体内，转化成真元，储存在丹田之中。
<br/>　　一转眼，四、五个小时过去了，
<br/>　　“轰！”
<br/>　　一直在修炼状态中的叶尘，身体猛然一震，继而身上的气息也随着暴涨！
<br/>　　金丹境第九重，到了！
<br/>　　顺利突破之后，叶尘又略微调息了片刻，随手撤下周围的禁制，
<br/>　　而此时，外面的拍卖场中，早已经人山人海，热闹非凡。
<br/>　　很显然，拍卖会已经即将拉开帷幕！
