　　<br/>　　“那行，我去问问！”<br/>　　<br/>　　“行，再问问姚爷以后能不能过来帮我照顾牲口，每个月给他三千元的工资。”<br/>　　<br/>　　“好！”<br/>　　<br/>　　陈默比价欣赏大海这点，只要说好了，那么他就会去做，而且办事情比较沉稳，是个好帮手。<br/>　　<br/>　　“老板，所有的蔬菜已经装好车了，而且也都称重完毕。”来采购的人跑过来对陈默说道。这是上次和凌经理一起来的，姓郝叫郝仁，很有意思的一个人名。<br/>　　<br/>　　“辛苦你了！”<br/>　　<br/>　　“客气了陈老板！这是所有蔬菜的装箱单，你看看，要是没有问题就签字吧，给你留一份，其他我还要带回去。”郝仁说道。<br/>　　<br/>　　陈默看了看，总共五千多斤的蔬菜，也没有多少，由于就是青菜和青瓜、韭菜、葱之类的，品种比较少，都是按照十元一斤算的。按照这个价格，陈默这一车也就能收入五万多远，还是不错的说。<br/>　　<br/>　　签完字，然后就递给郝仁。就见他将最后一页留给陈默，将其他几页全部都装好，才说道：“陈老板，关于款项，等回去后就去报账，争取下午下班之前给你到账。”<br/>　　<br/>　　因为这个钱款的事情，早在他来的时候，就被凌经理交代过，所以才这么说。<br/>　　<br/>　　“没有问题。”陈默点头说道。<br/>　　<br/>　　送走郝仁以后，陈默有交代了一下陈金贵等人。自己就一个人先去了厂棚搭建的地方，看了看工程进度。还不错，所有的都搭建的差不多了。也就剩下一些细节上的处理，另外就是水泥地坪的浇注。<br/>　　<br/>　　看了看没有啥好说的，也就慢慢溜达回家了。自己的皮卡车已经被大海修好以后还回来了，停在房子的前面。也就是换个车梆的事情，所以还是非常快速的，钥匙也还给了自己，所以陈默就像开车出去一趟，将鱼苗拉回来。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
