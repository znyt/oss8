　　苏莫跟着赖羽，在赤龙营中穿梭。
<br/>　　赤龙营的一座座石殿，均是排列有序，十座为一组，十组为一排，共计有三十余排，多达三千座不止。
<br/>　　而且每一组的石殿，都有相应的编号。
<br/>　　不多时，赖羽便带着苏莫，来到了最左边一组石殿之前。
<br/>　　这组石殿的门梁之上，均有二十九的标号，想必这就是二十九小队的住所。
<br/>　　“这里就是二十九组，上官昊，这座石殿归你！”赖羽走到一座石殿之前，对苏莫说道。
<br/>　　“嗯，多谢！”苏莫点头，向赖羽道谢。
<br/>　　“这是你的战甲，在赤龙营必须身穿战甲！”
<br/>　　赖羽手掌一番，从储物戒中取出一件赤色的铠甲，继续道：“从今日起，你就是赤龙营第二十九小队的一名士兵，我是二十九小队的队长赖羽，你归我统领，若有违抗，军法伺候！”
<br/>　　“明白！”苏莫伸手接过了战甲。
<br/>　　“嗯！”赖羽点了点头。
<br/>　　此时，二十九组所在的几座石殿中，纷纷有士兵走了出来，其它的八名小队成员全部走了出来。
<br/>　　“老大，这谁啊？”
<br/>　　“我们二十九队的新队员吗？”
<br/>　　几人望着苏莫，纷纷开口询问赖羽。
<br/>　　“这是上官昊，你们的新队员！”赖羽点头说道。
<br/>　　“是吗？修为不错！”几人打量了一番苏莫，发现了苏莫武帝境八重的修为。
<br/>　　“见过各位！”苏莫向众人抱拳，同时眼眸也在打量其他人。
<br/>　　这二十九小队之人，有高有矮，有胖有瘦，除了队长赖羽是武帝境九重巅峰修为之外，其他人均是武帝境七重或是八重的修为。
<br/>　　当然，这些人修为之所以这么高，甚至比很多神武学府的弟子还要高，并不是说天赋就有多强，只是年纪较长一些而已。
<br/>　　“上官兄客气了！”
<br/>　　“幸会！”
<br/>　　众人见苏莫彬彬有礼，虽然不至于心生好感，但也没有什么敌意。
<br/>　　“呵呵，上官昊可是八皇子殿下推荐的人！”赖羽轻笑着说道，眸中异芒一闪而过。
<br/>　　“哦！”众人闻言惊讶，是八皇子推荐的人，那基本上就是八皇子派系的了。
<br/>　　众人顿时沉默了下来，他们虽然没有任何派系，但心中可是清楚，赖羽和中队长祝英的关系不一般。
<br/>　　而祝英，可是三皇子的拥护者。
<br/>　　这上官昊加入了二十九小队，估计日子不好过了。
<br/>　　“老大，你们聊，我们继续修炼了！”几人纷纷向赖羽说道，随即均是返回了各自的石殿中。
<br/>　　苏莫见此，眸光一闪，这些人知道了他是八皇子推荐的人，好像都刻意的和他拉开了距离。
<br/>　　“上官昊，你也休息吧！”赖羽向苏莫挥了挥手，便转身离去了。
<br/>　　苏莫叹了口气，随即走进了属于他的石殿中。
<br/>　　这石殿很小，其内空间不足二十丈，但麻雀虽小，五脏俱全，殿内客厅、练功室等房间应有尽有。
<br/>　　苏莫坐在客厅中，已经换上了赖羽给他的铠甲，一身赤色的铠甲，也是闲的威风凛凛，仿佛一名将军一般。
<br/>　　“居然还是帝级上品的宝器！”打量着身上的铠甲，苏莫有些惊讶，这铠甲乃是用特殊的材料炼制而成，能护住全身，防御力极为惊人。
<br/>　　坐在客厅中，苏莫低头沉思，赤龙营中以实力高低，担任不同的职位。
<br/>　　挑战自己的队长，战而胜之，便能取而代之，看来他要先打败赖羽了，如此才能获得小队长的职务。
<br/>　　只不过，就算打败了赖羽，还是在祝英的手底下，肯定不能安生。
<br/>　　最主要的是，苏莫并不想按照古天意的意思，在赤龙营爬上高位，他只是想混一段日子罢了。
<br/>　　咚！咚！咚！
<br/>　　就在此时，门外响起轻微的敲门声。
<br/>　　苏莫灵识一扫，顿时有些意外，因为敲门之人正是二十九小队中的一人，此人身材肥胖，肥胖的圆脸上，小眼睛只剩下了一条缝，其穿着赤色的铠甲，显得颇为不协调。
<br/>　　身形一闪，来到门前，苏莫打开了石殿的大门。
<br/>　　“嘿嘿，上官兄，我能进去坐坐吗？”胖子轻声笑道，其看了看左右，仿佛是做贼一般。
<br/>　　“请进！”苏莫不知道对方有何事，但还是客气的将对方让进了石殿中。
<br/>　　“不知兄台有何事？”
<br/>　　关上殿门，苏莫请对方在客厅的石凳上就座，好奇的问道。
<br/>　　“上官兄，在下恒飞，先互相认识一下！”胖子笑眯眯的说道。
<br/>　　“嗯！”苏莫点了点头。
<br/>　　“嘿嘿！”
<br/>　　恒飞轻笑一声，道：“上官兄，我这次来，是想要告诉你，你若想在二十九小队待下去，可要有心理准备啊！”
<br/>　　“有什么心理准备？”苏莫疑惑。
<br/>　　“你不是八皇子殿下推荐的人吗，我们小队长和中队长，可都是三皇子的人！”恒飞轻声说道。
<br/>　　“是吗？”苏莫闻言一笑，向对方抱拳道：“多谢告知！”
<br/>　　“你不担心？”恒飞目光凝视着苏莫，小眼中精光闪烁。
<br/>　　“担心什么？”苏莫问道。
<br/>　　“他们会整死你的，以前可是已经整死了两位大皇子的人！”恒飞道。
<br/>　　“多谢恒兄提醒，不过，恒兄为何要提醒我？”苏莫面色不变，淡淡的问道。
<br/>　　“这……嘿嘿，在下只是想多交一个朋友而已！”恒飞嘿嘿的笑道。
<br/>　　苏莫目光微眯，紧紧的凝视着对方，他感觉这个胖子好像不简单，但也看不出对方具体的虚实。
<br/>　　此人的修为，和他相当，同样是武帝境八重的境界。
<br/>　　“是吗，那我就交了恒兄这个朋友！”苏莫微笑道。
<br/>　　“嘿嘿，上官兄一看就不是普通人，能交下上官兄这个朋友，恒某非常荣幸！”
<br/>　　恒飞意味深长的说道，言罢，他起身向苏莫抱拳，道：“上官兄，你好自为之，在下先走了！”
<br/>　　“不送！”苏莫道。
<br/>　　随即，恒飞便大步离开了石殿。
<br/>　　待恒飞离开，苏莫无奈的叹了口气，此人为何如此好心的来提醒，真是让人费解。
<br/>　　摇了摇头，他关上殿门，便准备静心修炼，但就在此时，他的眸光瞬间一凝，望在了恒飞刚才所坐的石凳。
<br/>　　那还算平滑光洁的石凳上，此刻正刻着一个小小的‘苏’字。
