　　第505章
<br/>　　聂隐娘的尸体从地牢中抬出来，身上蒙着白布，仵作再三验过，已经死得不能再死，诈死重生的事情不会在她身上发生。
<br/>　　程大雷本来想得挺好，还准备将聂隐娘忽悠到身边，成山寨一大助力。没想到这姑娘如此性绝，程大雷还没来得及忽悠，她就服毒自尽了。
<br/>　　归根结底还是程大雷没多少经验，如果将她收押之前，仔仔细细检查过，也不会有聂隐娘自尽的事情发生。
<br/>　　望着聂隐娘的尸体，程大雷默默无言，本也是俊俏人物，此刻却已经是香消玉损，一命呜呼。
<br/>　　“卿本佳人，奈何为贼。”程大雷呐呐叹了一声。
<br/>　　“大当家，这人的尸体怎么处理？”仵作问。
<br/>　　“还能怎么办，天亮找地方埋了吧。”
<br/>　　程大雷随口道了一句，聂隐娘的死出乎他的预料，心中的确觉得有些可惜，但除此之外，倒也不觉得如何。
<br/>　　毕竟程大雷也是从一次次大战中闯出来的，死在他斧下的人已经不知多少，早不是青牛山那个初出茅庐的雏儿。聂隐娘死了就死了吧，毕竟这姑娘本事再了得，也是自己的敌人，不能用之，必定除之。
<br/>　　好在她死了后，程大雷还获得一次抽奖次数，相比薛丁山，这聂隐娘可是个超级boss，也不知道她会给自己爆出什么。
<br/>　　……
<br/>　　距离凉州三千里外，无名山谷，听月庵中。
<br/>　　一白衣尼姑盘腿坐在湖边，手中一串念珠，她闭目合十，默默祷告。
<br/>　　啪地一声，手中念珠断开，骨碌碌的珠子啪啪散落在地上，骨碌碌滚远。
<br/>　　这白衣尼姑忽地睁开眼睛，双目陡射出两道寒芒，犹如那失犊的老虎，负伤的狮子。
<br/>　　半晌，她眼中的杀机散去，口中呐呐叹了一声：“我的徒儿死了。”
<br/>　　她恍若什么事情都没有发生，弯着腰将地上的念珠一粒一粒拣起，用麻绳串在一起。随之起身回到茅屋中，在锅里煮了一升小米，又将随身的衣物收拾一番。
<br/>　　她吃罢了饭，洗净了手，在茅屋内小憩片刻。随之背上包袱，托着拂尘，迈步走出了无名山谷。
<br/>　　这尼姑不知如何来历，也无人知道她拜的什么佛，念的是那一路经。山中有猎户樵夫，偶尔能在深山密林中窥见她。这白衣尼已不知多久没走出这无名山谷，屈指算来，这该是十年来第一次。
<br/>　　踏出山谷，头顶彤云密布，目光辨了辨方向，向西北行去。
<br/>　　西北有凉州。
<br/>　　……
<br/>　　凉州境内，又过去了十几日时间。程大雷终于从地牢中钻出来，可以站在朗朗乾坤下，呼吸几口新鲜空气。
<br/>　　自从聂隐娘死了之后，程大雷展现了从所未有的勤奋，开始管理山寨内的大小事务。
<br/>　　原因当然不是程大雷突然变成了工作狂，而是他在地牢里憋得太狠。
<br/>　　到这一日，白元飞从长安回来，亲自到程大雷面前汇报。
<br/>　　“回来了？”程大雷看到白元飞受了一圈，想这一路定然辛苦：“落凤城的情况怎样？”
<br/>　　“不是很好。”白元飞道：“落凤城三面环山，一面靠海，处在各方势力的夹缝中。除此之外，还有海贼时时登岸作乱，祸乱百姓。”
<br/>　　程大雷啧啧叹了一声：“看来李行哉的基础盘不是很好了，只是两地相隔太远，咱们也帮不了他什么，也不知道他会怎么开局。好了，以后多关注那边的情报，有情况变化第一时间汇报给我。”
<br/>　　“是。”
<br/>　　“好啦，你先下去好好歇一阵吧，阿喜现在正训练间谍，等你休息好了，就过去看看，看那边有什么是你能做的。”
<br/>　　“那大当家我先下去了。”
<br/>　　白元飞刚走不久，和珅就过来了，向程大雷汇报道：“大当家，有人前来求见。”
<br/>　　“谁呐？”程大雷眼皮都没抬：“谁想见我，会不会是什么刺客？”
<br/>　　程大雷当真被聂隐娘搞得都有些心理阴影。
<br/>　　“呃……好像不是刺客，而是将军府的人？”
<br/>　　“将军府……”程大雷皱皱眉头：“我和将军府没有瓜葛啊，他来见我做什么？”
<br/>　　“我与他谈过话，他的话说得很密，不露半点口风，大概是想见到大当家后再聊。大当家，你要不要见他？”
<br/>　　程大雷琢磨了一会，道：“暂时不见，先安排他在山寨住下，派人盯好，不要让他在山寨中乱走。等搞明白他是来干什么的，我再决定见或者不见。”
<br/>　　“大当家高啊！”
<br/>　　“高么？”程大雷呵呵笑笑：“也就一般高啦。”
<br/>　　其实最近想要见程大雷的人不少，江湖上来投靠的猛人，有些大世家向程大雷抛出橄榄枝，包括凉州宋伯康……但程大雷因为聂隐娘的事情缠着，也没有时间见他们。
<br/>　　现在事情终于告一段落，程大雷可以腾出手来处理一下外交工作。
<br/>　　他忽地想起，自己在击杀聂隐娘后，还获得一次抽奖机会，倒也不知道这聂隐娘会爆出来什么。
<br/>　　程大雷考虑了一下，选择抽奖。
<br/>　　嘟，获得绝世兵器青龙偃月刀。
<br/>　　青龙偃月刀，重八十二斤，又名冷艳锯。
<br/>　　程大雷眼睛一亮，终于，自己给关二爷凑齐了一套装备。
<br/>　　嘟，获得顶级武将史文恭。
<br/>　　嘟，随机降临顶级武将戚继光。
<br/>　　史文恭，座下千里狮子马，使一杆方天画戟，乃是梁山的一员大敌，最后……死得好惨啦。
<br/>　　戚继光，明朝名将，乃是文武双全的角色。
<br/>　　程大雷看着收到的系统提示，心情有些无语，怎么看这史文恭和戚继光都有很大差别。虽然系统给二人的评级都是顶级，但这戚继光明显是有升级潜力的，而这史文恭……怕顶级就是他的极限了。
<br/>　　果然呐，出现一个帮手，就会帮自己召唤出一个了不得的对手。薛丁山如此，聂隐娘如此，想这戚继光也必然如此。
<br/>　　不过这戚继光和聂隐娘不同，定然不会拎着大刀来杀自己，怕也是沦落到其他势力手中，日后说不得会有战场相遇的机会，也可能一生一世都碰不着面。
<br/>　　只是不知道，这戚继光会落到哪里。
