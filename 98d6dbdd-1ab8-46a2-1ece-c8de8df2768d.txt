　　这一战，一日之内，将会传遍整个虚空海。
<br/>　　而楚风眠的名字，也将响彻虚空海中的各个角落。
<br/>　　“小子，你不要得意！你今日敢让我们跪下，已经是彻底的得罪了天盟？你以为你是神力境无敌的人物？我们天盟的盟主，已经足以灭杀生死境高阶的存在，是真正的大人物，你激起他的怒火，也要死！”
<br/>　　一尊天盟弟子，跪在地上，一动不动，脸色涨得通红。
<br/>　　周围无数围观武者的目光，几乎是让他感受到了无尽的屈辱，但是就算如此，他的心中也依然不服输。
<br/>　　他的心中，有着寄托，他相信，天盟的盟主，肯定会杀了楚风眠，为他们报仇。
<br/>　　在所有天盟弟子心中，天盟的盟主，才是真正的无敌人物。
<br/>　　“不错，小子，你就算是逃到天涯海角，你也只有死路一条。”
<br/>　　又是一尊天盟弟子狂吼起来。
<br/>　　“是吗？”
<br/>　　楚风眠冷冷一笑，大手一挥，这在场所有天盟弟子的胸口，都是猛然裂开一个口子。
<br/>　　从其中，无数圣者精血直接被楚风眠抽了出来，融入到了他的身躯之中去。
<br/>　　“天盟的盟主？他来了，也是跟你们一样的下场。”
<br/>　　楚风眠冷笑一声。
<br/>　　什么天盟的盟主？
<br/>　　足以轰杀生死境高阶的存在？
<br/>　　楚风眠连不死魔尊都能轰杀，一个天盟的盟主，又算得了什么。
<br/>　　更何况，只要楚风眠的实力再进一步，将身躯达到完美，便是就足以度过生死大劫。
<br/>　　一旦是度过生死大劫，楚风眠在这北域中，就真的可以肆无忌惮，随意纵横了。
<br/>　　“你们所有人，都跪在这里，我要让所有人知道，敢来招惹我楚风眠的下场！”
<br/>　　楚风眠冷哼一声，转过头对着旁边的李长老开口道。
<br/>　　“走吧，去拍卖会。”
<br/>　　李长老看了一眼那些跪在地上的天盟弟子，犹豫了一下，还是没有开口。
<br/>　　带着楚风眠跟白萱，直接离开了。
<br/>　　一直等到楚风眠的身影远去，在场的无数人，才是松了口气。
<br/>　　楚风眠的手段，实在是太狠了，击败了天盟弟子不说，更是抽取了他们的圣者精血。
<br/>　　包括天盟的副盟主洪狱。
<br/>　　抽取了他们九成的圣者精血，也就意味着，这些天盟弟子，全部都是变成了废人了。
<br/>　　这对天盟来说，绝对是极大的损失了。
<br/>　　尤其是现在，这些天盟弟子，都是全部跪在了地上。
<br/>　　天商会城门前，本就是人来人往极为密集的地方，人人都可以看到这一幕。
<br/>　　这对天盟来说，都是一种绝对的侮辱。
<br/>　　在虚空海中，敢这样对付天盟的人，也仅此楚风眠一人了。
<br/>　　“此人的手段，实在是太过恐怖了，如今彻底撕破了脸皮，天盟也不会善罢甘休了。”
<br/>　　“那又如何？天盟的副盟主洪狱都败在了此人的手中，天盟就算是想要报复，也无能为力了。”
<br/>　　“洪狱算什么，天盟之中最为恐怖的，还是他们的盟主，这天盟就是由那盟主亲手建立的，要是天盟的盟主出手，也不好说。”
<br/>　　“不用管那么多，反正跟天盟平日里也没少欺压我们，我们只需要看热闹就是。”
<br/>　　一尊尊围观武者，也逐渐散了。
<br/>　　只有一部分都散开了，只是众人都清楚，在这虚空海中，有多了一尊可怕天才的存在。
<br/>　　另一边，李长老便是带着楚风眠跟白萱，二人一同向着混元大陆中心飞去。
<br/>　　如今这混元大陆的拍卖行，便是就由混元大陆的第一势力，通天阁举办。
<br/>　　通天阁的所在，正是在混元大陆的中心。
<br/>　　“楚风眠，这天盟的盟主，的确不是一尊简单的人物。”
<br/>　　一边飞行这，一边白萱在一旁暗自开口道。
<br/>　　“这天盟的盟主，名为混苍生，我也曾经听说过他的名字，据说他的确斩杀过生死境高阶的强者，实力深不可测，你可要小心。”
<br/>　　“哦？能够以神力境的境界，足以斩杀生死境高阶的强者，这种实力，的确不错，不过若是敢来招惹我，我也会让他付出代价！”
<br/>　　楚风眠笑着开口道，丝毫没有在意。
<br/>　　他的实力，已经在神力境中，近乎无敌，楚风眠不相信还有那个神力境武者，可能是他的对手。
<br/>　　更何况楚风眠只要是从拍卖行上，在买到一些诸天奇物的话，也足以是将他的身躯凝练到了极致。
<br/>　　到时候，在生死境高阶的强者中的一些老古董，楚风眠也不是没有抗衡的可能。
<br/>　　“只要是能够寻找到足够的奇物，布置出圣祭来，我的身躯很快就足以达到神力境的极致了。”
<br/>　　楚风眠想了一下空戒中的那枚龙蛋，心中也是无比激动。
<br/>　　能够碰到这龙蛋，也真是楚风眠的好运气了。
<br/>　　这龙蛋，的确是毫无生机的死蛋，但是却并非是寻找真龙的龙蛋，而是一枚涅槃之蛋。
<br/>　　在强大的存在，都会在岁月中陨落的一日，想要跟岁月抗衡，唯一的办法，便是涅槃重生。
<br/>　　这种涅槃之道，是从远古时代便是就流传出来的，等到一些强者寿元快尽的一刻，便是将身上所有的力量，都给凝聚一点，涅槃重生，化为新生。
<br/>　　再换取一世的寿元来。
<br/>　　眼前这龙蛋，楚风眠感觉的出，便是一尊龙族强者涅槃重生化为的，只是这尊强者，涅槃重生失败，失去了生机。
<br/>　　涅槃之道，逆天而行，真正可能成功的，不足十分之一。
<br/>　　其中强者虽然失去了生机，但是这龙蛋其中的灵力，却是都保存了下来。
<br/>　　能够选择涅槃重生的龙族，在龙族中至少都是达到龙皇级别的强者。
<br/>　　这龙蛋中蕴含的灵力，只怕是雄厚到极致，只是这其中的力量，并不能直接抽取，而是必须以一种圣祭之术，将其中的力量祭炼抽取出来才行。
<br/>　　这种圣祭之术，古老无比，本身是曾经远古满族，以活人身躯祭炼，抽取灵力所用的。
<br/>　　这龙蛋中的灵力，无法动用，但是动用圣祭之术的话，也足以将其中的力量全部抽出来。
<br/>　　“这龙蛋中蕴含的力量，只要是圣祭出来，足以将我的身躯，都达到大成，圣祭需要的材料众多，不过在颜古龙帝洞府中，我已经是得到了许多材料，只需要在购买其中最为重要的几件，便是就可以布置圣祭。”
<br/>　　楚风眠心中暗自想到。
