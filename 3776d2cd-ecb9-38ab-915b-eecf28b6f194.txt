　　老李这种不切实际的想法也是出于无奈的心态，天天被一群老货追在屁股后面要手铳的老李只觉得已经快要被逼疯了。
<br/>　　可是偏偏李承乾那个小混蛋却咬死了不松口，手铳除了用来奖励‘品学兼优’的老货之外，根本就是谁也不给，就连老头子想要都要不出来。
<br/>　　甚至逼问的急了就拿出一副无赖相，丢下一句：国之利器，岂可轻予！然后就是一声不吭的积极的消积抵抗。
<br/>　　最后李二没啥招了，把自己的一对儿手铳拿出来送给了长孙无忌，然后又把长乐的手铳骗来了一支，送给了候君集。
<br/>　　至于其他人……没办法，自求多福吧，骗闺女一支手铳已经很丢脸了，老李对长乐手中的另一支实在下不去手。
<br/>　　而且手铳也不是白骗的。
<br/>　　‘兰若寺’中，李承乾看着洋洋得意的长乐，脸色苦的一逼：“丽质啊，你是说你手里的玉佩是父皇给你的？”
<br/>　　“对啊，是我用一只手铳换的。”长乐拿着与李承乾挂的那只玉佩一模一样的玉佩炫耀着。
<br/>　　这是第七块，竟然真的有第七块，可是……为什么第七块会在长乐手里？这小丫头现在已经长大了，想从她手里骗东西可不容易，老头子要一只手铳都要用玉佩来换，自己如果想把玉佩换过来，这事儿怕是很麻烦。
<br/>　　“哥，你可是有六块的，想不想要这一块啊？”随着年龄的增长，长乐继承自长孙皇后强大的第六感开始发挥作用，总是能通过一些表现看到别人内心真实的世界。
<br/>　　“要个屁，这是父皇给你的，如果突然没了，父皇追究起来，你哥我可不想被打板子。”李承乾被看窥透心事有些羞恼，在长乐的头上轻轻弹了一下，正打算继续‘教训’她，忽然看到黑子站在院子外面探头探脑。
<br/>　　“进来吧，站在外面晃悠像什么样子。”收起与长乐调侃的表情，李承乾把黑子叫了进来。
<br/>　　这小子现在负责的事情越来越多，一般很少会跑到‘兰若寺’来，如果来了，那就是有大事要发生。
<br/>　　“殿下，西域要乱了。”进了院子，黑子扫了一眼院子里的众人，发现没什么碍眼的，便直言说道。
<br/>　　“西域要乱了？什么情况？”李承乾眉头一皱，西域必然会乱，这是毋庸置疑的，而且就算西域不乱他也会搅乱它。
<br/>　　可是现在正在准备面对吐蕃入侵，薛仁贵所率领的左武卫作为一颗钉子根本不可能擅离大非川，那么在西域可以用的兵力就只有已经再次深入西域的尉迟宝林所率领的一万五千人。
<br/>　　所以眼下的情况并不适合让西域乱起来，真要是乱了，凭借尉迟宝林的一万五千人马，或许很难实现当初设想的目标。
<br/>　　“汉子殿下授意于阗拒绝向大军提供军粮，附议者甚重。这是尉迟小将军在带兵讨伐之前，发出的战报。”黑子掏出一张不大的纸条递给李承乾。
<br/>　　“父皇那里知道了没有？”李承乾接过纸条，随口问道。
<br/>　　“陛下那里已经收到消息……，说是由您全权处理。”黑子犹豫着说道。
<br/>　　从来都是‘第七小组’的消息速度高于情报科，这一次他竟然晚了，这让黑子觉得有些不自在。
<br/>　　“嗯！”李承乾不动声色的答应一声，没有给黑子任何解释的机会。
<br/>　　“殿下……”
<br/>　　“‘第七小组’的事情我不想管，具体的事情你回去办，下次最好不要出现这样的情况。”李承乾摆手打断黑子，意有所指的说着。
<br/>　　长乐就在边上，李承乾没办法把事情说的太明白，但这样的说法已经足够让黑子明白他对‘第七小组’情报滞后的不满。
<br/>　　“喏！”黑子慎重的点点头，想了想又说道：“殿下对西域那边有什么指示？”
<br/>　　李承乾沉默片刻，抬起头目光灼灼的盯着黑子说道：“告诉尉迟，若是打输了，提头来见。”
<br/>　　“喏！”
<br/>　　“哥，真要打啊？李元昌可是还在于阗呢，如果打起来他会不会被当成人质？”自从知道上次绑架自己的是李元昌的护卫，长乐对李元昌就是直呼其名。
<br/>　　“人质？”李承乾愣了一下，很快就扭头看向不远处的一个角落：“小夜，去追上黑子让他回来。”
<br/>　　“哥，你这是怎么了？怎么一惊一乍的？”长乐被李承乾的反应吓了一跳，有些郁闷的问着。
<br/>　　“没事，哥就是想起一点事情。”李承乾安慰一下长乐，然后说道：“丽质啊，那个……哥还有点事儿，你……”。
<br/>　　“哦！”长乐不等李承乾说完，就给了他一个大大的白眼儿：“那我先走啦。”
<br/>　　“去吧，哥有空再去找你玩。”李承乾看着被夜魅叫回来的黑子，敷衍着对长乐说道。
<br/>　　“哼，臭大哥！”长乐被李承乾的态度气的狠狠一跺脚，带着自己的两个侍女气冲冲的走了，路过黑子的时候还顺带在他的腿上踢了一脚。
<br/>　　“臣恭送公主！”黑子被不痛不痒的踢了一脚，虽然不知道为什么，但脸上依旧装出一副痛苦的表情。
<br/>　　“行了行了，收起你那个嘴脸吧，人都走没影儿了。”李承乾看着有些做作的黑子，有些不耐烦的说道。
<br/>　　黑子咧嘴一笑不以为意，走到李承乾身边问道：“殿下还有什么吩咐？”
<br/>　　“师出不可无名，记得告诉尉迟一声，他这一次出征的名义就是解救汉王，因为于阗国以汉王为人质，所以兴兵讨伐，明白了没有？”
<br/>　　黑子眼角抽了抽，目光中带着高山仰止的佩服之情，伸出一根姆指：“殿下，高，实在是高！”
<br/>　　“滚犊子，快点去办事儿，耽误了事情小心你的一身皮。”李承乾抖着浑身的鸡皮疙瘩，虚踢一脚。
<br/>　　继吴辰之后，黑子这家伙越来越不着调，甚至隐隐超越吴辰的趋势。
<br/>　　当然，如果有外人在的时候，这家伙还是一如继往的正经，只是没人的时候会显得逗逼一些，不知道是不是被独孤玉凤那丫头给管懵了。
