        连星儿离开了酒店后，整个人有些神情恍惚地走在街道边。
<br/>
<br/>        其实，她知道明司寒是故意骗她的，但是，她还是因为他的谎话而伤了心。
<br/>
<br/>        她为什么要奢望他会喜欢上自己呢？
<br/>
<br/>        他应该没理由要喜欢上她的吧！
<br/>
<br/>        一路跌跌撞撞，走不下去的时候，连星儿聋拉着脑袋坐在了路边的石凳上。
<br/>
<br/>        这会儿，明司寒的事情还未让她缓过神来，临海城里，祖爷爷的事情又让她伤脑筋了。
<br/>
<br/>        傅家大庄园的管家打了通电话过来，说祖爷爷突然晕倒住院了。
<br/>
<br/>        连星儿顾不上自己的私人感情，火急火燎地从北城坐飞机飞回临海城。
<br/>
<br/>        她带着行李直接去了祖爷爷傅炎廷的病房，结果祖爷爷安然无恙地坐在轮椅上，只是想她这个小曾孙女推他去医院的中心花园里透透气。
<br/>
<br/>        “祖爷爷，你真的是把我吓一跳！”连星儿一边推着傅炎廷走在花园中的石子路上，一边抱怨道。
<br/>
<br/>        “你爹地最近动不动就出国，前几天刚回来，今天又出国了，忙得整天都不见人影。你妈咪每次来傅家大庄园都只是陪你奶奶，跟我这个老头子没话题可言。哎，我这个老头子太孤单了！”傅炎廷像个老顽童似的跟连星儿诉起苦来。
<br/>
<br/>        连星儿会心一笑，微微俯身，凑到傅炎廷的脑袋旁，打趣道：“那……我休学在家，天天陪着祖爷爷，如何？”
<br/>
<br/>        “那怎么行！你祖爷爷我要求不多，只希望你这小丫头，双休日来陪陪我就好。”傅炎廷轻轻地拍了拍连星儿搭在自己肩头上的小手。
<br/>
<br/>        连星儿微笑着应道：“好！以后，我的双休日，就归祖爷爷所属了！”
<br/>
<br/>        “诶！好！祖爷爷我没白疼你这小丫头！”傅炎廷笑不拢嘴地说。
<br/>
<br/>        “宋云汐，她是你妹妹！你怎么这么狠心，见死不救？”突然，一个男人的呵斥声，打断了连星儿和傅炎廷的话。
<br/>
<br/>        连星儿循声望去，只见前方的蔓藤廊坊里，站在一男一女。
<br/>
<br/>        男的，连星儿不认识。
<br/>
<br/>        倒是那个女的，却令连星儿印象深刻。
<br/>
<br/>        “祖爷爷，她……她不就是那个相亲时拒绝了哥哥的女机长吗？”连星儿吃惊道。
<br/>
<br/>        傅炎廷听连星儿这么一说，也眯起眼睛，仔细打量起前方的那个穿着墨蓝色职业装的女人来。
<br/>
<br/>        女人妆容精致，气质高雅。
<br/>
<br/>        反倒是一旁的男人，虽然西装革履，面庞英俊，却不及女人一半的气质。
<br/>
<br/>        “我妹妹？！”宋云汐冷笑，看着面前这个前男友郑温韦，鄙夷道，“一个小三的女儿，又抢了姐姐的男朋友的女人，也配当我妹妹！”
<br/>
<br/>        “云汐，我们之间的事情，你能不能不要怪罪到雨柔的身上？”郑温韦极其不耐烦地说，顿了顿后，又抱怨起宋云汐的不是，“我跟你在一起的那三年里，聚少离多。你常年跟飞机，性子又高傲孤冷。你叫我怎么跟你相处？”
<br/>
<br/>        “怎么跟我相处？呵！那你至少也要跟我提分手后，再跟我宋雨柔鬼混吧！你这一边在信息上跟我发着‘我永远爱你’，又趁着我学飞行员跟飞机的期间，跟宋雨柔搞一起是什么意思？”宋云汐铿锵有力地质问道。
<br/>
<br/>        “我说过，我跟你在一起的时候，和雨柔是清白的，你就是不信！你的猜忌让我很痛苦，你知道吗？你每天倒时差地给我打电话查岗，我有个时候在开会，你这样做，让我很没面子，你知不知道？”
<br/>
<br/>        “你没做对不起我的事情，又怕我查什么岗？”
<br/>
<br/>        “我知道，三年前，分手是我提出来的，所以你一直耿耿于怀。我说过，我们是性格不合，跟雨柔没关系。但是，都三年过去了！你依旧抓着这件事情不放！云汐，你是不是还爱着我？”郑温韦语重心长地问道。
<br/>
<br/>        宋云汐冷冷一笑：“我是瞎了眼吗？还爱着你？你有什么值得我继续爱？”
<br/>
<br/>        “云汐，我不想跟你吵。我只求你救雨柔！这样，好不好？你捐骨髓救雨柔！我娶你为妻！”郑温韦信誓旦旦地说道。
<br/>
<br/>        宋云汐顿时看着郑温韦傻眼了。
<br/>
<br/>        要说他郑温韦跟宋雨柔是清白的这种话，傻子才会信吧！
<br/>
<br/>        她若是答应做骨髓移植手术，那她将要面临的是“停飞”。
<br/>
<br/>        小时候，就是宋雨柔的母亲的插足，导致她的父母离异，她由母亲一个人抚养长大，在这期间，父亲对她是不闻不问，把所有的父爱都给了宋雨柔。
<br/>
<br/>        她凭什么拿自己的未来，去救一个小三的女儿？
