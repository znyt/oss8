<br/>        “当兵哪有不黑不瘦的？”傅夜沉忍俊不禁道，大掌直接握住了童筱颖捶在胸膛上的手。

<br/>        对于小晏的命运，从她还未怀上的时候，童筱颖就已经知道了。

<br/>        大哥也是承载着这种命运与重担出生的。

<br/>        这是一个上了百年的家族使命。

<br/>        以前童筱颖不理解，现在她只能选择默默地接受。

<br/>        兄妹两这样对比起来，星儿确实要幸福多了。

<br/>        好在，那丫头没被大哥傅夜沉给宠坏。

<br/>        这北城的天气比她家里那边要冷多了。

<br/>        明司寒见状，不由自主地拿起遥控器，将室内中央空调的温度稍稍调高了点。

<br/>        明司寒看她那看旅游指南还认真做笔记的模样，就好像是看到了一个高中生在备战高考。
