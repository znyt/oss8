<br/>    对于恐怖片，陈明只知道一条，那就是不听话是推动剧情的主线，这个不能碰，额，我碰一下，哪里不能去，额，我去一下，这个不能摸，额，我摸一下，然后，然后所有人都死了，对，这就是恐怖片。
<br/>    
<br/>        过了两天，紫霞仙子清点了一番人数，都是悟道境的人王，足足有着二十八位，他们已经在传送阵前等待了一刻钟，若不是带队的是紫霞仙子，早就有人发脾气了。
<br/>    
<br/>        紫霞仙子的威名，他们有所敬畏，并不敢在紫霞仙子面前放肆。
<br/>    
<br/>        就在这时，戴着‘滑稽’面具的一男一女来到了此处，男的一身青色的道袍，手中一杆拂尘，有不少修行者心生疑惑，都什么时代了，还有人带着拂尘做兵器？
<br/>    
<br/>        女的背后背着一柄巨剑，便是那最近盛传的卓青瑶，怪不得有这么大的面子，敢让紫霞仙子等待，原来来头很大，是燕山的人。
<br/>    
<br/>        紫霞仙子看到人已经到齐了，淡淡道：“既然人已经到齐了，那么我们就出发吧。”
<br/>    
<br/>        陈明跟着他们踏入传送阵，在传送阵之中感受了一番，作为一个低级阵法师，对于这传送阵简直再熟悉不过，陈明甚至在想，如果自己的阵法造诣够高，是不是可以随时布置下一个传送阵，把传送阵当成瞬移用。
<br/>    
<br/>        这样一来，自己可谓是神出鬼没了！
<br/>    
<br/>        估摸着其他人听到他这个想法，会惊呆的瞪出双眼，等于说传送阵还能这么玩？
<br/>    
<br/>        心中心念一起，陈明也是心生了这个想法，一时之间挥之不去，仔细看了一番这传送阵的布置手法，已经是明白了几分。
<br/>    
<br/>        不过想要布置，还是有点太难了，毕竟这传送阵都是上古流传下来的，材料太过难得了。
<br/>    
<br/>        不过，倒是可以考虑一下啥时候去偷一座传送阵，这个主意好像挺不错的样子啊。
<br/>    
<br/>        现在还是不要想这些，还是想着怎么增强自己的心念，让自己可以施展玄关剑舞吧，不然，空有一身玄关剑舞，却没有任何办法施展。
<br/>    
<br/>        走出传送阵，跟着紫霞仙子飞了一阵，在一处寒潭前面落下，紫霞仙子带着众人来到一处大殿之中，紫霞仙子道：“这里便是九幽寒泉，这潭水，便是入口，进入其中，便是进入了九幽寒泉之中，但是其中黑暗无比，消耗法力照明，太过浪费，散开感知，却是会被这九幽寒泉之中的寒气阻挡，所以诸位还是在雪山宗这里租借一枚夜明珠。”
<br/>    
<br/>        紫霞仙子不过一会儿便回来了，带过来的，同样还有着夜明珠，给陈明和卓青瑶一人一颗，“你们不要把这夜明珠弄丢了，回来之后要还的。”
<br/>    
<br/>        陈明直接收到了储物戒之中，紫霞仙子继续道：“你们最好带了足够的灵石，进入九幽寒泉之后，需要消耗法力抵挡寒气。”
<br/>    
<br/>        陈明点了点头，表示明白。
<br/>    
<br/>        半日之后，铁牛谷的人也到了，铁牛谷同样是古仙门，传闻他们祖师，便是依靠一头铁牛成仙的，故而宗门的名字叫做铁牛谷。
<br/>    
<br/>        铁牛谷的人，当头的是一个身披重甲的年轻人，胯下一头青眸玄牛，此人便是铁牛谷的传人，刘芒，浑身血气彭拜，看得出来是一位苦修。他在悟道榜上的排名，是第四，在他的身后，同样是跟着一群苦修，也是想要跟着铁牛谷的人，在九幽寒泉之中寻找机缘。
<br/>    
<br/>        剑王阁的人，一日之后就已经到了。
<br/>    
<br/>        天空之中一阵飞剑掠过，数十个人王落在地面上，惊得周围的其他宗门的人王惊叹连连，真是好一群剑仙，剑王阁的传人苏青阳左右扫了一眼，并没有发现最近盛传的卓青瑶，对于卓青瑶，他可是很想见上一面，毕竟卓青瑶手中的巨阙，在剑道之中地位很高。
<br/>    
<br/>        至于说抢夺卓青瑶的巨阙，他没那个兴趣，巨阙，不是谁都能拿得动的。
<br/>    
<br/>        他仅仅只是好奇。
<br/>    
<br/>        四大仙门到齐，其他有着大君坐镇的宗门，悟道境人王也是悉数到场，他们都清楚这件事背后的意义，如今大战在即，各大宗门都迫切的需要大君级的战力。
<br/>    
<br/>        悟道境想要超脱，求得便是一念超脱凡尘，悟道境的最后一个境界是法相境，唯有法相圆满，与法相心意相通，融法相于自身，便是超脱。
<br/>    
<br/>        而这，就需要还魂水。
<br/>    
<br/>        当然，此次前来的，也并不全是法相境的存在，道种境，神府境，同样有人前来，他们也需要部分机缘，而九幽寒泉，便是有着悟道境的机缘所在的地方。
<br/>    
<br/>        三天之后。
<br/>    
<br/>        一位白袍的雪山宗长老来到众人面前，陈明扫了一眼，乖乖，危险指数：2200！
<br/>    
<br/>        这就是大君？
<br/>    
<br/>        都快是自己的两倍了，这样的存在对付自己，估计只需要一招，看来若不是十万大山的九位大妖王都拥兵自重，妖族大君早把他们全宰了！
<br/>    
<br/>        雪山宗长老道：“此次开启九幽寒泉，诸位可以尽情地在其中寻找机缘，不过，还是要量力而行！为了以示公平，在三年之内，不会有任何人进入九幽寒泉之中。这里就是你们王级的天下，仙路渺茫，机缘天赐，去吧。”
<br/>    
<br/>        陈明左右扫了一眼，忽然在人群之中发现了一缕魔气，不过晃眼之间，那一缕魔气就已经消失不见，可能是自己看错了吧，这里可是仙道大会，魔宗的人怎么可能跑到这儿来，这不是耗子上街，找死吗？
<br/>    
<br/>        陈明看着众多王级一跃跳入九幽寒泉之中，陈明也是跟在紫霞仙子的身后，跟着紫霞仙子，一跃跳入九幽寒泉之中。
<br/>    
<br/>        雪山宗长老回到雪山宗之上，在山顶之上，朝着雪山老君拱手道：“老君，已经开启九幽寒泉了。老君，以你的眼光，你觉得这一次可能会出现大君吗？”
<br/>    
<br/>        老君微微点头，长叹一声道：“希望可以出现一位大君吧，整个大陆，已经十年没有出现新的大君了，大君，毕竟是大陆之上的最强战力，哪里会有那么简单，这是一道坎，一道极难跨过去的坎，岁月如刀斩天骄啊！”
